Pomocne może być ukrycie niektórych sektorów, aby lepiej widzieć inne. </p><h2> <a class="anchor" id="method" href="#method"><i class="fa fa-link"></i></a> metoda </h2><h3> <a class="anchor" id="overview" href="#overview"><i class="fa fa-link"></i></a> Przegląd </h3><p> Odpowiednie profile obciążeń są przypisane do zakładów przemysłowych, budynków mieszkalnych i budynków trzeciorzędnych według wyboru użytkownika. </p><h3> <a class="anchor" id="details" href="#details"><i class="fa fa-link"></i></a> Detale </h3><h4> <a class="anchor" id="residential-warm-water-supply" href="#residential-warm-water-supply"><i class="fa fa-link"></i></a> Zaopatrzenie w ciepłą wodę mieszkaniową </h4><p> Na podstawie powierzchni brutto budynków mieszkalnych i ich numeru identyfikacyjnego Nuts0, energia zużywana do zaopatrzenia w ciepłą wodę jest obliczana według poniższej listy. Z identyfikatorem Nuts2 określonego obszaru przypisywany jest profil obciążenia. Zasilanie ciepłej wody użytkowej jest zwielokrotniane przez współczynnik dostarczania ciepłej wody użytkowej w przypadku, gdy użytkownik chce dostosować wartość. </p><p> <em>Konkretna energia na obszar dostarczania ciepłej wody w budynkach mieszkalnych w różnych krajach</em> </p><p> | Nuts0 ID | Zaopatrzenie w ciepłą wodę w kWh / m² / rok | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21,38 | | CY | 8,80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14,35 | | FI | 10,15 | | FR | 9,66 | | DE | 8,27 | | EL | 12,51 | | HU | 13,66 | | IE | 15,91 | | IT | 14,01 | | LV | 15,71 | | LT | 13,36 | | LU | 8,29 | | MT | 10,99 | | NL | 8,91 | | PL | 10,00 | | PT | 9,48 | | RO | 11,48 | | SK | 21,51 | | SI | 21,74 | | ES | 23,34 | | SE | 13,54 | | UK | 49,03 | </p><h4> <a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Ogrzewanie mieszkaniowe </h4><p> Wykorzystywana jest gęstość ciepła dla sektora mieszkaniowego. Ponieważ ten zestaw danych zawiera już energię zużytą na ciepłą wodę, poprzednia obliczona energia ciepłej wody jest od niej odejmowana. Na podstawie identyfikatora Nuts2 ID dla każdego kafelka o gęstości ciepła przypisanej do użytkownika przypisany jest profil ogrzewania domowego. Ogrzewanie domu jest mnożone przez współczynnik ogrzewania domu, na wypadek, gdyby użytkownik chciał dostosować wartość. </p><h4> <a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Trzeci dopływ ciepłej wody </h4><p> Podobnie jak w przypadku ciepłej wody mieszkalnej, potrzebna energia jest obliczana na podstawie tabeli. Z identyfikatorem Nuts2 określonego obszaru przypisywany jest profil obciążenia. Trzeci dopływ ciepłej wody jest mnożony przez współczynnik trzeciorzędnego dopływu ciepłej wody na wypadek, gdyby użytkownik chciał dostosować wartość. </p><p> <em>Energia jednostkowa na obszar dostarczania ciepłej wody w budynkach trzeciorzędnych w różnych krajach</em> </p><p> | Nuts0 ID | Zaopatrzenie w ciepłą wodę w kWh / m² / rok | | ------------- |: -------------: | | AT | 6,57 | | BE | 13,88 | | BG | 15,88 | | HR | 9,42 | | CY | 6,26 | | CZ | 9,18 | | DK | 8,03 | | EE | 14,13 | | FI | 10,52 | | FR | 9,57 | | DE | 3,05 | | EL | 6,99 | | HU | 9,51 | | IE | 10,87 | | IT | 5,62 | | LV | 7,16 | | LT | 10,46 | | LU | 7,20 | | MT | 10,45 | | NL | 6,89 | | PL | 9,55 | | PT | 21,47 | | RO | 13,85 | | SK | 8,49 | | SI | 27,73 | | ES | 12,44 | | SE | 19,62 | | UK | 13,45 | </p><h4> <a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Trzecie ogrzewanie </h4><p> Trzecie ogrzewanie oblicza się podobnie jak ogrzewanie domowe. Trzecie ogrzewanie jest mnożone przez trzeciorzędny współczynnik ogrzewania w przypadku, gdy użytkownik chce dostosować wartość. </p><h4> <a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Zapotrzebowanie na ciepło przemysłowe </h4><p> Zapotrzebowanie na ciepło według branży jest sprawdzane w przemysłowej bazie danych i na podstawie identyfikatora Nuts0 i podsektora przypisany jest profil obciążenia. </p><h4> <a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Podsumowanie </h4><p> Pięć powstałych profili dodaje się razem w sposób absolutny. </p><h3> <a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Realizacja </h3><h4> <a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Załaduj profile </h4><p> Wspomniane profile obciążeń składają się z 8760 punktów, które reprezentują obciążenie na każdą godzinę z 365 dni. Więcej informacji na temat <strong><a href="https://gitlab.com/hotmaps/load_profile">profili obciążeń można znaleźć tutaj.</a></strong> Wszystkie profile obciążenia są znormalizowane, dzięki czemu całka jest równa 1. </p><h4> <a class="anchor" id="industrial-sites" href="#industrial-sites"><i class="fa fa-link"></i></a> Witryny przemysłowe </h4><p> Źródła ciepła są pobierane z <strong><a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">przemysłowej bazy danych.</a></strong> Na podstawie ich nadmiaru ciepła, identyfikatora Nuts0 i sektora przemysłowego tworzony jest profil obciążenia obejmujący każdą godzinę roku dla każdej lokalizacji. </p><h4> <a class="anchor" id="heat-densities" href="#heat-densities"><i class="fa fa-link"></i></a> Gęstości cieplne </h4><p> Sektor mieszkaniowy gęstości ciepła i sektor mieszkaniowy gęstości ciepła to pliki rastrowe, które można znaleźć <strong><a href="https://gitlab.com/hotmaps/heat">tutaj.</a></strong> </p><h4> <a class="anchor" id="gross-floor-areas" href="#gross-floor-areas"><i class="fa fa-link"></i></a> Powierzchnia brutto </h4><p> Powierzchnia mieszkalna brutto i powierzchnia mieszkalna brutto niemieszkalna to pliki rastrowe, które można znaleźć <strong><a href="https://gitlab.com/hotmaps/gfa_res_curr_density">tutaj</a></strong> i <strong><a href="https://gitlab.com/hotmaps/gfa_nonres_curr_density">tutaj.</a></strong> </p><h2> <a class="anchor" id="sample-run" href="#sample-run"><i class="fa fa-link"></i></a> Przykładowy przebieg </h2><p> Próbka uruchomiona w DK05. </p><figure><img alt="" src="https://github.com/HotMaps/hotmaps_wiki/blob/master/Images/cm_load_profile/sample_run.PNG"/><figcaption> <i>Próbka uruchomiona w DK05.
