Con l&#39;ID Nuts2 di un&#39;area specifica viene assegnato un profilo di carico. La fornitura di acqua calda residenziale viene moltiplicata per il fattore di fornitura di acqua calda residenziale nel caso in cui l&#39;utente desideri regolare il valore. </p><p> <em>Energia specifica per area per l&#39;approvvigionamento di acqua calda in edifici residenziali in diversi paesi</em> </p><p> | ID Nuts0 | Fornitura di acqua calda in kWh / m² / a | | ------------- |: -------------: | | AT | 21,67 | | BE | 31.95 | | BG | 12,93 | | Risorse umane | 21.38 | | CY | 8.80 | | CZ | 22,83 | | DK | 9.64 | | EE | 14.35 | | FI | 10.15 | | FR | 9.66 | | DE | 8.27 | | EL | 12.51 | | HU | 13.66 | | IE | 15.91 | | IT | 14.01 | | LV | 15.71 | | LT | 13.36 | | LU | 8.29 | | MT | 10.99 | | NL | 8.91 | | PL | 10.00 | | PT | 9.48 | | RO | 11,48 | | SK | 21.51 | | SI | 21,74 | | ES | 23.34 | | SE | 13.54 | | Regno Unito | 49.03 | </p><h4> <a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Riscaldamento residenziale </h4><p> Viene utilizzata la densità di calore per il settore residenziale. Poiché questo set di dati contiene già l&#39;energia utilizzata per l&#39;acqua calda, viene sottratta l&#39;energia dell&#39;acqua calda calcolata in precedenza. Sulla base dell&#39;ID Nuts2 viene assegnato un profilo di riscaldamento residenziale per ciascuna piastrella della densità di calore nella selezione utente. Il riscaldamento residenziale viene moltiplicato per il fattore di riscaldamento residenziale nel caso in cui l&#39;utente desideri regolare il valore. </p><h4> <a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Fornitura di acqua calda terziaria </h4><p> Simile alla fornitura di acqua calda residenziale, l&#39;energia necessaria viene calcolata con una tabella. Con l&#39;ID Nuts2 di un&#39;area specifica viene assegnato un profilo di carico. La fornitura di acqua calda terziaria viene moltiplicata per il fattore di fornitura di acqua calda terziaria nel caso in cui l&#39;utente desideri regolare il valore. </p><p> <em>Energia specifica per area per l&#39;approvvigionamento di acqua calda negli edifici terziari in diversi paesi</em> </p><p> | ID Nuts0 | Fornitura di acqua calda in kWh / m² / a | | ------------- |: -------------: | | AT | 6.57 | | BE | 13,88 | | BG | 15.88 | | Risorse umane | 9.42 | | CY | 6.26 | | CZ | 9.18 | | DK | 8.03 | | EE | 14.13 | | FI | 10.52 | | FR | 9.57 | | DE | 3.05 | | EL | 6.99 | | HU | 9.51 | | IE | 10.87 | | IT | 5.62 | | LV | 7.16 | | LT | 10.46 | | LU | 7,20 | | MT | 10.45 | | NL | 6,89 | | PL | 9.55 | | PT | 21,47 | | RO | 13,85 | | SK | 8.49 | | SI | 27,73 | | ES | 12,44 | | SE | 19.62 | | Regno Unito | 13,45 | </p><h4> <a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Riscaldamento terziario </h4><p> Il riscaldamento terziario è calcolato in modo simile al riscaldamento residenziale. Il riscaldamento terziario viene moltiplicato per il fattore di riscaldamento terziario nel caso in cui l&#39;utente desideri regolare il valore. </p><h4> <a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Domanda di calore industriale </h4><p> La domanda di calore da parte dell&#39;industria viene cercata nel database industriale e in base all&#39;ID Nuts0 e al sottosettore viene assegnato un profilo di carico. </p><h4> <a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Somma </h4><p> I cinque profili risultanti vengono sommati in modo assoluto. </p><h3> <a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Implementazione </h3><h4> <a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Carica profili </h4><p> I profili di carico menzionati consistono in 8760 punti che rappresentano il carico per ogni ora dei 365 giorni. Ulteriori informazioni sui <strong><a href="https://gitlab.com/hotmaps/load_profile">profili di carico sono disponibili qui.</a></strong> Tutti i profili di carico sono normalizzati in modo che l&#39;integrale sia uguale a 1. </p><h4> <a class="anchor" id="industrial-sites" href="#industrial-sites"><i class="fa fa-link"></i></a> Siti industriali </h4><p> Le fonti di calore sono prese dal <strong><a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">database industriale.</a></strong> Sulla base del calore in eccesso, dell&#39;ID Nuts0 e del settore industriale, viene creato un profilo di carico che copre ogni ora dell&#39;anno per ogni sito. </p><h4> <a class="anchor" id="heat-densities" href="#heat-densities"><i class="fa fa-link"></i></a> Densità di calore </h4><p> Il settore residenziale a densità di calore e il settore non residenziale a densità di calore sono file raster che possono essere trovati <strong><a href="https://gitlab.com/hotmaps/heat">qui.</a></strong> </p><h4> <a class="anchor" id="gross-floor-areas" href="#gross-floor-areas"><i class="fa fa-link"></i></a> Superfici lorde </h4><p> La superficie lorda residenziale e la superficie lorda non residenziale sono file raster che possono essere trovati <strong><a href="https://gitlab.com/hotmaps/gfa_res_curr_density">qui</a></strong> e <strong><a href="https://gitlab.com/hotmaps/gfa_nonres_curr_density">qui.</a></strong> </p><h2> <a class="anchor" id="sample-run" href="#sample-run"><i class="fa fa-link"></i></a> Esecuzione del campione </h2><p> Esecuzione del campione in DK05. </p><figure><img alt="" src="https://github.com/HotMaps/hotmaps_wiki/blob/master/Images/cm_load_profile/sample_run.PNG"/><figcaption> <i>Esecuzione del campione in DK05.
