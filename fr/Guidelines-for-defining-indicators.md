<h1><a class="anchor" id="guidelines-for-defining-indicators" href="#guidelines-for-defining-indicators"><i class="fa fa-link"></i></a>Lignes directrices pour la définition des indicateurs</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Table des matières</h2><ul><li> <a href="#indicators">Indicateurs</a><ul><li> <a href="#indicators_simple-indicators">Indicateurs simples</a></li><li> <a href="#indicators_cross-indicators">Indicateurs croisés</a></li></ul></li><li> <a href="#indicator-result">Résultat de l&#39;indicateur</a></li><li> <a href="#how-to-cite">Comment citer</a></li><li> <a href="#authors-and-reviewers">Auteurs et relecteurs</a></li><li> <a href="#license">Licence</a></li><li> <a href="#acknowledgement">Reconnaissance</a></li></ul><h2><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indicateurs</h2><p> Cette page explique comment gérer les indicateurs dans la boîte à outils Hotmaps. Un indicateur est une valeur affichée dans la boîte à outils Hotmaps côté client. L&#39;indicateur fait partie d&#39;un calque. Un indicateur est une valeur, pour une région spatiale spécifique, avec une unité. Un indicateur fait partie d&#39;un dictionnaire de couches. Un objet de couche contient un tableau nommé &quot;indicateurs&quot;. Ce tableau sera utilisé pour lister tous les indicateurs que vous souhaitez voir pour la couche.</p><pre> <code>layers = { &#39;heat_tot_curr_density_tif&#39;:{ &#39;tablename&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;from_indicator_name&#39;:&#39;stat_heat_tot_curr_density_tif&#39;, &#39;schema_scalelvl&#39;: &#39;stat&#39;, &#39;schema_hectare&#39;: &#39;geo&#39;, &#39;crs&#39;: &#39;3035&#39;, &#39;geo_column&#39;: &#39;geometry&#39;, &#39;table_type&#39;:&#39;raster&#39;, &#39;data_lvl&#39;:[&#39;NUTS 0&#39;,&#39;NUTS 1&#39;,&#39;NUTS 2&#39;,&#39;NUTS 3&#39;,&#39;LAU 2&#39;,&#39;Hectare&#39;], &#39;data_aggregated&#39;:True, &#39;scalelvl_column&#39;:&#39;&#39;, &#39;indicators&#39;:[ {&#39;table_column&#39;: &#39;sum&#39;, &#39;unit&#39;: &#39;MWh&#39;,&#39;indicator_id&#39;:&#39;consumption&#39;}, {&#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;,&#39;indicator_id&#39;:&#39;count_cell&#39;}, { &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;: &#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;: &#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;: &#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;: &#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; } ] } }</code></pre><ul><li> &#39;nom de la table&#39;</li></ul><p> Nom de la table DB. (Exemple: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &quot;from_indicator_name&quot;</li></ul><p> Nom de sous-table pour la sélection des indicateurs. <strong>Doit être unique.</strong> (Exemple: &#39;stat_heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;data_aggregated&#39;</li></ul><p> Les données sont-elles agrégées ou non (valeurs: True ou False)</p><ul><li> &#39;scalelvl_column&#39;</li></ul><p> Nom de la colonne de niveau d&#39;échelle s&#39;il est différent de celui par défaut (exemple: &#39;code&#39;)</p><ul><li> &#39;data_lvl&#39;</li></ul><p> Niveaux disponibles pour les données de la base de données</p><ul><li> &#39;schema_scalelvl&#39;</li></ul><p> Emplacement du schéma de table pour le niveau des écrous. (Exemple: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> &#39;schema_hectare&#39;</li></ul><p> Emplacement du schéma de table pour le niveau d&#39;hectare. (Exemple: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> «crs»</li></ul><p> Projection de la géométrie (Exemple: &#39;3035&#39;, &#39;4326&#39;, &#39;4258&#39;)</p><ul><li> &#39;geo_column&#39;</li></ul><p> Nom de la colonne de géométrie dans la base de données (Exemple: &#39;geom&#39;, &#39;geometry&#39;)</p><ul><li> &#39;table_type&#39;</li></ul><p> Type de couche dans la base de données (Valeurs: &#39;vector&#39; ou &#39;raster&#39;).</p><p> <em><strong>Important:</strong></em> s&#39;il s&#39;agit d&#39;un raster, les colonnes disponibles sont <strong>count, sum, mean, stddev, min et max</strong></p><ul><li> &#39;Indicateurs&#39;</li></ul><p> Il existe 2 types d&#39;indicateurs (indicateurs simples et indicateurs croisés).</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="simple-indicators" href="#simple-indicators"><i class="fa fa-link"></i></a> Indicateurs simples</h3><p> Un indicateur simple est un objet avec 3 paramètres.</p><pre> <code>{ &#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;, &#39;indicator_id&#39;:&#39;count_cell&#39; }</code></pre><ul><li> &#39;table_column&#39;</li></ul><p> Il s&#39;agit de la colonne du tableau qui est sélectionnée dans le tableau. (Exemple: &#39;count&#39;)</p><img alt="sélection de colonne de table" src="https://github.com/HotMaps/Hotmaps-toolbox-service/raw/master/api/assets/table_image.png"/><ul><li> &#39;unité&#39;</li></ul><p> C&#39;est l&#39;unité de l&#39;indicateur. (Exemple: &#39;cells&#39;, &#39;MWh&#39;)</p><ul><li> &#39;indicateur_id&#39;</li></ul><p> Il s&#39;agit de l&#39;identifiant de l&#39;indicateur (comme un identifiant). Ce nom <strong>doit être unique</strong> dans le tableau des indicateurs.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="cross-indicators" href="#cross-indicators"><i class="fa fa-link"></i></a> Indicateurs croisés</h3><p> Un indicateur croisé est un objet avec 7 paramètres. Le but de cet indicateur est de faire un calcul parmi des indicateurs simples.</p><pre> <code>{ &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;:&#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;:&#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;:&#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; }</code></pre><ul><li> &#39;reference_indicator_id_1&#39;</li></ul><p> Correspond à l&#39;identifiant d&#39;un indicateur simple. Ce nom <strong>doit être défini</strong> dans le tableau d&#39;indicateurs. C&#39;est la valeur numéro 1.</p><ul><li> &#39;reference_tablename_indicator_id_1&#39;</li></ul><p> Référence du nom de table du calque qui fait référence à la valeur numéro 1. (Exemple: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;opérateur&#39;</li></ul><p> Règle de calcul à appliquer aux 2 valeurs (Valeurs: &#39;/&#39; ou &#39;*&#39; ou &#39;+&#39; ou &#39;-&#39;)</p><ul><li> &#39;reference_indicator_id_1&#39;</li></ul><p> Correspond à l&#39;identifiant d&#39;un indicateur simple. Ce nom <strong>doit être défini</strong> dans le tableau d&#39;indicateurs. C&#39;est la valeur numéro 2.</p><ul><li> &#39;reference_tablename_indicator_id_2&#39;</li></ul><p> Référence du nom de table du calque qui fait référence à la valeur numéro 2. (Exemple: &#39;pop_tot_curr_density_tif&#39;)</p><ul><li> &#39;unité&#39;</li></ul><p> C&#39;est l&#39;unité de l&#39;indicateur. (Exemple: &#39;cells&#39;, &#39;MWh&#39;)</p><ul><li> &#39;Nom&#39;</li></ul><p> C&#39;est le nom de l&#39;indicateur (comme un identifiant). Ce nom <strong>doit être unique</strong> dans le tableau des indicateurs.</p><h5><a class="anchor" id="note--for-this-example,-the-calculation-below-is-done." href="#note--for-this-example,-the-calculation-below-is-done."><i class="fa fa-link"></i></a> Remarque: Pour cet exemple, le calcul ci-dessous est effectué.</h5><pre> <code>reference_tablename_indicator_id_1.reference_indicator_id_1 / reference_tablename_indicator_id_2.reference_indicator_id_2 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code></pre><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="indicator-result" href="#indicator-result"><i class="fa fa-link"></i></a> Résultat de l&#39;indicateur</h2><p> Le résultat des indicateurs est le suivant:</p><pre> <code>{ &quot;values&quot;: [ { &quot;unit&quot;: &quot;MWh&quot;, &quot;name&quot;: &quot;heat_tot_curr_density_tif_consumption&quot;, &quot;value&quot;: &quot;4112030.46&quot;
