<h1> <a class="anchor" id="guidelines-for-defining-indicators" href="#guidelines-for-defining-indicators"><i class="fa fa-link"></i></a> Richtlinien zur Definition von Indikatoren </h1><h2> <a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Inhaltsverzeichnis </h2><ul><li> <a href="#indicators">Indikatoren</a> <ul><li> <a href="#indicators_simple-indicators">Einfache Indikatoren</a> </li><li> <a href="#indicators_cross-indicators">Kreuzindikatoren</a> </li></ul></li><li> <a href="#indicator-result">Indikatorergebnis</a> </li><li> <a href="#how-to-cite">Wie zu zitieren</a> </li><li> <a href="#authors-and-reviewers">Autoren und Rezensenten</a> </li><li> <a href="#license">Lizenz</a> </li><li> <a href="#acknowledgement">Wissen</a> </li></ul><h2> <a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indikatoren </h2><p> Auf dieser Seite wird erläutert, wie Sie Indikatoren in der Hotmaps-Toolbox verwalten. Ein Indikator ist ein Wert, der in der Hotmaps-Toolbox auf der Clientseite angezeigt wird. Der Indikator ist Teil einer Ebene. Ein Indikator ist ein Wert für eine bestimmte räumliche Region mit einer Einheit. Ein Indikator ist Teil eines Ebenenwörterbuchs. Ein Ebenenobjekt enthält ein Array mit dem Namen &quot;Indikatoren&quot;. In dieser Tabelle werden alle Indikatoren aufgelistet, die für die Ebene angezeigt werden sollen. </p><pre> <code>layers = { &#39;heat_tot_curr_density_tif&#39;:{ &#39;tablename&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;from_indicator_name&#39;:&#39;stat_heat_tot_curr_density_tif&#39;, &#39;schema_scalelvl&#39;: &#39;stat&#39;, &#39;schema_hectare&#39;: &#39;geo&#39;, &#39;crs&#39;: &#39;3035&#39;, &#39;geo_column&#39;: &#39;geometry&#39;, &#39;table_type&#39;:&#39;raster&#39;, &#39;data_lvl&#39;:[&#39;NUTS 0&#39;,&#39;NUTS 1&#39;,&#39;NUTS 2&#39;,&#39;NUTS 3&#39;,&#39;LAU 2&#39;,&#39;Hectare&#39;], &#39;data_aggregated&#39;:True, &#39;scalelvl_column&#39;:&#39;&#39;, &#39;indicators&#39;:[ {&#39;table_column&#39;: &#39;sum&#39;, &#39;unit&#39;: &#39;MWh&#39;,&#39;indicator_id&#39;:&#39;consumption&#39;}, {&#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;,&#39;indicator_id&#39;:&#39;count_cell&#39;}, { &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;: &#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;: &#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;: &#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;: &#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; } ] } }</code> </pre><ul><li> &#39;Tabellenname&#39; </li></ul><p> Name der DB-Tabelle. (Beispiel: &#39;heat_tot_curr_density_tif&#39;) </p><ul><li> &#39;from_indicator_name&#39; </li></ul><p> Untertitelname für die Auswahl der Indikatoren. <strong>Muss einzigartig sein.</strong> (Beispiel: &#39;stat_heat_tot_curr_density_tif&#39;) </p><ul><li> &#39;data_aggregated&#39; </li></ul><p> Sind die Daten aggregiert oder nicht (Werte: Richtig oder falsch) </p><ul><li> &#39;scalelvl_column&#39; </li></ul><p> Spaltenname auf Skalenebene, falls abweichend vom Standardspaltennamen (Beispiel: &#39;Code&#39;) </p><ul><li> &#39;data_lvl&#39; </li></ul><p> Für die Daten in der Datenbank verfügbare Ebenen </p><ul><li> &#39;schema_scalelvl&#39; </li></ul><p> Position des Tabellenschemas für die Mutternebene. (Beispiel: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;) </p><ul><li> &#39;schema_hectare&#39; </li></ul><p> Position des Tabellenschemas für die Hektarebene. (Beispiel: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;) </p><ul><li> &#39;crs&#39; </li></ul><p> Projektion der Geometrie (Beispiel: &#39;3035&#39;, &#39;4326&#39;, &#39;4258&#39;) </p><ul><li> &#39;geo_column&#39; </li></ul><p> Name der Geometriespalte in der Datenbank (Beispiel: &#39;Geom&#39;, &#39;Geometrie&#39;) </p><ul><li> &#39;table_type&#39; </li></ul><p> Typ der Ebene in der Datenbank (Werte: &#39;Vektor&#39; oder &#39;Raster&#39;). </p><p> <em><strong>Wichtig:</strong></em> Wenn es sich um ein Raster handelt, sind die verfügbaren Spalten <strong>count, sum, mean, stddev, min und max</strong> </p><ul><li> &#39;Indikatoren&#39; </li></ul><p> Es gibt zwei Arten von Indikatoren (einfache Indikatoren und Kreuzindikatoren). </p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a> </p> <h3> <a class="anchor" id="simple-indicators" href="#simple-indicators"><i class="fa fa-link"></i></a> Einfache Indikatoren </h3><p> Ein einfacher Indikator ist ein Objekt mit 3 Parametern. </p><pre> <code>{ &#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;, &#39;indicator_id&#39;:&#39;count_cell&#39; }</code> </pre><ul><li> &#39;table_column&#39; </li></ul><p> Dies ist die Tabellenspalte, die in der Tabelle ausgewählt ist. (Beispiel: &#39;count&#39;) </p><p><img alt="Tabellenauswahl" src="https://github.com/HotMaps/Hotmaps-toolbox-service/raw/master/api/assets/table_image.png"/></p><ul><li> &#39;Einheit&#39; </li></ul><p> Dies ist die Einheit des Indikators. (Beispiel: &#39;Zellen&#39;, &#39;MWh&#39;) </p><ul><li> &#39;Indikator_ID&#39; </li></ul><p> Dies ist die Indikator-ID des Indikators (wie eine ID). Dieser Name <strong>muss</strong> im Array der Indikatoren <strong>eindeutig sein</strong> . </p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a> </p> <h3> <a class="anchor" id="cross-indicators" href="#cross-indicators"><i class="fa fa-link"></i></a> Kreuzindikatoren </h3><p> Ein Kreuzindikator ist ein Objekt mit 7 Parametern. Das Ziel dieses Indikators ist es, eine Berechnung unter einfachen Indikatoren durchzuführen. </p><pre> <code>{ &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;:&#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;:&#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;:&#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; }</code> </pre><ul><li> &#39;reference_indicator_id_1&#39; </li></ul><p> Entspricht der Kennung eines einfachen Indikators. Dieser Name <strong>muss</strong> im Indikatorarray <strong>definiert</strong> werden. Es ist Wert Nummer 1. </p><ul><li> &#39;reference_tablename_indicator_id_1&#39; </li></ul><p> Referenz des Layer-Tabellennamens, der auf den Wert Nummer 1 verweist. (Beispiel: &#39;heat_tot_curr_density_tif&#39;) </p><ul><li> &#39;Operator&#39; </li></ul><p> Berechnungsregel für die 2 Werte (Werte: &#39;/&#39; oder &#39;*&#39; oder &#39;+&#39; oder &#39;-&#39;) </p><ul><li> &#39;reference_indicator_id_1&#39; </li></ul><p> Entspricht der Kennung eines einfachen Indikators. Dieser Name <strong>muss</strong> im Indikatorarray <strong>definiert</strong> werden. Es ist Wert Nummer 2. </p><ul><li> &#39;reference_tablename_indicator_id_2&#39; </li></ul><p> Referenz des Layer-Tabellennamens, der auf den Wert Nummer 2 verweist. (Beispiel: &#39;pop_tot_curr_density_tif&#39;) </p><ul><li> &#39;Einheit&#39; </li></ul><p> Dies ist die Einheit des Indikators. (Beispiel: &#39;Zellen&#39;, &#39;MWh&#39;) </p><ul><li> &#39;Name&#39; </li></ul><p> Dies ist der Name des Indikators (wie eine ID). Dieser Name <strong>muss</strong> im Array der Indikatoren <strong>eindeutig sein</strong> . </p><h5> <a class="anchor" id="note--for-this-example,-the-calculation-below-is-done." href="#note--for-this-example,-the-calculation-below-is-done."><i class="fa fa-link"></i></a> Hinweis: In diesem Beispiel wird die folgende Berechnung durchgeführt. </h5><pre> <code>reference_tablename_indicator_id_1.reference_indicator_id_1 / reference_tablename_indicator_id_2.reference_indicator_id_2 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code> </pre><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a> </p> <h2> <a class="anchor" id="indicator-result" href="#indicator-result"><i class="fa fa-link"></i></a> Indikatorergebnis </h2><p> Das Ergebnis der Indikatoren ist wie folgt: </p><pre> <code>{ &quot;values&quot;: [ { &quot;unit&quot;: &quot;MWh&quot;, &quot;name&quot;: &quot;heat_tot_curr_density_tif_consumption&quot;, &quot;value&quot;: &quot;4112030.46&quot;
