href="#acknowledgement">Anerkendelse</a></li></ul><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introduktion</h2><p> Databasen, der er bygget til Hotmaps, er et rumligt datavarehus, der er gjort tilgængeligt for platformen til visualisering og beregning.</p><p> Der er forskellige slags data:</p><ul><li> raster datasæt (på hektar niveau)</li><li> vektordatasæt (NUTS / LAU-regioner, interessepunkter, ...)</li><li> ikke-rumlige datasæt (for det meste NUTS / LAU-relateret og knyttet til eksisterende vektorlag)</li></ul><p> Du kan få adgang til datasættene via en geospatial server (GeoServer), der leverer forskellige tjenester (WMS / WFS / ...), der giver enhver mulighed for at vise, inkludere eller bruge disse datasæt.</p><p> <a href="https://gitlab.com/hotmaps">Datasættene</a> kan findes på <a href="https://gitlab.com/hotmaps">Gitlab-arkivet</a></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="query" href="#query"><i class="fa fa-link"></i></a> Forespørgsel</h2><p> Bemærk, at aggregeringer, når du vælger NUTS- eller LAU-regioner, er forudberegnet og derfor hurtigere end aggregeringer af hektarvalg, der beregnes on-the-fly.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indikatorer</h2><p> Indikatorer er defineret <strong><a href="https://github.com/HotMaps/Hotmaps-toolbox-service/blob/develop/api/app/models/indicators.py">her</a></strong> . Afhængigt af lagene er forskellige statistiske parametre såsom minimumsværdi, maksimumværdi og middelværdi blandt indikatorerne. En forklaring på definition af indikatorer for de lag, der er integreret i Hotmaps-databasen, kan findes <strong><a href="https://github.com/HotMaps/Hotmaps-toolbox-service/blob/develop/api/app/models/INDICATORS.md">her</a></strong> .</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="technologies" href="#technologies"><i class="fa fa-link"></i></a> Teknologier</h2><p> De anvendte teknologier til databasen er:</p><ul><li> PostgreSQL version 9.6</li><li> PostGIS version 2.3</li><li> GeoServer version 2.11.2</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="references" href="#references"><i class="fa fa-link"></i></a> Referencer</h2><ul><li> Simon Pezzutto, Stefano Zambotti, Silvia Croce, Pietro Zambelli, Giulia Garegnani, Chiara Scaramuzzino, Ramón Pascual Pascuas, Alyona Zubaryeva, Franziska Haas, Dagmar Exner (EURAC), Andreas Mueller (e-think), Michael Hartner (TUW), Tobias Fleiter , Anna-Lena Klingler, Matthias Kühnbach, Pia Manz, Simon Marwitz, Matthias Rehfeldt, Jan Steinbach, Eftim Popovski (Fraunhofer ISI) Bedømt af Lukas Kranzl, Sara Fritz (TUW) Hotmaps-projekt, <strong>D2.3 WP2-rapport -
