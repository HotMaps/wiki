Ein Indikator ist ein Wert, der in der Hotmaps-Toolbox auf der Clientseite angezeigt wird. Der Indikator ist Teil einer Ebene. Der Indikator ist ein Wert für einen bestimmten räumlichen Bereich mit einer Einheit. Ein Indikator ist Teil eines Ebenenwörterbuchs. Ein Layer-Objekt enthält ein Array mit dem Namen "Indikatoren". In dieser Tabelle werden alle Indikatoren aufgelistet, die Sie für den Layer anzeigen möchten. </p><pre> <code>layers = { 'heat_tot_curr_density_tif':{ 'tablename':'heat_tot_curr_density_tif', 'from_indicator_name':'stat_heat_tot_curr_density_tif', 'schema_scalelvl': 'stat', 'schema_hectare': 'geo', 'crs': '3035', 'geo_column': 'geometry', 'table_type':'raster', 'data_lvl':['NUTS 0','NUTS 1','NUTS 2','NUTS 3','LAU 2','Hectare'], 'data_aggregated':True, 'scalelvl_column':'', 'indicators':[ {'table_column': 'sum', 'unit': 'MWh','indicator_id':'consumption'}, {'table_column': 'count', 'unit': 'cells','indicator_id':'count_cell'}, { 'reference_indicator_id_1': 'consumption', 'reference_tablename_indicator_id_1': 'heat_tot_curr_density_tif', 'operator': '/', 'reference_indicator_id_2': 'count_cell', 'reference_tablename_indicator_id_2': 'pop_tot_curr_density_tif', 'unit':'MWh/person', 'indicator_id': 'heat_tot_curr_density_tif_per_pop_tot_curr_density_tif' } ] } }</code> </pre><ul><li> 'Tabellenname' </li></ul><p> Name der DB-Tabelle. (Beispiel: 'heat_tot_curr_density_tif') </p><ul><li> 'from_indicator_name' </li></ul><p> Subtabellenname für die Auswahl der Indikatoren. <strong>Muss einzigartig sein.</strong> (Beispiel: 'stat_heat_tot_curr_density_tif') </p><ul><li> 'data_aggregated' </li></ul><p> Sind die Daten aggregiert oder nicht (Werte: Richtig oder Falsch) </p><ul><li> 'scalelvl_column' </li></ul><p> Spaltenname der Skalierungsebene, falls abweichend von der Standardspalte (Beispiel: "Code") </p><ul><li> 'data_lvl' </li></ul><p> Für die Daten in der Datenbank verfügbare Ebenen </p><ul><li> 'schema_scalelvl' </li></ul><p> Speicherort des Tabellenschemas für die Nuts-Ebene. (Beispiel: 'geo', 'stat', 'public') </p><ul><li> 'schema_hectare' </li></ul><p> Speicherort des Tabellenschemas für die Hektarebene. (Beispiel: 'geo', 'stat', 'public') </p><ul><li> 'crs' </li></ul><p> Projektion der Geometrie (Beispiel: '3035', '4326', '4258') </p><ul><li> 'geo_column' </li></ul><p> Name der Geometriespalte in der Datenbank (Beispiel: 'geom', 'geometry') </p><ul><li> 'table_type' </li></ul><p> Typ des Layers in der Datenbank (Werte: 'Vektor' oder 'Raster'). </p><p> <em><strong>Wichtig:</strong></em> Wenn es sich um ein Raster handelt, stehen die Spalten <strong>count, sum, mean, stddev, min und max zur Verfügung</strong> </p><ul><li> "Indikatoren" </li></ul><p> Es gibt zwei Arten von Indikatoren (einfache Indikatoren und Kreuzindikatoren). </p><h3> Einfacher Indikator </h3><p> Ein einfacher Indikator ist ein Objekt mit 3 Parametern. </p><pre> <code>{ 'table_column': 'count', 'unit': 'cells', 'indicator_id':'count_cell' }</code> </pre><ul><li> 'table_column' </li></ul><p> Dies ist die in der Tabelle ausgewählte Tabellenspalte. (Beispiel: 'count') </p><p><img alt="tabellenspaltenauswahl" src="/api/assets/table_image.png"/></p><ul><li> 'Einheit' </li></ul><p> Dies ist die Einheit des Indikators. (Beispiel: "Zellen", "MWh") </p><ul><li> 'indicator_id' </li></ul><p> Dies ist die Indikator-ID des Indikators (wie eine ID). Dieser Name <strong>muss</strong> im Array des Indikators <strong>eindeutig sein</strong> . </p><h3> Kreuzanzeige </h3><p> Ein Kreuzindikator ist ein Objekt mit 7 Parametern. Das Ziel dieses Indikators ist es, eine Berechnung zwischen einfachen Indikatoren und Kreuzindikatoren durchzuführen. </p><pre> <code>{ 'reference_indicator_id_1': 'consumption', 'reference_tablename_indicator_id_1':'heat_tot_curr_density_tif', 'operator': '/', 'reference_indicator_id_2':'count_cell', 'reference_tablename_indicator_id_2':'pop_tot_curr_density_tif', 'unit':'MWh/person', 'indicator_id':'heat_tot_curr_density_tif_per_pop_tot_curr_density_tif' }</code> </pre><ul><li> 'reference_indicator_id_1' </li></ul><p> Entspricht der Kennung eines einfachen Indikators. Dieser Name <strong>muss</strong> im Indikator-Array definiert werden. Es ist der Wert Nummer 1. </p><ul><li> 'reference_tablename_indicator_id_1' </li></ul><p> Referenz des Layer-Tabellennamens, der auf den Wert 1 verweist. (Beispiel: 'heat_tot_curr_density_tif') </p><ul><li> 'Operator' </li></ul><p> Berechnungsregel für die 2 Werte (Werte: '/' oder '*' oder '+' oder '-') </p><ul><li> 'reference_indicator_id_1' </li></ul><p> Entspricht der Kennung eines einfachen Indikators. Dieser Name <strong>muss</strong> im Indikator-Array definiert werden. Es ist der Wert Nummer 2. </p><ul><li> 'reference_tablename_indicator_id_2' </li></ul><p> Referenz des Layer-Tabellennamens, der auf den Wert 2 verweist. (Beispiel: 'pop_tot_curr_density_tif') </p><ul><li> 'Einheit' </li></ul><p> Dies ist die Einheit des Indikators. (Beispiel: "Zellen", "MWh") </p><ul><li> 'Name' </li></ul><p> Dies ist der Name des Indikators (wie eine ID). Dieser Name <strong>muss</strong> im Array des Indikators <strong>eindeutig sein</strong> . </p><h5> Hinweis: In diesem Beispiel ist die Berechnung wie folgt ausgeführt. </h5><pre> <code>reference_indicator_id_1.reference_indicator_id_1 / reference_indicator_id_1.reference_indicator_id_1 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code> </pre><h2> Indikator Ergebnis </h2><p> Das Ergebnis der Indikatoren lautet wie folgt: </p><pre> <code>{ "values": [ { "unit": "MWh", "name": "heat_tot_curr_density_tif_consumption", "value": "4112030.46" }, { "unit": "cells", "name": "heat_tot_curr_density_tif_count_cell", "value": "46764" }, { "unit": "MWh/person", "name": "heat_tot_curr_density_tif_per_pop_tot_curr_density_tif", "value": "38.0092476775893146" } ], "name": "heat_tot_curr_density_tif" }</code> </pre><h2> Verweise </h2><h2> Wie man zitiert </h2><p> Mostafa Fallahnejad, im Hotmaps-Wiki, https://github.com/HotMaps/hotmaps_wiki/wiki/Guidelines-for-defining-indicators (April 2019) </p><h2> Autoren und Gutachter </h2><p> Diese Seite wurde von Mostafa Fallahnejad * geschrieben. </p><ul><li> [] Diese Seite wurde von Lukas Kranzl * rezensiert. </li></ul><p> * <a href="https://eeg.tuwien.ac.at/">Arbeitsgruppe Energiewirtschaft -
