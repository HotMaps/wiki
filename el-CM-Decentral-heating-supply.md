<h2> Πίνακας περιεχομένων </h2><ul><li> <a href="#introduction">Εισαγωγή</a> </li><li> <a href="#inputs-and-outputs">Είσοδοι και έξοδοι</a> </li><li> <a href="#method">Μέθοδος</a> </li><li> <a href="#GitHub-Repository-of-this-calculation-module">Αποθήκη GitHub αυτής της υπομονάδας υπολογισμού</a> </li><li> <a href="#sample-run">Εκτέλεση δείγματος</a> <ul><li> <a href="#test-run-1-single-family-house-existing-building">Τρέχουσα δοκιμή 1: μονοκατοικία, υπάρχον κτίριο</a> </li><li> <a href="#test-run-2-multi---family-house-new-building">Δοκιμή 2: Πολυκατοικία, νέο κτίριο</a> </li></ul></li><li> <a href="#references">βιβλιογραφικές αναφορές</a> </li><li> <a href="#how-to-cite">Πώς να αναφέρω</a> </li><li> <a href="#authors-and-reviewers">Συγγραφείς και κριτές</a> </li><li> <a href="#license">Αδεια</a> </li><li> <a href="#acknowledgement">Αναγνώριση</a> </li></ul><h2> Εισαγωγή </h2><p> Η μονάδα υπολογισμού υπολογισμού υπολογισμού κόστους θέρμανσης (CM) υπολογίζει το επίπεδο κόστους της θερμότητας (LCOH) για τις διάφορες τεχνολογίες στον τομέα των κατοικιών ή των υπηρεσιών. <strong>Αυτή η ενότητα μπορεί να εκτελεστεί για την περιοχή NUTS 3.</strong> Οι τεχνολογίες είναι οι εξής: </p><ul><li> Η αντλία θερμότητας αέρα-αέρα, αέρα-σε-νερό και αλατούχα-σε-νερό, </li><li> Ηλεκτρική θερμάστρα, </li><li> Boiler βιο-πετρελαίου, </li><li> Λέβητα πετρελαίου, </li><li> Αυτόματη και χειροκίνητη βιομάζα, </li><li> Ξυλόσομπα, </li><li> Φυσικό αέριο, </li><li> Ηλιακή θερμική </li></ul><p> Τα αποτελέσματα θα εμφανίζονται για τον επιλεγμένο τύπο κτιρίου και κατηγορία κτιρίων. </p><p> 🔺 <mark> <strong>Σημείωση 1:</strong> Η τρέχουσα έκδοση στην εργαλειοθήκη είναι μια δοκιμαστική έκδοση. Σκεφτόμαστε να χρησιμοποιήσουμε πρόσθετα δεδομένα και να επεκτείνουμε τη λειτουργικότητα αυτής της ενότητας υπολογισμού. </mark></p><p> 🔺 <mark> <strong>Σημείωση 2:</strong> Λόγω του ευρέος φάσματος των μεθόδων φορολόγησης και των επιστροφών φόρου σε διάφορες χώρες, δεν εξετάζεται σε αυτή τη μονάδα υπολογισμού. </mark></p><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code> </ins> </p><h2> Είσοδοι και έξοδοι </h2><p> Οι παράμετροι εισόδου και τα επίπεδα, καθώς και τα επίπεδα και οι παράμετροι εξόδου είναι οι εξής. </p><p> <strong>Οι παράμετροι εισαγωγής είναι:</strong> </p><ul><li> Ακαθάριστο εμβαδόν δαπέδου του κτιρίου σε [ <em><strong>m <sup>2</sup></strong></em> ]: μια τιμή μεταξύ <em><em>1</em></em> και <em><em>10000</em></em> </li><li> Επιτόκιο: μια τιμή μεταξύ <em><em>0</em></em> και <em><em>1</em></em> </li><li> Τομέας: κατοικίες ή υπηρεσίες (🔺 Επί του παρόντος υποστηρίζονται μόνο κατοικίες) </li><li> Τύπος κτιρίου: μονοκατοικία ή πολυκατοικία </li><li> Κτίριο κατηγορίας: υπάρχον, ανακαινισμένο ή νέο κτίριο </li><li> Τύπος ζήτησης: θέρμανση ή ψύξη (🔺 Σήμερα υποστηρίζεται μόνο θέρμανση) </li><li> Έτος: (🔺 Επί του παρόντος, υποστηρίζεται μόνο για το έτος 2015) </li></ul><p> <strong>Τα αποτελέσματα είναι:</strong> </p><ul><li> γραφικές παραστάσεις </li><li> δείκτες </li></ul><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code> </ins> </p><h2> Μέθοδος </h2><p> Στο πρώτο βήμα, με βάση την επιλεγμένη ζώνη, εξάγεται από τη βάση δεδομένων hotmap η ειδική ζήτηση θέρμανσης για τον επιλεγμένο τομέα, τον τύπο κτιρίου, τον τύπο κτιρίου και τον τύπο της ζήτησης. Παράμετροι που σχετίζονται με το κόστος του συστήματος θέρμανσης σε επιλεγμένη ζώνη που λαμβάνεται επίσης από τη βάση δεδομένων Hotmaps. Με τον πολλαπλασιασμό της ειδικής ζήτησης θέρμανσης και της ακαθάριστης επιφάνειας δαπέδου του κτιρίου, επιτυγχάνεται η ζήτηση ενέργειας. Στη συνέχεια, για τον υπολογισμό του επιπέδου κόστους της θερμότητας, θα πρέπει να ληφθούν τα εξής βήματα: </p><pre> <code># final energy demand [kWh] final_energy_demand = energy_demand / efficiency_heating_system # OPEX: Operational Expenditure (EUR) OPEX = fix_operation_and_maintenance * heat_load + variable_operation_and_maintenance * final_energy_demand # CAPEX: Capital Expenditure (EUR) CAPEX = heat_load * specific_investment_cost_of_heating_system * annuity_factor # energy costs (EUR) energy_costs = final_energy_demand * energy_price # total costs heat supply (EUR) total_costs = OPEX + CAPEX + energy_costs # levelized cost of heat (LCOH) [EUR/kWh] LCOH = total_costs / energy_demand</code> </pre><p> Η παραπάνω διαδικασία εκτελείται για όλα τα συστήματα θέρμανσης που αναφέρονται στην ενότητα <a href="#Introduction">Εισαγωγή</a> . Η περίληψη των αποτελεσμάτων παρουσιάζεται με τη μορφή γραφημάτων και δεικτών στην εργαλειοθήκη. </p><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code> </ins> </p><h2> Αποθήκη GitHub αυτής της υπομονάδας υπολογισμού </h2><p> <a href="https://github.com/HotMaps/lcoh/tree/develop">Εδώ</a> εμφανίζεται η εξέλιξη της αιμορραγίας για αυτή τη μονάδα υπολογισμού. </p><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code> </ins> </p><h2> Εκτέλεση δείγματος </h2><p> Εδώ, η μονάδα υπολογισμού λειτουργεί για ένα κτίριο με συνολική επιφάνεια <em>100 m <sup>2</sup></em> στη Φρανκφούρτη φορά για μια μονοκατοικία, υφιστάμενη κατηγορία κτιρίου και μία φορά για ένα σπίτι πολλών οικογενειών, νέα κατηγορία κτιρίου. </p><p> Αρχικά, χρησιμοποιήστε τη γραμμή &quot;Μετάβαση στη θέση&quot;
