Vastuuvapauslausekkeen lisäksi tarjotaan lisätietoja tuetuista selaimista, Hotmaps-projektin tavoitteista, linkki Hotmaps-projektin verkkosivustolle ja tietovarastoihin.</p><p><a name="Fig1"><img alt="intro_gif" src="../images/general_tool_functionalities_and_structure/intro.gif"/></a> <em>Kuva 1 Lyhyt esittely Hotmaps-työkalupakista</em></p><p> Hotmaps-käyttöliittymä on GIS-pohjainen käyttöliittymä. Sulkemalla vastuuvapauslausekkeen käyttäjä näkee Euroopan kartan. Oletusarvoisesti kuvataan EU-28-maiden lämmönkysyntätiheyskartta ja NUTS 2 -rajat. Näiden kahden kartan lisäksi käyttöliittymässä näkyy joitain työkaluja ja painikkeita. Nämä työkalut on kuvattu seuraavassa kuvassa.</p><p><a name="Fig2"><img alt="ensi silmäys" src="../images/general_tool_functionalities_and_structure/gui_navigation.png"/></a> <em>Kuva 2 Aloitussivu</em></p><p> Täältä löydät ensi silmäyksellä 3 käyttöliittymän osaa:</p><ol><li> <a href="#upper-toolbar">työkalurivi vasemmassa yläkulmassa</a> ,</li><li> itse kartta,</li><li> <a href="#tiles">joitain työkaluja kartan tyylin muuttamiseen</a> .</li><li> kaksi linkkiä oikeassa yläkulmassa: tähän <a href="https://wiki.hotmaps.eu/">Hotmaps Wikiin</a> ja <a href="https://www.hotmaps-project.eu/">Hotmaps Project -sivustoon</a> .</li></ol><p> Seuraavissa luvuissa kolme ensimmäistä kohtaa selitetään yksityiskohtaisesti.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="upper-toolbar" href="#upper-toolbar"><i class="fa fa-link"></i></a> Ylempi työkalupalkki</h2><p><img alt="ensi silmäys" src="../images/general_tool_functionalities_and_structure/toolbar_up.png"/></p><p> Tällä työkalurivillä voit:</p><ol><li> <a href="#upper-toolbar_connect"><em>Yhdistä:</em></a> rekisteröidy ja kirjaudu verkkosovellukseen työn tallentamiseksi,</li><li> <a href="#upper-toolbar_go-to-place"><em>Siirry paikkaan:</em></a> lähennä tietylle alueelle kirjoittamalla nimi,</li><li> <a href="#upper-toolbar_layers"><em>Tasot:</em></a> näytä kerroksen sivupalkki,</li><li> <a href="#upper-toolbar_selection-tools"><em>Valintatyökalut</em> :</a> ota käyttöön tai poista käytöstä valintatyökalut,</li><li> <a href="#upper-toolbar_show-result"><em>Näytä tulos:</em></a> näytä valitsemiesi alueiden sivupalkki,</li><li> <a href="#upper-toolbar_feedback"><em>Palaute:</em></a> anna meille palautetta työkalusta,</li><li> <a href="#upper-toolbar_save-session"><em>Tallenna istunto:</em></a> voit tallentaa valitut tasot ja zoomaustason ja hakea ne myöhemmin,</li><li> <a href="#upper-toolbar_save-session-folder"><em>Tallenna istuntokansio:</em></a> josta näet luettelon kaikista tallennetuista istunnoista.</li></ol><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="connect" href="#connect"><i class="fa fa-link"></i></a> Kytkeä</h3><p> Täällä voit:</p><ul><li> Luo tili</li><li> Kirjaudu sisään tiliisi tallentaaksesi edistymisesi</li><li> Tai nollaa salasanasi, jos unohdat sen</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h4><a class="anchor" id="login" sähköpostiosoitteellasi ja salasanallasi (katso alla oleva kuva <a href="#fig4">4</a> ).</p><p><img alt="login_gif" src="../images/general_tool_functionalities_and_structure/login.gif"/></p><p><a <em>hotmaps-</em> verkkosivustolle. Kun olet lähettänyt <a href="#fig5">lomakkeen</a> , saat sähköpostiviestin tilin aktivoimiseksi. Tililläsi voit tallentaa edistymisesi.</p><p><img alt="register_gif" src="../images/general_tool_functionalities_and_structure/register.gif"/></p><p><a name="Fig5"><img alt="rekisteröidy" src="../images/general_tool_functionalities_and_structure/register.png"/></a></p><p> <em>Kuva 5: Rekisteröintilomake</em></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h4><a class="anchor" id="recover" href="#recover"><i class="fa fa-link"></i></a> Palauta</h4><p> Jos unohdat salasanasi, voit palauttaa sen tässä valikossa (katso alla oleva kuva <a href="#fig6">6.</a> ).
