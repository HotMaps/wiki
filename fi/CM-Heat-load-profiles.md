<h1><a class="anchor" id="cm-heat-load-profiles" href="#cm-heat-load-profiles"><i class="fa fa-link"></i></a>CM Lämpökuormaprofiilit</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Sisällysluettelo</h2><ul><li> <a href="#in-a-glance">Yhdellä silmäyksellä</a></li><li> <a href="#introduction">Johdanto</a></li><li> <a href="#inputs-and-outputs">Tulot ja lähdöt</a><ul><li> <a href="#inputs-and-outputs_input-layers-and-parameters">Syöttökerrokset ja parametrit</a></li><li> <a href="#inputs-and-outputs_output">Tuotos</a></li></ul></li><li> <a href="#method">Menetelmä</a><ul><li> <a href="#method_overview">Yleiskatsaus</a></li><li> <a href="#method_details">Yksityiskohdat</a></li><li> <a href="#method_implementation">Toteutus</a></li></ul></li><li> <a href="#github-repository-of-this-calculation-module">Tämän laskentamoduulin GitHub-arkisto</a></li><li> <a href="#sample-run">Näyte ajo</a></li><li> <a href="#how-to-cite">Kuinka lainata</a></li><li> <a href="#authors-and-reviewers">Kirjoittajat ja arvostelijat</a></li><li> <a href="#license">Lisenssi</a></li><li> <a href="#acknowledgement">Tunnustus</a></li></ul><h2><a class="anchor" id="in-a-glance" href="#in-a-glance"><i class="fa fa-link"></i></a> Yhdellä silmäyksellä</h2><p> Tämä moduuli tuottaa tilalämmityksen ja käyttöveden tuotannon kysyntäprofiilit valitulle alueelle. Se käyttää oletuskuormitusprofiileja asuin- ja palvelualalla NUTS2-tasolla ja skaalaa sen tiettyyn kysyntään näillä aloilla ja tarjoaa kuormaprofiilit erotettuna tilan lämmitykseen ja käyttöveden tuottamiseen.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Johdanto</h2><p> Hotmaps tarjoaa yleiset ja vuodekohtaiset lämpökuormaprofiilit NUTS2-tasolla. Käyttäjä saattaa haluta mukauttaa oletusprofiileja omistamiensa tietojen perusteella. Tämä laskentamoduuli antaa mahdollisuuden skaalata profiilien eri elementtejä ylös tai alas ja saada uusi profiili.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Tulot ja lähdöt</h2><h3><a class="anchor" id="input-layers-and-parameters" href="#input-layers-and-parameters"><i class="fa fa-link"></i></a> Syöttökerrokset ja parametrit</h3><p> Hotmapsin oletustietojoukosta käytetään useita tasoja ja profiileja:</p><ul><li> Lämpötiheysasuntosektori</li><li> Lämpötiheys muualla kuin asunnoissa</li><li> Asunnon bruttopinta-ala</li><li> Kokonaispinta-ala muu kuin asuin</li><li> Teollinen tietokanta</li><li> Kuormaprofiilit teollisuudelle</li><li> Kuormaprofiilit asuntojen lämmitykseen</li><li> Kuormaprofiilit tertiäärilämmitystä varten</li><li> Kuumavesiprofiilit kotitalouskäyttöön</li><li> Latausprofiilit tertiääriselle käyttövedelle</li></ul><h3><a class="anchor" id="user-inputs" href="#user-inputs"><i class="fa fa-link"></i></a> Käyttäjän panokset</h3><p> Käyttäjän tulee antaa seuraavat parametrit:</p><ul><li> <strong>Asuntojen lämmityskerroin</strong> : Asuntojen lämmitystarve kerrotaan tällä kertoimella.</li><li> <strong>Tertiäärinen lämmityskerroin</strong> : Tertiäärinen lämmitystarve kerrotaan tällä kertoimella.</li><li> <strong>Asuinrakennusten käyttöveden kerroin</strong> : Asuntojen käyttöveden tarve kerrotaan tällä kertoimella.</li><li> <strong>Tertiäärinen käyttöveden kerroin</strong> : Tertiäärisen käyttöveden tarve kerrotaan tällä kertoimella.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Tuotos</h3><h4><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indikaattorit</h4><ul><li> <strong>Lämmön kysyntä toimialoittain</strong> : <strong>Teollisuuden</strong> vuosittainen lämmöntarve valitulla alueella.</li><li> <strong>Lämmön tarve</strong> asuntolämmityksellä: Asunnon lämmitykseen vaadittava <strong>lämmitys</strong> vuodessa valitulla alueella.</li><li> <strong>Lämmön tarve asuinrakennusten lämpimän vesihuollon kautta</strong> : <strong>Asunnon lämpimän vesihuollon</strong> edellyttämä lämmitys vuodessa valitulla alueella.</li><li> <strong>Lämmön kysyntä tertiäärisellä lämmityksellä</strong> : Vuosilämpö vaaditaan tertiäärisen sektorin lämmitykselle valitulla alueella.</li><li> <strong>Lämmön kysyntä korkea-asteen sektorilla lämminvesihuolto</strong> : Vuosittainen lämpö vaaditaan tertiäärisen sektorin lämpimän veden saannille valitulla alueella.</li><li> <strong>Pään</strong> kokonaiskysyntä: Vuotuinen kokonaislämpö tarvitaan valitulla alueella.</li></ul><h4><a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Grafiikka</h4><ul><li> <strong>Lämmön kysyntäprofiilit</strong> : graafinen esitys yllä luetelluista indikaattoreista vuoden aikana.</li></ul><p> X-akseli edustaa aikaa ja y-akseli kuvaa tehoa MW: na. Voi olla hyödyllistä piilottaa tietyt sektorit, jotta näet paremmin muut. Tämä voidaan tehdä kaksoisnapsauttamalla legendaa.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="method" href="#method"><i class="fa fa-link"></i></a> Menetelmä</h2><h3><a class="anchor" id="overview" href="#overview"><i class="fa fa-link"></i></a> Yleiskatsaus</h3><p> Vastaavat kuormaprofiilit määritetään käyttäjän valinnassa teollisuusalueille, asuin- ja korkea-asteen rakennuksille.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="details" href="#details"><i class="fa fa-link"></i></a> Yksityiskohdat</h3><h4><a class="anchor" id="residential-warm-water-supply" href="#residential-warm-water-supply"><i class="fa fa-link"></i></a> Asuntojen lämminvesihuolto</h4><p> Asuinrakennusten bruttopinta-alan ja niiden NUTS 0 ID: n perusteella lämminvesihuoltoon käytetty energia lasketaan seuraavan luettelon avulla. Käyttämällä valitun alueen NUTS 2 -tunnusta määritetään kuormitusprofiili. Asunnon käyttöveden tarjonta kerrotaan asunnon käyttöveden kertoimella, jos käyttäjä haluaa säätää arvoa.</p><p> <em>Spesifinen energia alueittain lämpimän veden saamiseksi asuinrakennuksissa eri maissa</em></p><p> | Pähkinät0 ID | Lämmin vesihuolto, kWh / m² / a | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21.38 | | CY | 8.80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14.35 | | FI | 10.15 | | FR | 9,66 | | DE | 8.27 | | EL | 12.51 | | HU | 13,66 | | IE | 15.91 | | IT | 14.01 | | LV | 15.71 | | LT | 13.36 | | LU | 8.29 | | MT | 10,99 | | NL | 8,91 | | PL | 10.00 | | PT | 9.48 | | RO | 11.48 | | SK | 21.51 | | SI | 21,74 | | ES | 23.34 | | SE | 13.54 | | UK | 49,03 |</p><h4><a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Asuntojen lämmitys</h4><p> Käytetään asuntosektorin lämmöntarpeen tiheyttä.
