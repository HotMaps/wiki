</p></li><li><p> Tertiär uppvärmningsfaktor </p><p> Den tertiära uppvärmningsbehovet multipliceras med denna faktor. </p></li><li><p> Bostadsfaktor för varmvattenförsörjning </p><p> Behovet av varmvatten för bostäder multipliceras med denna faktor. </p></li><li><p> Tertiär varmvattenförsörjningsfaktor </p><p> Den tertiära efterfrågan på varmvatten multipliceras med denna faktor. </p></li></ul><h3> <a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Produktion </h3><h4> <a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> indikatorer </h4><ul><li><p> Värmebehov efter bransch </p><p> Årlig värmebehov från bransch inom utvalt område. </p></li><li><p> Värmebehov med bostadsvärme </p><p> Årlig värme som krävs för uppvärmning av bostäder i utvalt område. </p></li><li><p> Värme efterfrågan på bostäder varmvattenförsörjning </p><p> Årlig värme krävs för varmvattenförsörjning i bostäder i utvalt område. </p></li><li><p> Värmeefterfrågan med tertiär uppvärmning </p><p> Årlig värme som krävs för uppvärmning av tertiär sektor i valt område. </p></li><li><p> Värmeefterfrågan från tertiär varmvattenförsörjning </p><p> Årlig värme som krävs för varmvattenförsörjning i tertiär sektor i valt område. </p></li><li><p> Total efterfrågan på huvudet </p><p> Årlig total värme som krävs i utvalt område. </p></li></ul><h4> <a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Grafik </h4><ul><li><p> Profiler för värmebehov </p><p> Grafik som visar ovan angivna Det kan vara bra att dölja vissa sektorer för att bättre se andra. </p><h2> <a class="anchor" id="method" href="#method"><i class="fa fa-link"></i></a> Metod </h2><h3> <a class="anchor" id="overview" href="#overview"><i class="fa fa-link"></i></a> Översikt </h3><p> Motsvarande belastningsprofiler tilldelas industriområden, bostäder och högre byggnader i användarvalet. </p><h3> <a class="anchor" id="details" href="#details"><i class="fa fa-link"></i></a> detaljer </h3><h4> <a class="anchor" id="residential-warm-water-supply" href="#residential-warm-water-supply"><i class="fa fa-link"></i></a> Bostadsvattentillförsel </h4><p> Baserat på bostadsbyggnadens bruttoytor och deras Nuts0-ID beräknas energin som används för varmvattenförsörjning med följande lista. Med Nuts2 ID för ett specifikt område tilldelas en lastprofil. Varmvattenförsörjningen för bostäder multipliceras med varmvattenförsörjningsfaktorn för bostäder om användaren vill justera värdet. </p><p> <em>Specifik energi per område för varmvattenförsörjning i bostadshus i olika länder</em> </p><p> | Nötter0 ID | Varmvattenförsörjning i kWh / m² / a | | ------------- |: -------------: | | AT | 21.67 | | BE | 31,95 | | BG | 12,93 | | HR | 21.38 | | CY | 8,80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14.35 | | FI | 10.15 | | FR | 9,66 | | DE | 8.27 | | EL | 12.51 | | HU | 13,66 | | IE | 15,91 | | IT | 14.01 | | LV | 15,71 | | LT | 13.36 | | LU | 8.29 | | MT | 10,99 | | NL | 8,91 | | PL | 10.00 | | PT | 9.48 | | RO | 11.48 | | SK | 21.51 | | SI | 21,74 | | ES | 23.34 | | SE | 13.54 | | UK | 49.03 | </p><h4> <a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Bostadsuppvärmning </h4><p> Värmetätheten för bostadssektorn används. Eftersom detta datasæt redan innehåller energin som används för varmt vatten, dras den tidigare beräknade varmvattenenergin bort från den. Baserat på Nuts2 ID tilldelas en bostadsvärmeprofil för varje kakel av värmetätheten i användarvalet. Bostadsuppvärmningen multipliceras med bostadsvärmefaktorn i fall användaren vill justera värdet. </p><h4> <a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Tertiär varmvattenförsörjning </h4><p> I likhet med varmvattenförsörjningen i bostäder beräknas energin som behövs med en tabell. Med Nuts2 ID för ett specifikt område tilldelas en lastprofil. Den tertiära varmvattenförsörjningen multipliceras med den tertiära varmvattenförsörjningsfaktorn i fall användaren vill justera värdet. </p><p> <em>Specifik energi per område för varmvattenförsörjning i tertiära byggnader i olika länder</em> </p><p> | Nötter0 ID | Varmvattenförsörjning i kWh / m² / a | | ------------- |: -------------: | | AT | 6,57 | | BE | 13,88 | | BG | 15,88 | | HR | 9.42 | | CY | 6.26 | | CZ | 9.18 | | DK | 8,03 | | EE | 14.13 | | FI | 10.52 | | FR | 9,57 | | DE | 3.05 | | EL | 6,99 | | HU | 9,51 | | IE | 10,87 | | IT | 5,62 | | LV | 7.16 | | LT | 10.46 | | LU | 7,20 | | MT | 10.45 | | NL | 6,89 | | PL | 9,55 | | PT | 21.47 | | RO | 13,85 | | SK | 8.49 | | SI | 27,73 | | ES | 12.44 | | SE | 19.62 | | UK | 13.45 | </p><h4> <a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Tertiär uppvärmning </h4><p> Den tertiära uppvärmningen beräknas liknande bostadsuppvärmningen. Den tertiära uppvärmningen multipliceras med den tertiära uppvärmningsfaktorn om användaren vill justera värdet. </p><h4> <a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Industriell värmebehov </h4><p> Industriens värmebehov letas upp i den industriella databasen och baseras på Nuts0 ID och undersektorn tilldelas en lastprofil. </p><h4> <a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Summering </h4><p> De fem resulterande profilerna läggs samman på ett absolut sätt. </p><h3> <a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Genomförande </h3><h4> <a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Ladda profiler </h4><p> De nämnda lastprofilerna består av 8760 punkter som representerar lasten för varje timme under de 365 dagarna. Mer information om <strong><a href="https://gitlab.com/hotmaps/load_profile">lastprofilerna hittar du här.</a></strong> Alla lastprofiler normaliseras så att integralen är lika med 1. </p><h4> <a class="anchor" id="industrial-sites" href="#industrial-sites"><i class="fa fa-link"></i></a> Industriella anläggningar </h4><p> Värmekällorna hämtas från den <strong><a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">industriella databasen.</a></strong> Baserat på deras överskottsvärme, Nuts0 ID och industrisektorn skapas en lastprofil som täcker varje timme på året för varje webbplats. </p><h4> <a class="anchor" id="heat-densities" href="#heat-densities"><i class="fa fa-link"></i></a> Värmetäthet </h4><p> Värmetäthetsbostadssektorn och icke-bostadssektorn för värmetäthet är rasterfiler som finns <strong><a href="https://gitlab.com/hotmaps/heat">här.</a></strong> </p><h4> <a class="anchor" id="gross-floor-areas" href="#gross-floor-areas"><i class="fa fa-link"></i></a> Brutto golvytor </h4><p> Bruttoboligbostadshus och icke-bostadsområde är rasterfiler som kan hittas <strong><a href="https://gitlab.com/hotmaps/gfa_res_curr_density">här</a></strong> och <strong><a href="https://gitlab.com/hotmaps/gfa_nonres_curr_density">här.</a></strong> </p><h2> <a class="anchor" id="sample-run" href="#sample-run"><i class="fa fa-link"></i></a> Provkörning </h2><p> Provkörning i DK05. </p><figure><img alt="" src="https://github.com/HotMaps/hotmaps_wiki/blob/master/Images/cm_load_profile/sample_run.PNG"/><figcaption> <i>Provkörning i DK05.
