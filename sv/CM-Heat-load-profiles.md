<h1><a class="anchor" id="cm-heat-load-profiles" href="#cm-heat-load-profiles"><i class="fa fa-link"></i></a>CM Värmebelastningsprofiler</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Innehållsförteckning</h2><ul><li> <a href="#in-a-glance">I en blick</a></li><li> <a href="#introduction">Introduktion</a></li><li> <a href="#inputs-and-outputs">In- och utgångar</a><ul><li> <a href="#inputs-and-outputs_input-layers-and-parameters">Mata in lager och parametrar</a></li><li> <a href="#inputs-and-outputs_output">Produktion</a></li></ul></li><li> <a href="#method">Metod</a><ul><li> <a href="#method_overview">Översikt</a></li><li> <a href="#method_details">Detaljer</a></li><li> <a href="#method_implementation">Genomförande</a></li></ul></li><li> <a href="#github-repository-of-this-calculation-module">GitHub-förvar för denna beräkningsmodul</a></li><li> <a href="#sample-run">Provkörning</a></li><li> <a href="#how-to-cite">Hur man citerar</a></li><li> <a href="#authors-and-reviewers">Författare och granskare</a></li><li> <a href="#license">Licens</a></li><li> <a href="#acknowledgement">Bekräftelse</a></li></ul><h2><a class="anchor" id="in-a-glance" href="#in-a-glance"><i class="fa fa-link"></i></a> I en blick</h2><p> Den här modulen genererar belastningsprofiler för efterfrågan på rymdvärme och varmvattenproduktion för en vald region. Den använder standardbelastningsprofiler för bostads- och tertiärsektorn på NUTS2-nivå och skalar den till en given efterfrågan inom dessa sektorer och tillhandahåller lastprofilerna separerade för uppvärmning och varmvattenproduktion.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introduktion</h2><p> Hotmaps tillhandahåller generiska och årsspecifika värmebelastningsprofiler på NUTS2-nivå. Användaren kanske vill anpassa standardprofilerna baserat på den information som han / hon har. Denna beräkningsmodul ger möjlighet att skala upp olika eller nedåtgående element i profilerna uppåt eller nedåt och få en ny profil.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> In- och utgångar</h2><h3><a class="anchor" id="input-layers-and-parameters" href="#input-layers-and-parameters"><i class="fa fa-link"></i></a> Mata in lager och parametrar</h3><p> Flera lager och profiler används från standarddatamängden för Hotmaps:</p><ul><li> Värmesäkerhetsbostadssektor</li><li> Värmedensitet, icke-bostadsområde</li><li> Brutto golvyta bostäder</li><li> Brutto golvyta ej för bostäder</li><li> Industriell databas</li><li> Ladda profiler för industrin</li><li> Lastprofiler för bostadsvärme</li><li> Lastprofiler för tertiär uppvärmning</li><li> Lastprofiler för tappvarmvatten</li><li> Lastprofiler för tertiärt varmvatten</li></ul><h3><a class="anchor" id="user-inputs" href="#user-inputs"><i class="fa fa-link"></i></a> Användaringångar</h3><p> Användaren bör ange följande parametrar:</p><ul><li> <strong>Bostadsvärmefaktor</strong> : Behovet av uppvärmning av bostäder multipliceras med denna faktor.</li><li> <strong>Tertiär uppvärmningsfaktor</strong> : Det tertiära uppvärmningsbehovet multipliceras med denna faktor.</li><li> <strong>Bostadsvarmvattenförsörjningsfaktor: Behovet av varmvatten för</strong> bostäder multipliceras med denna faktor.</li><li> <strong>Tertiär varmvattenförsörjningsfaktor</strong> : Det tertiära varmvattenbehovet multipliceras med denna faktor.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Produktion</h3><h4><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indikatorer</h4><ul><li> <strong>Värmebehov efter bransch</strong> : Årlig värmebehov efter bransch i det valda området.</li><li> <strong>Värmebehov från bostadsvärme</strong> : Årlig värme som krävs för bostadsvärme i det valda området.</li><li> <strong>Värmebehov av varmvattenförsörjning</strong> : Årlig värme som krävs för varmvattenförsörjning i det valda området.</li><li> <strong>Värmebehov genom uppvärmning av tertiär sektor</strong> : Årlig värme som krävs för uppvärmning av tertiär sektor i det valda området.</li><li> <strong>Värmebehov per varmvattenförsörjning i tertiär sektor</strong> : Årlig värme som krävs för varmvattenförsörjning i tertiär sektor i det valda området.</li><li> <strong>Totalt behov av huvud</strong> : Årlig total värme som krävs i det valda området.</li></ul><h4><a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Grafik</h4><ul><li> <strong>Värmebehovsprofiler</strong> : En grafik som visar ovanstående indikatorer under året.</li></ul><p> X-axeln representerar tiden och y-axeln visar effekten i MW. Det kan vara bra att dölja vissa sektorer för att bättre se andra. Detta kan göras genom att dubbelklicka på förklaringen.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="method" href="#method"><i class="fa fa-link"></i></a> Metod</h2><h3><a class="anchor" id="overview" href="#overview"><i class="fa fa-link"></i></a> Översikt</h3><p> Motsvarande belastningsprofiler tilldelas industriområden, bostäder och tertiära byggnader i användarvalet.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="details" href="#details"><i class="fa fa-link"></i></a> Detaljer</h3><h4><a class="anchor" id="residential-warm-water-supply" href="#residential-warm-water-supply"><i class="fa fa-link"></i></a> Varmvattenförsörjning i bostäder</h4><p> Baserat på bruttoytan för bostadshus och deras NUTS 0 ID, beräknas energin som används för varmvattenförsörjning enligt följande lista. Med hjälp av NUTS 2 ID för det valda området tilldelas en lastprofil. Varmvattenförsörjningen för bostäder multipliceras med bostadsvarmvattenförsörjningsfaktorn om användaren vill justera värdet.</p><p> <em>Specifik energi per område för varmvattenförsörjning i bostadshus i olika länder</em></p><p> | Nuts0 ID | Varmvattenförsörjning i kWh / m² / a | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21.38 | | CY | 8.80 | | CZ | 22,83 | | DK | 9.64 | | EE | 14.35 | | FI | 10.15 | | FR | 9.66 | | DE | 8,27 | | EL | 12.51 | | HU | 13.66 | | IE | 15.91 | | IT | 14.01 | | LV | 15,71 | | LT | 13.36 | | LU | 8,29 | | MT | 10.99 | | NL | 8,91 | | PL | 10.00 | | PT | 9.48 | | RO | 11.48 | | SK | 21.51 | | SI | 21,74 | | ES | 23.34 | | SE | 13.54 | | Storbritannien | 49.03 |</p><h4><a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Bostadsvärme</h4><p> Värmebehovstätheten för bostadssektorn används. Eftersom denna datamängd redan innehåller energin som används för varmt vatten, dras den tidigare beräknade energin för varmvatten från den. Baserat på NUTS 2 ID tilldelas en bostadsuppvärmningsprofil för varje kakel av värmedensiteten i användarvalet. Bostadsvärmen multipliceras med bostadsvärmefaktorn om användaren vill justera värdet.</p><h4><a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Tertiär varmvattenförsörjning</h4><p> I likhet med varmvattenförsörjningen för bostäder beräknas den energi som behövs med en tabell. Med NUTS 2 ID för ett visst område tilldelas en lastprofil. Tertiär varmvattenförsörjning multipliceras med tertiär varmvattenförsörjningsfaktor om användaren vill justera värdet.</p><p> <em>Specifik energi per område för varmvattenförsörjning i tertiära byggnader i olika länder</em></p><p> | Nuts0 ID | Varmvattenförsörjning i kWh / m² / a | | ------------- |: -------------: | | AT | 6,57 | | BE | 13,88 | | BG | 15,88 | | HR | 9.42 | | CY | 6.26 | | CZ | 9.18 | | DK | 8,03 | | EE | 14.13 | | FI | 10.52 | | FR | 9.57 | | DE | 3.05 | | EL | 6,99 | | HU | 9.51 | | IE | 10,87 | | IT | 5,62 | | LV | 7.16 | | LT | 10.46 | | LU | 7.20 | | MT | 10.45 | | NL | 6,89 | | PL | 9.55 | | PT | 21.47 | | RO | 13,85 | | SK | 8.49 | | SI | 27,73 | | ES | 12.44 | | SE | 19.62 | | Storbritannien | 13.45 |</p><h4><a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Tertiär uppvärmning</h4><p> Tertiär uppvärmning beräknas på samma sätt som bostadsvärmen. Tertiär uppvärmning multipliceras med tertiär uppvärmningsfaktor om användaren vill justera värdet.</p><h4><a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Industriellt värmebehov</h4><p> Branschens värmebehov slås upp i den industriella databasen och baserat på NUTS 0 ID och delsektorn tilldelas en lastprofil.</p><h4><a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Summering</h4><p> De fem resulterande profilerna sammanställs på ett absolut sätt för att få den slutliga profilen.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Genomförande</h3><h4><a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Ladda profiler</h4><p> De nämnda lastprofilerna består av 8760 poäng som representerar belastningen för varje timme under 365 dagar. Mer information om <strong><a href="https://gitlab.com/hotmaps/load_profile">lastprofilerna finns här.</a></strong> Alla belastningsprofiler normaliseras så att integralen är lika med 1.</p><h4><a class="anchor" id="industrial-sites" href="#industrial-sites"><i class="fa fa-link"></i></a> Industriella platser</h4><p> Värmekällorna hämtas från den <strong><a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">industriella databasen.</a></strong> Baserat på deras överskottsvärme skapas NUTS 0 ID och industrisektorn en lastprofil som täcker varje timme på året för varje anläggning.</p><h4><a class="anchor" id="heat-densities" href="#heat-densities"><i class="fa fa-link"></i></a> Värmedensiteter</h4><p> Värmebehovstätheten för bostads- och icke-bostadsbranschen är rasterfiler som finns <strong><a href="https://gitlab.com/hotmaps/heat">här.</a></strong></p><h4><a class="anchor" id="gross-floor-areas" href="#gross-floor-areas"><i class="fa fa-link"></i></a> Brutto golvyta</h4><p> Brutto golvyta bostadsområde och brutto golvyta icke-bostäder är rasterfiler som finns <strong><a href="https://gitlab.com/hotmaps/gfa_res_curr_density">här</a></strong> och <strong><a href="https://gitlab.com/hotmaps/gfa_nonres_curr_density">här.</a></strong></p><p> <a
