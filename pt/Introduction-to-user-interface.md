Além da mensagem de isenção de responsabilidade, são fornecidas informações adicionais sobre os objetivos da caixa de ferramentas Hotmaps, bem como o link para o site do projeto Hotmaps e os repositórios de dados. </p><p> <a name="Fig1"><img alt="intro_gif" src="https://github.com/HotMaps/hotmaps_wiki/blob/master/Images/general_tool_functionalities_and_structure/intro.gif"/></a> <em>Fig. 1 Breve introdução à caixa de ferramentas Hotmas</em> </p><p> A GUI do Hotmaps é uma interface baseada em GIS. Essas ferramentas são ilustradas na <a href="#fig2">Fig.2</a> </p><p> <a name="Fig2"><img alt="primeiro olhar"
