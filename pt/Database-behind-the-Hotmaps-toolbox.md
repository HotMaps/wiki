href="#acknowledgement">Reconhecimento</a></li></ul><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introdução</h2><p> O banco de dados construído para Hotmaps é um data warehouse espacial disponibilizado para a plataforma para fins de visualização e computação.</p><p> Existem diferentes tipos de dados:</p><ul><li> conjuntos de dados raster (em nível de hectare)</li><li> conjuntos de dados vetoriais (regiões NUTS / LAU, pontos de interesse, ...)</li><li> conjuntos de dados não espaciais (principalmente relacionados a NUTS / LAU e vinculados a camadas vetoriais existentes)</li></ul><p> Os conjuntos de dados podem ser acessados por meio de um servidor geoespacial (GeoServer) que fornece diversos serviços (WMS / WFS / ...) que permite a qualquer pessoa exibir, incluir ou usar esses conjuntos de dados.</p><p> Os conjuntos de dados podem ser encontrados no <a href="https://gitlab.com/hotmaps">repositório Gitlab</a></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="query" href="#query"><i class="fa fa-link"></i></a> Inquerir</h2><p> Observe que as agregações ao selecionar as regiões NUTS ou LAU são pré-calculadas e, portanto, mais rápidas do que as agregações de seleções de hectares, que são calculadas rapidamente.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indicadores</h2><p> Os indicadores são definidos <strong><a href="https://github.com/HotMaps/Hotmaps-toolbox-service/blob/develop/api/app/models/indicators.py">aqui</a></strong> . Dependendo das camadas, vários parâmetros estatísticos como valor mínimo, valor máximo e valor médio estão entre os indicadores. Uma explicação sobre a definição de indicadores para as camadas integradas ao banco de dados Hotmaps pode ser encontrada <strong><a href="https://github.com/HotMaps/Hotmaps-toolbox-service/blob/develop/api/app/models/INDICATORS.md">aqui</a></strong> .</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="technologies" href="#technologies"><i class="fa fa-link"></i></a> Tecnologias</h2><p> As tecnologias utilizadas para o banco de dados são:</p><ul><li> PostgreSQL versão 9.6</li><li> PostGIS versão 2.3</li><li> GeoServer versão 2.11.2</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="references" href="#references"><i class="fa fa-link"></i></a> Referências</h2><ul><li> Simon Pezzutto, Stefano Zambotti, Silvia Croce, Pietro Zambelli, Giulia Garegnani, Chiara Scaramuzzino, Ramón Pascual Pascuas, Alyona Zubaryeva, Franziska Haas, Dagmar Exner (EURAC), Andreas Mueller (e-think), Michael Hartner (TUW), Tobias , Anna-Lena Klingler, Matthias Kühnbach, Pia Manz, Simon Marwitz, Matthias Rehfeldt, Jan Steinbach, Eftim Popovski (Fraunhofer ISI) Revisado por Lukas Kranzl, Sara Fritz (TUW) Hotmaps Project, <strong>D2.3 Relatório WP2 -
