<h1><a class="anchor" id="cm-heat-load-profiles" href="#cm-heat-load-profiles"><i class="fa fa-link"></i></a>Perfis de carga de calor CM</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Índice</h2><ul><li> <a href="#in-a-glance">Em um relance</a></li><li> <a href="#introduction">Introdução</a></li><li> <a href="#inputs-and-outputs">Entradas e saídas</a><ul><li> <a href="#inputs-and-outputs_input-layers-and-parameters">Camadas de entrada e parâmetros</a></li><li> <a href="#inputs-and-outputs_output">Resultado</a></li></ul></li><li> <a href="#method">Método</a><ul><li> <a href="#method_overview">Visão geral</a></li><li> <a href="#method_details">Detalhes</a></li><li> <a href="#method_implementation">Implementação</a></li></ul></li><li> <a href="#github-repository-of-this-calculation-module">Repositório GitHub deste módulo de cálculo</a></li><li> <a href="#sample-run">Execução de amostra</a></li><li> <a href="#how-to-cite">Como citar</a></li><li> <a href="#authors-and-reviewers">Autores e revisores</a></li><li> <a href="#license">Licença</a></li><li> <a href="#acknowledgement">Reconhecimento</a></li></ul><h2><a class="anchor" id="in-a-glance" href="#in-a-glance"><i class="fa fa-link"></i></a> Em um relance</h2><p> Este módulo gera perfis de carga de demanda para aquecimento ambiente e geração de água quente para uma região selecionada. Ele usa perfis de carga padrão para o setor residencial e terciário no nível NUTS2 e o dimensiona para uma determinada demanda nesses setores e fornece os perfis de carga separados para aquecimento ambiente e geração de água quente.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introdução</h2><p> Hotmaps fornece perfis de carga de calor genéricos e específicos para o ano no nível NUTS2. O usuário pode desejar adaptar os perfis padrão com base nas informações que possui. Este módulo de cálculo permite aumentar ou diminuir os diferentes elementos constituintes dos perfis e obter um novo perfil.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Entradas e saídas</h2><h3><a class="anchor" id="input-layers-and-parameters" href="#input-layers-and-parameters"><i class="fa fa-link"></i></a> Camadas de entrada e parâmetros</h3><p> Várias camadas e perfis são usados a partir do conjunto de dados padrão dos Hotmaps:</p><ul><li> Setor residencial de densidade de calor</li><li> Setor não residencial de densidade de calor</li><li> Área bruta de construção residencial</li><li> Área bruta não residencial</li><li> Banco de dados industrial</li><li> Perfis de carga para a indústria</li><li> Perfis de carga para aquecimento residencial</li><li> Perfis de carga para aquecimento terciário</li><li> Perfis de carga para água quente sanitária residencial</li><li> Perfis de carga para água quente sanitária terciária</li></ul><h3><a class="anchor" id="user-inputs" href="#user-inputs"><i class="fa fa-link"></i></a> Entradas do usuário</h3><p> O usuário deve fornecer os seguintes parâmetros:</p><ul><li> <strong>Fator de aquecimento residencial</strong> : A demanda de aquecimento residencial é multiplicada por este fator.</li><li> <strong>Fator de aquecimento terciário</strong> : A demanda de aquecimento terciário é multiplicada por este fator.</li><li> <strong>Fator de abastecimento de água quente residencial</strong> : A demanda residencial de água quente é multiplicada por este fator.</li><li> <strong>Fator de abastecimento de água quente terciária</strong> : A demanda de água quente terciária é multiplicada por este fator.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Resultado</h3><h4><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indicadores</h4><ul><li> <strong>Demanda de calor por setor</strong> : demanda de calor anual por setor na área selecionada.</li><li> <strong>Demanda de calor por aquecimento residencial</strong> : calor anual necessário para aquecimento residencial na área selecionada.</li><li> <strong>Demanda de calor pelo suprimento residencial de água quente</strong> : calor anual necessário para o suprimento residencial de água quente na área selecionada.</li><li> <strong>Demanda de calor pelo aquecimento do setor terciário</strong> : Calor anual necessário para o aquecimento do setor terciário na área selecionada.</li><li> <strong>Demanda de calor pelo suprimento de água quente do setor terciário</strong> : Calor anual necessário para o suprimento de água quente do setor terciário na área selecionada.</li><li> <strong>Demanda total da cabeça</strong> : Calor total anual necessário na área selecionada.</li></ul><h4><a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Gráficos</h4><ul><li> <strong>Perfis de demanda de calor</strong> : um gráfico mostrando os indicadores listados acima ao longo do ano.</li></ul><p> O eixo x representa o tempo e o eixo y mostra a potência em MW. Pode ser útil ocultar certos setores para ver melhor os outros. Isso pode ser feito clicando duas vezes na legenda.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="method" href="#method"><i class="fa fa-link"></i></a> Método</h2><h3><a class="anchor" id="overview" href="#overview"><i class="fa fa-link"></i></a> Visão geral</h3><p> Os perfis de carga correspondentes são atribuídos a locais industriais, edifícios residenciais e terciários na seleção do usuário.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="details" href="#details"><i class="fa fa-link"></i></a> Detalhes</h3><h4><a class="anchor" id="residential-warm-water-supply" href="#residential-warm-water-supply"><i class="fa fa-link"></i></a> Abastecimento residencial de água quente</h4><p> Com base na área bruta de construção de edifícios residenciais e sua ID NUTS 0, a energia usada para o abastecimento de água quente é calculada com a seguinte lista. Usando o NUTS 2 ID da área selecionada, um perfil de carga é atribuído. O fornecimento de água quente residencial é multiplicado pelo fator de fornecimento de água quente residencial, caso o usuário queira ajustar o valor.</p><p> <em>Energia específica por área para abastecimento de água quente em edifícios residenciais em diferentes países</em></p><p> | Nuts0 ID | Abastecimento de água quente em kWh / m² / a | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | RH | 21,38 | | CY | 8,80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14,35 | | FI | 10,15 | | FR | 9,66 | | DE | 8,27 | | EL | 12,51 | | HU | 13,66 | | IE | 15,91 | | TI | 14,01 | | LV | 15,71 | | LT | 13,36 | | LU | 8,29 | | MT | 10,99 | | NL | 8,91 | | PL | 10,00 | | PT | 9,48 | | RO | 11,48 | | SK | 21,51 | | SI | 21,74 | | ES | 23,34 | | SE | 13,54 | | UK | 49,03 |</p><h4><a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Aquecimento residencial</h4><p> É utilizada a densidade de demanda de calor para o setor residencial. Uma vez que este conjunto de dados já contém a energia usada para água quente, a energia de água quente calculada anteriormente é subtraída dele. Com base no NUTS 2 ID, um perfil de aquecimento residencial é atribuído para cada ladrilho da densidade de calor na seleção do usuário. O aquecimento residencial é multiplicado pelo fator de aquecimento residencial caso o usuário queira ajustar o valor.</p><h4><a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Abastecimento de água quente terciária</h4><p> Semelhante ao abastecimento residencial de água quente, a energia necessária é calculada com uma tabela. Com o NUTS 2 ID de uma área específica, um perfil de carga é atribuído. O fornecimento de água quente terciária é multiplicado pelo fator de fornecimento de água quente terciário, caso o usuário queira ajustar o valor.</p><p> <em>Energia específica por área para abastecimento de água quente em edifícios terciários em diferentes países</em></p><p> | Nuts0 ID | Abastecimento de água quente em kWh / m² / a | | ------------- |: -------------: | | AT | 6,57 | | BE | 13,88 | | BG | 15,88 | | RH | 9,42 | | CY | 6,26 | | CZ | 9,18 | | DK | 8,03 | | EE | 14,13 | | FI | 10,52 | | FR | 9,57 | | DE | 3,05 | | EL | 6,99 | | HU | 9,51 | | IE | 10,87 | | TI | 5,62 | | LV | 7,16 | | LT | 10,46 | | LU | 7,20 | | MT | 10,45 | | NL | 6,89 | | PL | 9,55 | | PT | 21,47 | | RO | 13,85 | | SK | 8,49 | | SI | 27,73 | | ES | 12,44 | | SE | 19,62 | | UK | 13,45 |</p><h4><a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Aquecimento terciário</h4><p> O aquecimento terciário é calculado de forma semelhante ao aquecimento residencial. O aquecimento terciário é multiplicado pelo fator de aquecimento terciário, caso o usuário queira ajustar o valor.</p><h4><a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Demanda de calor industrial</h4><p> A demanda de calor por indústria é consultada no banco de dados industrial e com base na NUTS 0 ID e no subsetor, um perfil de carga é atribuído.</p><h4><a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Soma</h4><p> Os cinco perfis resultantes são agregados de forma absoluta para obter o perfil final.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Implementação</h3><h4><a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Perfis de carga</h4><p> Os perfis de carga mencionados consistem em 8.760 pontos que representam a carga para cada hora dos 365 dias. Mais informações sobre os <strong><a href="https://gitlab.com/hotmaps/load_profile">perfis de carga podem ser encontradas aqui.</a></strong> Todos os perfis de carga são normalizados de forma que a integral seja igual a 1.</p><h4><a class="anchor" id="industrial-sites" href="#industrial-sites"><i class="fa fa-link"></i></a> Sites industriais</h4><p> As fontes de calor são retiradas do <strong><a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">banco de dados industrial.</a></strong> Com base em seu excesso de calor, NUTS 0 ID e setor industrial, um perfil de carga cobrindo todas as horas do ano é criado para cada local.</p><h4><a class="anchor" id="heat-densities" href="#heat-densities"><i class="fa fa-link"></i></a> Densidades de calor</h4><p> A densidade da demanda de calor dos setores residencial e não residencial são arquivos raster que podem ser encontrados <strong><a href="https://gitlab.com/hotmaps/heat">aqui.</a></strong></p><h4><a class="anchor" id="gross-floor-areas" href="#gross-floor-areas"><i class="fa fa-link"></i></a> Áreas brutas de piso</h4><p> A área bruta de construção residencial e a área de construção não residencial são arquivos raster que podem ser encontrados <strong><a href="https://gitlab.com/hotmaps/gfa_res_curr_density">aqui</a></strong> e <strong><a href="https://gitlab.com/hotmaps/gfa_nonres_curr_density">aqui.</a></strong></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor"
