<h1><a class="anchor" id="guidelines-for-writing-a-hotmaps-wiki-page" href="#guidelines-for-writing-a-hotmaps-wiki-page"><i class="fa fa-link"></i></a>Diretrizes para escrever uma página Wiki do Hotmaps</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Índice</h2><ul><li> <a href="#introduction">Introdução</a></li><li> <a href="#wiki-toolbar-and-sidebar">Barra de ferramentas e barra lateral do Wiki</a></li><li> <a href="#add-figures-and-images">Adicione figuras e imagens</a></li><li> <a href="#classification-of-wiki-pages">Classificação de páginas wiki</a></li><li> <a href="#default-structures-of-the-wiki-pages">Estruturas padrão das páginas wiki</a><ul><li> <a href="#default-structures-of-the-wiki-pages_hotmaps-data-set-repository-structure">Estrutura do repositório do conjunto de dados Hotmaps</a></li><li> <a href="#default-structures-of-the-wiki-pages_general-functionalities-of-the-toolbox">Funcionalidades gerais da caixa de ferramentas</a></li><li> <a href="#default-structures-of-the-wiki-pages_calculation-modules">Módulos de cálculo</a></li><li> <a href="#default-structures-of-the-wiki-pages_how-to-apply-the-hotmaps-toolbox">Como aplicar a caixa de ferramentas Hotmaps</a></li><li> <a href="#default-structures-of-the-wiki-pages_developers-area">Área de desenvolvedores</a></li></ul></li><li> <a href="#authors-and-reviewers">Autores e revisores</a></li><li> <a href="#license">Licença</a></li><li> <a href="#acknowledgement">Reconhecimento</a></li></ul><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introdução</h2><p> A plataforma wiki Hotmaps fornece funcionalidades flexíveis para criar e editar as páginas wiki. Para poder criar ou modificar qualquer página wiki, você deve estar logado na plataforma wiki. Isso só é possível se você tiver uma conta GitHub e tiver uma permissão de provedor wiki atribuída. Para obter uma permissão, envie um e-mail para &quot;support (at) hotmaps.eu&quot; incluindo seu ID do GitHub. Em seguida, avaliaremos sua solicitação e responderemos a ela. Para fazer login, use o botão de login do canto superior direito. Depois de fazer o login, você pode criar e modificar as páginas.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="wiki-toolbar-and-sidebar" href="#wiki-toolbar-and-sidebar"><i class="fa fa-link"></i></a> Barra de ferramentas e barra lateral do Wiki</h2><p> No topo de cada página, você vê a barra de ferramentas de edição. A figura abaixo mostra essa barra de ferramentas e explica resumidamente a função de cada botão.</p><img alt="" src="/en/Guidelines-for-writing-a-Hotmaps-Wiki-page/toolbar_new.png"/><p> A barra lateral é visível em todas as páginas wiki. Observe que <strong>qualquer modificação na barra lateral é aplicada a todas as páginas wiki</strong> . Em outras palavras, a barra lateral não é específica para a página que você está editando. Para modificar a barra lateral, pressione o botão &quot;editar&quot; na barra de ferramentas do wiki. A modificação da barra lateral pode ser feita por meio da seção na parte inferior da página de edição, conforme mostrado na figura a seguir.</p><img alt="" src="/en/Guidelines-for-writing-a-Hotmaps-Wiki-page/sidebar.png"/><p> Para ampliar a janela de edição, arraste-a em direção ao canto inferior direito, conforme mostrado na figura a seguir.</p><img alt="" src="/en/Guidelines-for-writing-a-Hotmaps-Wiki-page/enlarge_edit_window_new.png"/> [** `To Top` **] (# sumário)<h2><a class="anchor" id="add-figures-and-images" href="#add-figures-and-images"><i class="fa fa-link"></i></a> Adicione figuras e imagens</h2><p> Para adicionar uma imagem à sua página wiki, você tem duas opções:</p><p> <strong>Método I:</strong></p><ul><li> Primeiro, carregue a imagem em sua página wiki desejada usando o botão &quot;Carregar&quot; na barra de ferramentas.</li><li> Anote o link fornecido para a imagem enviada.</li><li> Comece a editar a página wiki e use o link para a imagem.</li></ul><p> <strong>Método II:</strong></p><ul><li> Comece a editar sua página wiki desejada.</li><li> Arraste e solte a imagem no local do texto em que deseja mostrar a figura.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="classification-of-wiki-pages" href="#classification-of-wiki-pages"><i class="fa fa-link"></i></a> Classificação de páginas wiki</h2><p> As páginas wiki do Hotmaps são categorizadas em cinco classes diferentes:</p><ul><li> Estrutura do repositório do conjunto de dados Hotmaps</li><li> Funcionalidades gerais da caixa de ferramentas</li><li> Módulos de cálculo</li><li> Como aplicar a caixa de ferramentas Hotmaps</li><li> Área de desenvolvedores</li></ul><p> Para cada classe de página wiki, uma estrutura com várias seções é sugerida (consulte a próxima seção nesta página). As seções em <strong>NEGRITO</strong> devem fazer parte da página Wiki. O resto, por outro lado, deve ser adicionado, se aplicável. Como uma página de amostra, por favor, dê uma olhada <a href="/en/CM-District-heating-potential-areas-user-defined-thresholds">nesta página wiki</a> .</p><p> As páginas do Wiki podem se tornar muito longas devido a figuras incorporadas. Para ajudar os leitores a navegar em uma página com facilidade e facilidade para encontrar o que procuram, adicione um “ <strong>Índice</strong> ” no topo de cada página que você está escrevendo. Além disso, use um botão “ <a href="#table-of-contents"><strong><code>To Top</code></strong></a> ” no final de cada seção com um hiperlink para o “Índice” no topo da página wiki.</p><p> Caso você deva usar suas próprias figuras, primeiro crie uma nova pasta para sua página wiki aqui e, em seguida, empurre todas as figuras para a pasta. Evite também o uso de imagens volumosas, pois o repositório tem capacidade limitada.</p><p> Depois de terminar de escrever sua página wiki, peça a seus colegas para revisar sua página e dar feedback para você. Desta forma, podemos minimizar os erros de escrita e garantir que as páginas wiki sejam escritas de forma clara. Adicione o nome do revisor na seção &quot;Autores e revisores&quot;. Siga a mesma estrutura da página de amostra fornecida.</p><p> <strong>NOTA</strong> : As páginas do Wiki geralmente são escritas na linguagem Markdown (alguns formatos adicionais também são possíveis). Se você não está familiarizado com o Markdown, leia <a href="https://guides.github.com/features/mastering-markdown/">aqui</a> o breve tutorial fornecido pelo GitHub.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="default-structures-of-the-wiki-pages" href="#default-structures-of-the-wiki-pages"><i class="fa fa-link"></i></a> Estruturas padrão das páginas wiki</h2><h3><a class="anchor" id="hotmaps-data-set-repository-structure" href="#hotmaps-data-set-repository-structure"><i class="fa fa-link"></i></a> Estrutura do repositório do conjunto de dados Hotmaps</h3><p> Aqui, o ponto mais importante é manter o controle de todos os repositórios Hotmaps no GitLab e manter os links atualizados.
