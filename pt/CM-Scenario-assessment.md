<h1><a class="anchor" id="cm-scenario-assessment" href="#cm-scenario-assessment"><i class="fa fa-link"></i></a>Avaliação do cenário CM</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Índice</h2><ul><li> <a href="#at-a-glance">Num relance</a></li><li> <a href="#introduction">Introdução</a></li><li> <a href="#scenario-assessment-inputs-and-outputs">Entradas e saídas da avaliação de cenário</a></li><li> <a href="#interpretation-of-input-parameters">Interpretação de parâmetros de entrada</a></li><li> <a href="#interpretation-of-results">interpretação de resultados</a></li><li> <a href="#sample-run">Execução de amostra</a></li><li> <a href="#how-to-cite">Como citar</a></li><li> <a href="#authors-and-reviewers">Autores e revisores</a></li><li> <a href="#license">Licença</a></li><li> <a href="#acknowledgement">Reconhecimento</a></li></ul><h2><a class="anchor" id="at-a-glance" href="#at-a-glance"><i class="fa fa-link"></i></a> Num relance</h2><p> Este módulo permite comparar diferentes cenários de aquecimento. O módulo não está integrado ao Hotmaps Toolbox, mas é fornecido como uma pasta de trabalho Excel separada que combina os resultados de outros módulos de cálculo a serem executados anteriormente. O modelo, tal como está, permite comparar até 9 cenários diferentes, mas se devidamente adaptado, pode funcionar em qualquer número de cenários. Planilhas separadas são configuradas para agrupar os indicadores-chave de cada módulo de cálculo, enquanto os indicadores-chave são automaticamente agrupados na tabela final e representados em gráficos.</p><p> <a href="https://github.com/HotMaps/scenario_assessment/raw/master/data/ScenarioAssessment_Template.xlsx"><strong>MANUAL DE AVALIAÇÃO DE CENÁRIO: CLIQUE AQUI PARA BAIXAR</strong></a></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introdução</h2><p> A abordagem para avaliar uma série de diferentes cenários de aquecimento segue esta sequência de cálculos:</p><ul><li> O estoque de construção para a área selecionada,</li><li> Custos, emissões e demanda de energia para o fornecimento descentralizado de calor,</li><li> Custos, emissões e demanda de energia para o fornecimento de calor distrital,</li><li> Custos da distribuição do calor distrital.</li></ul><p> Em cada cenário, uma certa divisão entre o fornecimento de calor central e descentralizado é assumida juntamente com as hipóteses sobre o desenvolvimento da demanda de calor na região. Por exemplo, uma análise de sensibilidade para um conjunto de diferentes tamanhos de sistema de aquecimento urbano com diferentes tecnologias de abastecimento pode ser comparada. O cenário pode variar de acordo com a capacidade dos geradores de calor no sistema de aquecimento urbano, as taxas de conexão ao sistema de aquecimento urbano ou os custos e preços das diferentes tecnologias. Da mesma forma, para fornecimento descentralizado, vários parâmetros do sistema podem ser variados e os efeitos no sistema de aquecimento geral na região podem ser analisados.</p><p> O usuário executa independentemente os diferentes módulos de cálculos para cada cenário em consideração, copia os resultados nas planilhas relevantes da pasta de trabalho do Excel e, em seguida, os compara para diferentes configurações de cenário. Este procedimento é descrito nas seções a seguir nesta página wiki.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="scenario-assessment-inputs-and-outputs" href="#scenario-assessment-inputs-and-outputs"><i class="fa fa-link"></i></a> Entradas e saídas da avaliação de cenário</h2><p> As entradas compiladas na pasta de trabalho de avaliação de cenário são geradas com os seguintes módulos de cálculo:</p><p> <strong>1. <a href="/en/CM-Decentral-heating-supply">CM - Fornecimento de aquecimento descentral</a></strong></p><p> <strong>2. <a href="/en/CM-District-heating-supply-dispatch">CM - Despacho de abastecimento de aquecimento urbano</a></strong></p><p> <strong>3. <a href="/en/CM-District-heating-potential-economic-assessment">CM - Potencial de aquecimento urbano: avaliação econômica</a></strong></p><p> As entradas e saídas desses módulos devem ser salvas para cada cenário e depois compiladas na pasta de trabalho de avaliação de cenário. As células a serem preenchidas são marcadas corretamente e uma legenda com codificação de cores é relatada na primeira planilha da pasta de trabalho, conforme explicado no parágrafo a seguir <a href="#interpretation-of-input-parameters">Interpretação dos parâmetros de entrada</a> . Os três módulos de cálculo mencionados acima requerem algumas entradas que você já pode ter, ou que você pode ter que calcular executando outros módulos, conforme especificado no parágrafo a seguir, bem como na <strong><a href="/en/training-material">seção Treinamento</a></strong> .</p><p> A planilha “Mesa Final” calcula e apresenta uma comparação dos seguintes indicadores para cada cenário definido tanto nas figuras quanto nos gráficos:</p><ul><li> Custo por ano para diferentes categorias de custos e tecnologias em <em><strong>M EUR / ano</strong></em> ,</li><li> Custos nivelados de fornecimento de calor em <em><strong>EUR / MWh</strong></em> ,</li><li> Emissões de CO2 para as diferentes tecnologias em <em><strong>tCO <sub>2</sub> / ano</strong></em> ,</li><li> Demanda de energia final para as diferentes tecnologias em <em><strong>GWh / ano</strong></em> ,</li><li> Demanda de energia útil para as diferentes tecnologias em <em><strong>GWh / ano</strong></em> ,</li><li> Quotas de energias renováveis (RES) e aquecimento urbano (DH).</li></ul><p> Para cada cenário, os seguintes indicadores são mostrados na forma de um gráfico de barras empilhadas:</p><ul><li> Custos gerais (agregados) de fornecimento de calor por sistema de aquecimento urbano e sistema de aquecimento descentralizado em <em><strong>M EUR / ano</strong></em></li><li> Emissões gerais (agregadas) de CO <sub>2</sub> por sistema de aquecimento urbano e sistema de aquecimento descentralizado em <em><strong>tCO <sub>2</sub> / ano</strong></em></li><li> Energia final geral (agregada) e demanda de energia útil em <em><strong>GWh / ano</strong></em></li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="interpretation-of-input-parameters" href="#interpretation-of-input-parameters"><i class="fa fa-link"></i></a> Interpretação de parâmetros de entrada</h2><p> Os parâmetros de entrada necessários para executar este módulo são divididos em:</p><ul><li> Parâmetros a serem selecionados / preenchidos nos módulos de cálculo da Caixa de Ferramentas: destacados em turquesa na pasta de trabalho.</li><li> Parâmetros a serem selecionados / preenchidos na pasta de trabalho, mas já fornecidos como valores padrão: destacados em verde na pasta de trabalho.</li><li> Parâmetros fornecidos como dados de saída na caixa de ferramentas, a serem copiados para a pasta de trabalho nos campos destacados em amarelo.</li><li> Parâmetros calculados e fornecidos como dados de saída na pasta de trabalho, para serem copiados em outra planilha como entrada para elaboração posterior: destacados em vermelho.</li></ul><p> Além disso, a pasta de trabalho tem instruções em texto vermelho e cálculos adicionais que não devem ser editados em células cinza. A imagem abaixo mostra a aparência do código de cores indicado na folha de rosto da pasta de trabalho.</p><img src="/en/CM-Scenario-assessment/CM-Scenario-Assessment-Colour-Coding.jpg"/><h3><a class="anchor" id="final-table---overview-of-assessment-results" href="#final-table---overview-of-assessment-results"><i class="fa fa-link"></i></a> Mesa Final - Visão geral dos resultados da avaliação</h3><p> A planilha chamada &quot;Mesa Final&quot; mostra principalmente os parâmetros gerais de produção tanto em figuras quanto em gráficos. Além disso, ele possui alguns parâmetros de entrada que precisam ser fornecidos pelo usuário.</p><p> Uma <em>Tabela de Cenários</em> (R4: Z17) está pronta para ser preenchida com uma breve descrição dos diferentes cenários avaliados. Localizada logo acima dos gráficos, esta tabela pode ser um lembrete para o usuário ou esclarecimento para outros colaboradores.</p><p> Mais abaixo na planilha existem alguns parâmetros que já foram preenchidos, mas podem ser ajustados, desde que não fiquem vazios:</p><ul><li> <em>Fator renovável</em> (F102: H116): é usado para calcular a energia renovável final a partir da demanda total de energia final.</li><li> <em>Fator de eficiência geral</em> (K102): é usado para calcular a demanda total de energia útil a partir da demanda total de energia final.</li><li> <em>Perdas da rede</em> (F121: N121): é utilizado para calcular a procura de fornecimento de calor ao sistema de aquecimento urbano a partir da quota de energia útil fornecida pelo aquecimento urbano (ver a figura seguinte).</li></ul><img src="/en/CM-Scenario-assessment/picture3.jpg"/><h3><a class="anchor" id="definition-of-the-building-stock-and-calculation-of-indicators-for-decentral-heating-supply" href="#definition-of-the-building-stock-and-calculation-of-indicators-for-decentral-heating-supply"><i class="fa fa-link"></i></a> Definição do estoque predial e cálculo de indicadores para aquecimento descentralizado</h3><p> Um conjunto de três folhas é usado para descrever o estoque de construção da região sob investigação. As três folhas são utilizadas para recolher os resultados do CM - Fornecimento de aquecimento descentralizado para edifícios individuais, combinar os edifícios individuais a um parque edificado e calcular os indicadores de fornecimento de aquecimento descentralizado necessários para a avaliação do cenário global. O conteúdo e o preenchimento das diferentes fichas são descritos a seguir.</p><p> <strong>1. Edifícios unifamiliares</strong> Esta folha de trabalho é preparada para recolher os resultados do <a href="/en/CM-Decentral-heating-supply">CM - Fornecimento de aquecimento descentral</a> para edifícios unifamiliares. Dois tipos diferentes de edifícios unifamiliares podem ser considerados na versão padrão da pasta de trabalho. No entanto, o usuário pode adaptar a pasta de trabalho para combinar qualquer número de diferentes edifícios unifamiliares (por exemplo, de diferentes períodos de construção ou diferentes tipos, como independente ou geminada).</p><p> <strong>2. Edifícios MF e não res</strong> Esta planilha é preparada para coletar os resultados do <a href="/en/CM-Decentral-heating-supply">CM - Fornecimento de aquecimento descentral</a> para edifícios multifamiliares e não residenciais (não res). 8 tipos diferentes de edifícios multifamiliares e não residenciais podem ser considerados na versão padrão da pasta de trabalho. No entanto, o usuário pode adaptar a pasta de trabalho a fim de combinar qualquer número de diferentes edifícios multifamiliares e não residenciais (por exemplo, de diferentes períodos de construção ou diferentes tipos, como pequenos edifícios multifamiliares, blocos de apartamentos, escritórios, hotéis e restaurantes, educação , etc.).</p><p> <strong>Estoque de construção e fornecimento de dezembro</strong> Nesta ficha, o estoque de construção da região sob investigação é compilado a partir dos tipos de construção definidos e calculados nas duas planilhas anteriores (C15: L15). Além disso, é definida a divisão das tecnologias presumidas para serem utilizadas nos diferentes tipos de edifícios (C17: L25) e são calculados os indicadores de abastecimento de aquecimento descentralizado no parque imobiliário (E44: E77). Para isso, também a demanda geral de calor na região sob investigação (E9), bem como o preço de CO <sub>2</sub> assumido (E10), devem ser definidos pelo usuário.</p><h3><a class="anchor" id="ref.-decentral-dec.-heating-supply" href="#ref.-decentral-dec.-heating-supply"><i class="fa fa-link"></i></a> Ref. Fornecimento de aquecimento descentralizado (dezembro)</h3><p> Nesta planilha, vários cenários para o fornecimento de aquecimento descentralizado na região sob investigação podem ser definidos. Para fazer isso, os resultados do cálculo na planilha anterior (Estoque de construção e fornecimento de dezembro, células E44: E77) podem ser copiados para as células marcadas em amarelo nesta planilha. A planilha em sua versão padrão é preparada para compilar 9 cenários diferentes. O usuário é livre para decidir quais cenários para o fornecimento descentralizado de aquecimento são coletados nesta ficha (por exemplo, diferentes tecnologias divididas em diferentes tipos de edifícios, diferentes preços de CO <sub>2</sub> ou diferentes demandas de calor na região).</p><h3><a class="anchor" id="ref.-dh-economic-assessment" href="#ref.-dh-economic-assessment"><i class="fa fa-link"></i></a> Ref. Avaliação econômica DH</h3><p> Nesta planilha, vários cenários para a construção de redes de aquecimento urbano na região sob investigação podem ser definidos. Para isso, o usuário executa diferentes cenários com o <a href="/en/CM-District-heating-potential-economic-assessment">CM - Potencial de aquecimento urbano: avaliação econômica</a> e copia os resultados do módulo da caixa de ferramentas para esta planilha (células C29: T44). De um conjunto mais amplo de cenários calculados, o usuário escolhe 9 cenários a serem usados para a avaliação geral do cenário (C52: K67). O usuário é livre para decidir quais cenários para a construção de redes de aquecimento urbano são coletados nesta ficha (por exemplo, diferentes participações de mercado do aquecimento urbano em áreas de aquecimento urbano, diferentes tetos de custo da rede ou diferentes demandas gerais de calor na região).</p><h3><a class="anchor" id="ref.-dh-supply-dispatch" href="#ref.-dh-supply-dispatch"><i class="fa fa-link"></i></a> Ref. Despacho de suprimentos DH</h3><p> Nesta planilha, vários cenários para o fornecimento de calor a potenciais sistemas de aquecimento urbano podem ser definidos. Para isso, o usuário executa diferentes cenários com o <a href="/en/CM-District-heating-supply-dispatch">CM - Despacho de abastecimento de aquecimento urbano</a> e copia os resultados do módulo da caixa de ferramentas para esta planilha (células D84: L98 para os resultados na seção denominada &quot;INDICADORES&quot; e células D104: L195 para os resultados na seção chamada &quot;GRÁFICOS&quot;). O usuário é livre para decidir quais cenários para o fornecimento de calor a potenciais sistemas de aquecimento urbano são coletados nesta ficha (por exemplo, diferentes combinações de tecnologias, diferentes demandas de calor nos sistemas de aquecimento urbano, diferentes preços de CO <sub>2</sub> ou diferentes preços de portadores de energia) .</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="interpretation-of-results" href="#interpretation-of-results"><i class="fa fa-link"></i></a> interpretação de resultados</h2><p> Após o preenchimento dos dados do cenário nas três planilhas (Ref. Dec. abastecimento de aquecimento, Ref. DH avaliação econômica, Ref. DH expedição de abastecimento), os indicadores para os cenários definidos são calculados automaticamente na planilha &quot;Tabela final&quot;. A tabela nas células B7: N88 mostra os resultados desses cenários gerais. Os indicadores são calculados para custos, emissões, demanda de energia, bem como parcelas de energia renovável e aquecimento urbano.</p><img src="/en/CM-Scenario-assessment/picture1.jpg"/><p> Em geral, todos estes resultados são calculados individualmente para a descentralização e também para a área abastecida com aquecimento urbano e são aplicados ao cenário global com base nas respetivas quotas. Portanto, eles são listados separadamente como componentes individuais do respectivo parâmetro.</p><h3><a class="anchor" id="costs" href="#costs"><i class="fa fa-link"></i></a> Custos</h3><p> O arquivo de avaliação de cenário mostra uma divisão dos custos gerais em despesas de capital (CAPEX), despesas operacionais (OPEX), despesas com energia e também com CO <sub>2</sub> . Essas diferentes categorias de custos são apresentadas tanto para o fornecimento central quanto para o descentralizado na região. Além disso, são apresentados os custos da rede de distribuição. Todos esses componentes de custo são apresentados em <em><strong>M EUR / ano</strong></em> . Além dos custos anuais globais, também os custos nivelados do fornecimento de calor (LCOH) são mostrados na tabela. Eles são apresentados em <em><strong>EUR / MWh.</strong></em> Ambos os custos anuais, bem como o LCOH, também são apresentados em números. Eles podem ser encontrados ao lado da tabela, conforme mostrado na próxima figura.</p><img src="/en/CM-Scenario-assessment/picture2_new.jpg"/><h3><a class="anchor" id="emissions" href="#emissions"><i class="fa fa-link"></i></a> Emissões</h3><p> O arquivo de avaliação de cenário também fornece as quantidades absolutas de emissões de CO <sub>2</sub> nos diferentes cenários. Os valores são apresentados para as diferentes tecnologias de abastecimento, distinguindo entre abastecimento individual (descentralizado) e abastecimento de aquecimento urbano (central). Os valores são dados em <em><strong>tCO <sub>2</sub> / ano</strong></em> . Novamente, as quantidades de emissões também são mostradas na forma de um diagrama ao lado da tabela.</p><h3><a class="anchor" id="energy-demand-final-and-useful-energy" href="#energy-demand-final-and-useful-energy"><i class="fa fa-link"></i></a> Demanda de energia (energia final e útil)</h3><p> Na tabela também são apresentadas as demandas de energia final e útil. Na demanda final de energia também são contidas as perdas no sistema de aquecimento urbano. Portanto, os valores finais da demanda de energia para o fornecimento de aquecimento urbano (central) refletem a demanda do portador de energia nas usinas do sistema de aquecimento urbano. Ambos, os valores de demanda de energia útil e final são apresentados em <em><strong>GWh / ano</strong></em> .</p><h3><a class="anchor" id="shares-of-renewable-energy-and-district-heating" href="#shares-of-renewable-energy-and-district-heating"><i class="fa fa-link"></i></a> Ações de energia renovável e aquecimento urbano</h3><p> Por fim, a tabela oferece uma comparação de quatro parcelas diferentes: a parcela das energias renováveis no abastecimento descentralizado, na central e na oferta geral de calor na área, e a parcela do aquecimento urbano / central na área. Quanto aos demais valores, esses valores também são apresentados em figuras ao lado da tabela.</p><img src="/en/CM-Scenario-assessment/picture4.jpg"/><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="sample-run" href="#sample-run"><i class="fa fa-link"></i></a> Execução de amostra</h2><p> (Tomaszów Mazowiecki)</p><p> Com o download do arquivo Excel, você receberá uma versão pronta para preencher da avaliação do cenário. Após o cálculo bem sucedido dos cenários utilizando os módulos CM - Fornecimento de aquecimento descentral, CM - Potencial de aquecimento urbano: avaliação económica e CM - Despacho de fornecimento de aquecimento urbano, os respectivos resultados dos módulos podem ser copiados para as folhas correspondentes do documento. Nos três cenários a seguir para a cidade polonesa Tomaszów Mazowiecki são avaliados como um exemplo.</p><h3><a class="anchor" id="ref.-dec.-heating-supply" href="#ref.-dec.-heating-supply"><i class="fa fa-link"></i></a> Ref. Fornecimento de aquecimento de dezembro</h3><p> Neste exemplo, um cenário de fornecimento de aquecimento descentralizado para a cidade de Tomaszów Mazowiecki na Polônia foi calculado usando a caixa de ferramentas Hotmaps junto com as folhas na seção estoque de construção de acordo com o procedimento descrito <a href="CM-Scenario-assessment#interpretation-of-input-parameters_definition-of-the-building-stock-and-calculation-of-indicators-for-decentral-heating-supply">nesta seção da página</a> . Este cenário para fornecimento descentralizado foi copiado para os cenários 1, 2 e 3 no arquivo de avaliação de cenário, planilha &quot;Ref. Dez. fornecimento de aquecimento&quot;, a fim de usar o mesmo cenário de fornecimento de aquecimento individual em todos os três cenários gerais da cidade que são comparados .</p><img src="/en/CM-Scenario-assessment/sample_run_picture1.jpg"/><h3><a class="anchor" id="ref.-dh-economic-assessment" href="#ref.-dh-economic-assessment"><i class="fa fa-link"></i></a> Ref. Avaliação econômica DH</h3><p> Neste exemplo, um cenário de construção de rede de aquecimento distrital para a cidade de Tomaszów Mazowiecki na Polônia foi calculado usando a caixa de ferramentas Hotmaps. Este cenário foi copiado para os cenários 1, 2 e 3 no arquivo de avaliação de cenário, planilha &quot;Ref. Avaliação econômica DH&quot; para usar o mesmo cenário de construção de rede de aquecimento distrital em todos os três cenários gerais da cidade que são comparados.</p><img src="/en/CM-Scenario-assessment/sample_run_picture2.jpg"/><h3><a class="anchor" id="ref.-dh-supply-dispatch" href="#ref.-dh-supply-dispatch"><i class="fa fa-link"></i></a> Ref. Despacho de suprimentos DH</h3><p> Neste exemplo, três cenários diferentes de fornecimento de aquecimento urbano para a cidade de Tomaszów Mazowiecki, na Polônia, foram calculados usando a caixa de ferramentas Hotmaps. Os cenários refletem diferentes combinações de tecnologias de abastecimento, nomeadamente diferentes capacidades de bombas de calor, excesso de calor e caldeiras apenas de calor.</p><img src="/en/CM-Scenario-assessment/sample_run_picture3.jpg"/><h3><a class="anchor" id="final-table" href="#final-table"><i class="fa fa-link"></i></a> Mesa final</h3><p> Depois de copiar os dados individuais, os cenários compilados podem ser examinados na Tabela Final da planilha. Por um lado, existe uma tabela que mostra os resultados exatos e, por outro lado, você pode encontrar os gráficos correspondentes no lado direito da tabela. Neste exemplo, os efeitos do uso de diferentes capacidades de tecnologias de abastecimento no sistema de aquecimento urbano podem ser avaliados com base nos custos, emissões, demanda de energia e
