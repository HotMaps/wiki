href="#introduction"><i class="fa fa-link"></i></a> Introdução</h2><p> Essas diretrizes descrevem como a caixa de ferramentas Hotmaps pode ser usada para analisar custos e potenciais para aquecimento e resfriamento eficientes e renováveis em nível nacional. <em><strong>O guia é especialmente orientado para o desenvolvimento de resultados de acordo com a avaliação abrangente dos potenciais nacionais de aquecimento e refrigeração a que se refere o artigo 14.º, n.º 1, da Diretiva de Eficiência Energética (DEE) na sua versão atual e tem em conta a recomendação da Comissão (UE ) 2019/1659 sobre este procedimento.</strong></em></p><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code></ins></p><h2><a class="anchor" id="part-i--overview-of-heating-and-cooling" href="#part-i--overview-of-heating-and-cooling"><i class="fa fa-link"></i></a> Parte I: Visão geral do aquecimento e resfriamento</h2><p> Os conjuntos de dados e caixa de ferramentas Hotmaps fornecem duas entradas diferentes para esta parte: primeiro, os conjuntos de dados Hotmaps fornecem dados padrão para vários dos dados necessários para incluir nesta parte I da avaliação abrangente. Em segundo lugar, a caixa de ferramentas Hotmaps é basicamente uma ferramenta de mapeamento que não apenas permite uma representação geográfica dos dados padrão na caixa de ferramentas; mas também, o usuário carregou dados para a conta de uso na caixa de ferramentas. A seguir, descrevemos os diferentes Hotmaps de formulário de dados padrão e vinculamos as descrições de como usar a função de upload da caixa de ferramentas de Hotmaps.</p><p> Os seguintes dados relevantes para a Parte I do Anexo VIII estão disponíveis no banco de dados Hotmaps:</p><ul><li> Ponto 2: fornecimento atual de aquecimento e refrigeração<ul><li> (b) (v) instalações industriais:<ul><li> <a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_industryBenchmarks">Excesso de calor em locais industriais</a></li><li> <a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">Nomes de empresas de sites industriais</a></li><li> <a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_industryBenchmarks">Subsetor de sites industriais</a></li></ul></li><li> Todos os outros:<ul><li> Nenhum dado padrão está contido para pontos de abastecimento, mas os próprios dados podem ser carregados e exibidos</li><li> <a href="https://wiki.hotmaps.eu/en/Introduction-to-user-interface">Como criar uma conta</a></li><li> <a href="https://wiki.hotmaps.eu/en/Data-upload-functionalities">Como fazer upload dos próprios dados para a caixa de ferramentas</a></li></ul></li></ul></li><li> Ponto 3: um mapa cobrindo todo o território nacional<ul><li> (a) Áreas de demanda de aquecimento e resfriamento (não para demanda industrial)<ul><li> <a href="https://gitlab.com/hotmaps/heat/heat_res_curr_density">Densidade de calor residencial</a></li><li> <a href="https://gitlab.com/hotmaps/heat/heat_nonres_curr_density">Densidade de calor não residencial</a></li><li> <a href="https://gitlab.com/hotmaps/heat/heat_tot_curr_density">Densidade de calor total</a></li></ul></li><li> (b) + (c) Pontos de abastecimento existentes e planejados<ul><li> Nenhum dado padrão está contido para pontos de abastecimento, mas os próprios dados podem ser carregados e exibidos</li><li> <a href="https://wiki.hotmaps.eu/en/Introduction-to-user-interface">Como criar uma das tendências da demanda por aquecimento e refrigeração<ul><li> Os cenários padrão para todos os EU28 MS estão disponíveis no projeto H2020 <a href="http://www.cheetah-project.eu/">CHEETAH</a></li><li> Esses dados também são integrados na <a href="https://wiki.hotmaps.eu/en/CM-Demand-projection">projeção CM - Demanda</a> e usados para os cálculos neste módulo</li></ul></li></ul><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code></ins></p><h2><a class="anchor" id="part-iii--analysis-of-the-economic-potential-for-efficiency-in-heating-and-cooling" href="#part-iii--analysis-of-the-economic-potential-for-efficiency-in-heating-and-cooling"><i class="fa fa-link"></i></a> Parte III: Análise do potencial econômico para eficiência em aquecimento e resfriamento</h2><p> A caixa de ferramentas Hotmaps contém vários módulos de cálculo (CMs) que podem ser usados para analisar o potencial econômico de eficiência em aquecimento e resfriamento. A seguir, uma possível abordagem para gerar resultados para a avaliação abrangente com a caixa de ferramentas Hotmaps é descrita também vinculando aos respectivos dados padrão e módulos de cálculo.</p><h3><a class="anchor" id="overview-of-the-hotmaps-approach" href="#overview-of-the-hotmaps-approach"><i class="fa fa-link"></i></a> Visão geral da abordagem Hotmaps</h3><p> Para avaliar o potencial econômico de eficiência em aquecimento e resfriamento, é importante distinguir entre áreas potencialmente abastecidas por aquecimento urbano e áreas onde o fornecimento descentralizado provavelmente será mais eficiente economicamente. Assim, <strong>a abordagem Hotmaps é construída fortemente nas quatro etapas a seguir</strong> :</p><ol><li><p> Identificar diferentes representativos, casos típicos de aquecimento urbano no país / região sob investigação;</p></li><li><p> Realizar análises da construção / expansão da rede de aquecimento urbano e do fornecimento de calor urbano para os casos representativos identificados;</p></li><li><p> Calcular indicadores de fornecimento de calor descentralizado;</p></li><li><p> Compare diferentes cenários de aquecimento urbano e fornecimento de calor descentralizado e cálculos de sensibilidade.</p></li></ol><p> A figura a seguir mostra essa abordagem graficamente. As diferentes etapas serão explicadas com mais detalhes nos capítulos seguintes dessas diretrizes.</p><img alt="" src="../images/Hotmaps_ApproachNational_Overview.png"/><p> <em>Figura: Abordagem Hotmaps para analisar o potencial econômico para a eficiência em aquecimento e refrigeração no decorrer do artigo 14 da Diretiva de Eficiência Energética (EED)</em></p><p> Em todas essas etapas, <strong>vários cenários e sensibilidades devem ser levados em consideração</strong> :</p><ul><li> Diferentes níveis de economia de calor (implementado nas etapas 1, 2 e 3)</li><li> Diferentes níveis de quotas de aquecimento urbano no fornecimento total de calor (implementado na etapa 2)</li><li> Diferentes preços de energia futuros (implementados nas etapas 2 e 3)</li><li> Diferentes tempos de depreciação e taxas de desconto (cálculos socioeconômicos vs. cálculos econômicos privados) (implementados nas etapas 2 e 3)</li></ul><p> Os seguintes <strong>indicadores</strong> resultantes <strong>podem ser recuperados dos Módulos de Cálculo de Hotmaps</strong> (CMs):</p><ul><li> Potencial econômico:<ul><li> Custos nivelados de calor (LCOH) [EUR / MWh]:<ul><li> <a href="https://wiki.hotmaps.eu/en/CM-District-heating-supply-dispatch">CM - Despacho do fornecimento de aquecimento urbano</a> para custos de fornecimento de calor ao aquecimento urbano</li><li> <a href="https://wiki.hotmaps.eu/en/CM-Decentral-heating-supply">CM - Fornecimento de aquecimento descentralizado</a> para custos de fornecimento de calor descentralizado</li><li> <a href="https://wiki.hotmaps.eu/en/CM-Excess-heat-transport-potential">CM - Potencial</a> de transporte de calor em excesso para custos de transporte de calor em excesso para redes de aquecimento urbano potenciais</li></ul></li><li> Custos específicos da rede de aquecimento urbano (expansão e / ou nova construção) [EUR / MWh]:<ul><li> <a href="https://wiki.hotmaps.eu/en/CM-District-heating-potential-economic-assessment">CM - Potencial de aquecimento urbano: avaliação econômica</a></li></ul></li></ul></li><li> Emissões de GEE:<ul><li> Emissões de CO2 [kt]:<ul><li> <a href="https://wiki.hotmaps.eu/en/CM-District-heating-supply-dispatch">CM - Despacho de fornecimento de aquecimento urbano</a> para emissões de CO2 do fornecimento de calor para aquecimento urbano</li><li> <a href="https://wiki.hotmaps.eu/en/CM-Decentral-heating-supply">CM - Fornecimento de aquecimento descentralizado</a> para emissões de CO2 do fornecimento de calor descentralizado</li></ul></li></ul></li><li> O impacto na participação de FER pode ser calculado com base nos resultados dos seguintes MCs:<ul><li> <a href="https://wiki.hotmaps.eu/en/CM-District-heating-supply-dispatch">CM - Despacho do fornecimento de aquecimento urbano</a> para a parcela de FER do fornecimento de calor ao aquecimento urbano</li><li> <a href="https://wiki.hotmaps.eu/en/CM-Decentral-heating-supply">CM - Fornecimento de aquecimento descentralizado</a> para a parcela de FER do fornecimento de calor descentralizado</li></ul></li></ul><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code></ins></p><h3><a class="anchor" id="different-steps" href="#different-steps"><i class="fa fa-link"></i></a> Etapas Diferentes</h3><ul><li> <a href="https://wiki.hotmaps.eu/en/Step-1-Identification-of-different-representative-cases-for-district-heating">Etapa 1: Identificação de diferentes casos representativos para aquecimento urbano</a><ul><li> Cálculo da demanda de calor futura e mapas de densidade de área de construção</li><li> Identificação de áreas potencialmente interessantes para aquecimento urbano</li><li> Identificação de áreas representativas típicas de aquecimento urbano para análise posterior</li></ul></li><li> <a representativas</a><ul><li> Avaliação econômica do potencial de aquecimento urbano</li><li> Estimativa dos custos de transporte do excesso de calor para áreas de aquecimento urbano</li><li> Desenvolvimento de perfis de carga térmica futuros</li><li> Cálculo dos custos e emissões do fornecimento de calor no aquecimento urbano</li></ul></li><li> <a
