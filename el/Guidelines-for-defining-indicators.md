<h1><a class="anchor" id="guidelines-for-defining-indicators" href="#guidelines-for-defining-indicators"><i class="fa fa-link"></i></a>Οδηγίες για τον καθορισμό δεικτών</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Πίνακας περιεχομένων</h2><ul><li> <a href="#indicators">Δείκτες</a><ul><li> <a href="#indicators_simple-indicators">Απλοί δείκτες</a></li><li> <a href="#indicators_cross-indicators">Διασταυρούμενοι δείκτες</a></li></ul></li><li> <a href="#indicator-result">Αποτέλεσμα δείκτη</a></li><li> <a href="#how-to-cite">Πώς να παραθέσω</a></li><li> <a href="#authors-and-reviewers">Συγγραφείς και κριτικοί</a></li><li> <a href="#license">Αδεια</a></li><li> <a href="#acknowledgement">Αναγνώριση</a></li></ul><h2><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Δείκτες</h2><p> Αυτή η σελίδα εξηγεί τον τρόπο διαχείρισης των δεικτών στην εργαλειοθήκη Hotmaps. Μια ένδειξη είναι μια τιμή που εμφανίζεται στην εργαλειοθήκη Hotmaps στην πλευρά του πελάτη. Ο δείκτης είναι μέρος ενός επιπέδου. Ένας δείκτης είναι μια τιμή, για μια συγκεκριμένη χωρική περιοχή, με μια μονάδα. Ένας δείκτης είναι μέρος ενός λεξικού επιπέδου. Ένα αντικείμενο επιπέδου περιέχει έναν πίνακα που ονομάζεται &quot;δείκτες&quot;. Αυτός ο πίνακας θα χρησιμοποιηθεί για τη λίστα όλων των δεικτών που θέλετε να δείτε για το επίπεδο.</p><pre> <code>layers = { &#39;heat_tot_curr_density_tif&#39;:{ &#39;tablename&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;from_indicator_name&#39;:&#39;stat_heat_tot_curr_density_tif&#39;, &#39;schema_scalelvl&#39;: &#39;stat&#39;, &#39;schema_hectare&#39;: &#39;geo&#39;, &#39;crs&#39;: &#39;3035&#39;, &#39;geo_column&#39;: &#39;geometry&#39;, &#39;table_type&#39;:&#39;raster&#39;, &#39;data_lvl&#39;:[&#39;NUTS 0&#39;,&#39;NUTS 1&#39;,&#39;NUTS 2&#39;,&#39;NUTS 3&#39;,&#39;LAU 2&#39;,&#39;Hectare&#39;], &#39;data_aggregated&#39;:True, &#39;scalelvl_column&#39;:&#39;&#39;, &#39;indicators&#39;:[ {&#39;table_column&#39;: &#39;sum&#39;, &#39;unit&#39;: &#39;MWh&#39;,&#39;indicator_id&#39;:&#39;consumption&#39;}, {&#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;,&#39;indicator_id&#39;:&#39;count_cell&#39;}, { &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;: &#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;: &#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;: &#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;: &#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; } ] } }</code></pre><ul><li> &quot;όνομα tablen&quot;</li></ul><p> Όνομα του πίνακα DB. (Παράδειγμα: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &quot;from_indicator_name&quot;</li></ul><p> Subtablename για την επιλογή των δεικτών. <strong>Πρέπει να είναι μοναδικό.</strong> (Παράδειγμα: &#39;stat_heat_tot_curr_density_tif&#39;)</p><ul><li> &quot;δεδομένα_συγκεντρωτικά&quot;</li></ul><p> Είναι τα δεδομένα συγκεντρωτικά ή όχι (Τιμές: True ή False)</p><ul><li> &quot;scalelvl_column&quot;</li></ul><p> Όνομα στήλης επιπέδου κλίμακας εάν είναι διαφορετικό από το προεπιλεγμένο (Παράδειγμα: &#39;code&#39;)</p><ul><li> «δεδομένα_lvl»</li></ul><p> Επίπεδα διαθέσιμα για τα δεδομένα στη βάση δεδομένων</p><ul><li> &quot;schema_scalelvl&quot;</li></ul><p> Θέση σχήματος πίνακα για το επίπεδο ξηρών καρπών. (Παράδειγμα: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> &quot;schema_hectare&quot;</li></ul><p> Θέση σχήματος πίνακα για το επίπεδο εκταρίου. (Παράδειγμα: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> «crs»</li></ul><p> Προβολή της γεωμετρίας (Παράδειγμα: &#39;3035&#39;, &#39;4326&#39;, &#39;4258&#39;)</p><ul><li> &quot;geo_column&quot;</li></ul><p> Όνομα της στήλης γεωμετρίας στη βάση δεδομένων (Παράδειγμα: «geom», «γεωμετρία»)</p><ul><li> &quot;table_type&quot;</li></ul><p> Τύπος επιπέδου στη βάση δεδομένων (Τιμές: «διάνυσμα» ή «ράστερ»).</p><p> <em><strong>Σημαντικό:</strong></em> Εάν είναι ράστερ, οι διαθέσιμες στήλες είναι <strong>μετρήσεις, άθροισμα, μέσος όρος, stddev, ελάχιστο και μέγιστο</strong></p><ul><li> «Δείκτες»</li></ul><p> Υπάρχουν 2 τύποι δεικτών (Απλοί δείκτες &amp; Διασταυρούμενοι δείκτες).</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="simple-indicators" href="#simple-indicators"><i class="fa fa-link"></i></a> Απλοί δείκτες</h3><p> Ένας απλός δείκτης είναι ένα αντικείμενο με 3 παραμέτρους.</p><pre> <code>{ &#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;, &#39;indicator_id&#39;:&#39;count_cell&#39; }</code></pre><ul><li> &quot;πίνακας_ στήλη&quot;</li></ul><p> Αυτή είναι η στήλη πίνακα που επιλέγεται στον πίνακα. (Παράδειγμα: &#39;count&#39;)</p><img alt="Επιλογή πίνακα" src="https://github.com/HotMaps/Hotmaps-toolbox-service/raw/master/api/assets/table_image.png"/><ul><li> &#39;μονάδα&#39;</li></ul><p> Αυτή είναι η μονάδα του δείκτη. (Παράδειγμα: «κελιά», «MWh»)</p><ul><li> &quot;ένδειξη_id&quot;</li></ul><p> Αυτό είναι το αναγνωριστικό δείκτη του δείκτη (Όπως ένα αναγνωριστικό). Αυτό το όνομα <strong>πρέπει να είναι μοναδικό</strong> στη σειρά δεικτών.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="cross-indicators" href="#cross-indicators"><i class="fa fa-link"></i></a> Διασταυρούμενοι δείκτες</h3><p> Ένας σταυρός δείκτης είναι ένα αντικείμενο με 7 παραμέτρους. Ο στόχος αυτού του δείκτη είναι να κάνει έναν υπολογισμό μεταξύ απλών δεικτών.</p><pre> <code>{ &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;:&#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;:&#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;:&#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; }</code></pre><ul><li> &quot;αναφορά_indicator_id_1&quot;</li></ul><p> Αντιστοιχεί στο αναγνωριστικό ενός απλού δείκτη. Αυτό το όνομα <strong>πρέπει να οριστεί</strong> στη συστοιχία δεικτών. Είναι η τιμή αριθμός 1.</p><ul><li> &quot;Reference_tablename_indicator_id_1&quot;</li></ul><p> Αναφορά του ονόματος καρτέλας επιπέδου που αναφέρεται στον αριθμό τιμής 1. (Παράδειγμα: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;χειριστής&#39;</li></ul><p> Κανόνας υπολογισμού που εφαρμόζεται στις 2 τιμές (Τιμές: &#39;/&#39; ή &#39;*&#39; ή &#39;+&#39; ή &#39;-&#39;)</p><ul><li> &quot;αναφορά_indicator_id_1&quot;</li></ul><p> Αντιστοιχεί στο αναγνωριστικό ενός απλού δείκτη. Αυτό το όνομα <strong>πρέπει να οριστεί</strong> στη συστοιχία δεικτών. Είναι τιμή 2.</p><ul><li> &quot;reference_tablename_indicator_id_2&quot;</li></ul><p> Αναφορά του ονόματος του επιπέδου στρώματος που αναφέρεται στον αριθμό 2. (Παράδειγμα: &#39;pop_tot_curr_density_tif&#39;)</p><ul><li> &#39;μονάδα&#39;</li></ul><p> Αυτή είναι η μονάδα του δείκτη. (Παράδειγμα: «κελιά», «MWh»)</p><ul><li> &#39;όνομα&#39;</li></ul><p> Αυτό είναι το όνομα του δείκτη (Όπως ένα αναγνωριστικό). Αυτό το όνομα <strong>πρέπει να είναι μοναδικό</strong> στη σειρά δεικτών.</p><h5><a class="anchor" id="note--for-this-example,-the-calculation-below-is-done." href="#note--for-this-example,-the-calculation-below-is-done."><i class="fa fa-link"></i></a> Σημείωση: Για αυτό το παράδειγμα, ο παρακάτω υπολογισμός έχει γίνει.</h5><pre> <code>reference_tablename_indicator_id_1.reference_indicator_id_1 / reference_tablename_indicator_id_2.reference_indicator_id_2 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code></pre><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="indicator-result" href="#indicator-result"><i class="fa fa-link"></i></a> Αποτέλεσμα δείκτη</h2><p> Το αποτέλεσμα των δεικτών έχει ως εξής:</p><pre> <code>{ &quot;values&quot;: [ { &quot;unit&quot;: &quot;MWh&quot;, &quot;name&quot;: &quot;heat_tot_curr_density_tif_consumption&quot;, &quot;value&quot;: &quot;4112030.46&quot;
