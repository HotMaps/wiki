<h1><a class="anchor" id="guidelines-for-defining-indicators" href="#guidelines-for-defining-indicators"><i class="fa fa-link"></i></a>Lineamientos para definir indicadores</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Tabla de contenido</h2><ul><li> <a href="#indicators">Indicadores</a><ul><li> <a href="#indicators_simple-indicators">Indicadores simples</a></li><li> <a href="#indicators_cross-indicators">Indicadores cruzados</a></li></ul></li><li> <a href="#indicator-result">Resultado del indicador</a></li><li> <a href="#how-to-cite">Cómo citar</a></li><li> <a href="#authors-and-reviewers">Autores y revisores</a></li><li> <a href="#license">Licencia</a></li><li> <a href="#acknowledgement">Reconocimiento</a></li></ul><h2><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indicadores</h2><p> Esta página explica cómo administrar indicadores en la caja de herramientas Hotmaps. Un indicador es un valor que se muestra en la caja de herramientas Hotmaps en el lado del cliente. El indicador es parte de una capa. Un indicador es un valor, para una región espacial específica, con una unidad. Un indicador es parte de un diccionario de capas. Un objeto de capa contiene una matriz denominada &quot;indicadores&quot;. Esta tabla se utilizará para enumerar todos los indicadores que desea ver para la capa.</p><pre> <code>layers = { &#39;heat_tot_curr_density_tif&#39;:{ &#39;tablename&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;from_indicator_name&#39;:&#39;stat_heat_tot_curr_density_tif&#39;, &#39;schema_scalelvl&#39;: &#39;stat&#39;, &#39;schema_hectare&#39;: &#39;geo&#39;, &#39;crs&#39;: &#39;3035&#39;, &#39;geo_column&#39;: &#39;geometry&#39;, &#39;table_type&#39;:&#39;raster&#39;, &#39;data_lvl&#39;:[&#39;NUTS 0&#39;,&#39;NUTS 1&#39;,&#39;NUTS 2&#39;,&#39;NUTS 3&#39;,&#39;LAU 2&#39;,&#39;Hectare&#39;], &#39;data_aggregated&#39;:True, &#39;scalelvl_column&#39;:&#39;&#39;, &#39;indicators&#39;:[ {&#39;table_column&#39;: &#39;sum&#39;, &#39;unit&#39;: &#39;MWh&#39;,&#39;indicator_id&#39;:&#39;consumption&#39;}, {&#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;,&#39;indicator_id&#39;:&#39;count_cell&#39;}, { &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;: &#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;: &#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;: &#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;: &#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; } ] } }</code></pre><ul><li> &#39;nombre de la tabla&#39;</li></ul><p> Nombre de la tabla DB. (Ejemplo: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;from_indicator_name&#39;</li></ul><p> Nombre de subtabla para la selección de indicadores. <strong>Debe ser único.</strong> (Ejemplo: &#39;stat_heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;data_aggregated&#39;</li></ul><p> ¿Los datos están agregados o no (valores: verdadero o falso)?</p><ul><li> &#39;scalelvl_column&#39;</li></ul><p> El nombre de la columna de nivel de escala si es diferente al predeterminado (Ejemplo: &#39;código&#39;)</p><ul><li> &#39;data_lvl&#39;</li></ul><p> Niveles disponibles para los datos en la base de datos</p><ul><li> &#39;schema_scalelvl&#39;</li></ul><p> Ubicación del esquema de la mesa para el nivel de las nueces. (Ejemplo: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> &#39;schema_hectare&#39;</li></ul><p> Ubicación del esquema de tabla para el nivel de hectárea. (Ejemplo: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> &#39;crs&#39;</li></ul><p> Proyección de la geometría (Ejemplo: &#39;3035&#39;, &#39;4326&#39;, &#39;4258&#39;)</p><ul><li> &#39;geo_column&#39;</li></ul><p> Nombre de la columna de geometría en la base de datos (Ejemplo: &#39;geom&#39;, &#39;geometría&#39;)</p><ul><li> &#39;table_type&#39;</li></ul><p> Tipo de capa en la base de datos (Valores: &#39;vector&#39; o &#39;raster&#39;).</p><p> <em><strong>Importante:</strong></em> si es un ráster, las columnas disponibles son <strong>count, sum, mean, stddev, min y max</strong></p><ul><li> &#39;Indicadores&#39;</li></ul><p> Hay 2 tipos de indicadores (indicadores simples e indicadores cruzados).</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="simple-indicators" href="#simple-indicators"><i class="fa fa-link"></i></a> Indicadores simples</h3><p> Un indicador simple es un objeto con 3 parámetros.</p><pre> <code>{ &#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;, &#39;indicator_id&#39;:&#39;count_cell&#39; }</code></pre><ul><li> &#39;columna_tabla&#39;</li></ul><p> Esta es la columna de la tabla que se selecciona en la tabla. (Ejemplo: &#39;contar&#39;)</p><img alt="selección de mesa" src="https://github.com/HotMaps/Hotmaps-toolbox-service/raw/master/api/assets/table_image.png"/><ul><li> &#39;unidad&#39;</li></ul><p> Esta es la unidad del indicador. (Ejemplo: &#39;celdas&#39;, &#39;MWh&#39;)</p><ul><li> &#39;id_indicador&#39;</li></ul><p> Este es el identificador de indicador del indicador (como una ID). Este nombre <strong>debe ser único</strong> en la matriz de indicadores.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="cross-indicators" href="#cross-indicators"><i class="fa fa-link"></i></a> Indicadores cruzados</h3><p> Un indicador cruzado es un objeto con 7 parámetros. El objetivo de este indicador es realizar un cálculo entre indicadores simples.</p><pre> <code>{ &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;:&#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;:&#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;:&#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; }</code></pre><ul><li> &#39;reference_indicator_id_1&#39;</li></ul><p> Corresponde al identificador de un indicador simple. Este nombre <strong>debe estar definido</strong> en la matriz de indicadores. Es el valor número 1.</p><ul><li> &#39;reference_tablename_indicator_id_1&#39;</li></ul><p> Referencia del nombre de la tabla de la capa que hace referencia al valor número 1. (Ejemplo: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;operador&#39;</li></ul><p> Regla de cálculo para aplicar a los 2 valores (Valores: &#39;/&#39; o &#39;*&#39; o &#39;+&#39; o &#39;-&#39;)</p><ul><li> &#39;reference_indicator_id_1&#39;</li></ul><p> Corresponde al identificador de un indicador simple. Este nombre <strong>debe estar definido</strong> en la matriz de indicadores. Es el valor número 2.</p><ul><li> &#39;reference_tablename_indicator_id_2&#39;</li></ul><p> Referencia del nombre de la tabla de la capa que hace referencia al valor número 2. (Ejemplo: &#39;pop_tot_curr_density_tif&#39;)</p><ul><li> &#39;unidad&#39;</li></ul><p> Esta es la unidad del indicador. (Ejemplo: &#39;celdas&#39;, &#39;MWh&#39;)</p><ul><li> &#39;nombre&#39;</li></ul><p> Este es el nombre del indicador (como una identificación). Este nombre <strong>debe ser único</strong> en la matriz de indicadores.</p><h5><a class="anchor" id="note--for-this-example,-the-calculation-below-is-done." href="#note--for-this-example,-the-calculation-below-is-done."><i class="fa fa-link"></i></a> Nota: Para este ejemplo, se realiza el cálculo a continuación.</h5><pre> <code>reference_tablename_indicator_id_1.reference_indicator_id_1 / reference_tablename_indicator_id_2.reference_indicator_id_2 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code></pre><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="indicator-result" href="#indicator-result"><i class="fa fa-link"></i></a> Resultado del indicador</h2><p> El resultado de los indicadores es el siguiente:</p><pre> <code>{ &quot;values&quot;: [ { &quot;unit&quot;: &quot;MWh&quot;, &quot;name&quot;: &quot;heat_tot_curr_density_tif_consumption&quot;, &quot;value&quot;: &quot;4112030.46&quot;
