<h1><a class="anchor" id="cm-heat-load-profiles" href="#cm-heat-load-profiles"><i class="fa fa-link"></i></a>Perfiles de carga térmica CM</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Tabla de contenido</h2><ul><li> <a href="#in-a-glance">De un vistazo</a></li><li> <a href="#introduction">Introducción</a></li><li> <a href="#inputs-and-outputs">Entradas y salidas</a><ul><li> <a href="#inputs-and-outputs_input-layers-and-parameters">Capas de entrada y parámetros</a></li><li> <a href="#inputs-and-outputs_output">Salida</a></li></ul></li><li> <a href="#method">Método</a><ul><li> <a href="#method_overview">Visión de conjunto</a></li><li> <a href="#method_details">Detalles</a></li><li> <a href="#method_implementation">Implementación</a></li></ul></li><li> <a href="#github-repository-of-this-calculation-module">Repositorio de GitHub de este módulo de cálculo</a></li><li> <a href="#sample-run">Ejecución de muestra</a></li><li> <a href="#how-to-cite">Cómo citar</a></li><li> <a href="#authors-and-reviewers">Autores y revisores</a></li><li> <a href="#license">Licencia</a></li><li> <a href="#acknowledgement">Reconocimiento</a></li></ul><h2><a class="anchor" id="in-a-glance" href="#in-a-glance"><i class="fa fa-link"></i></a> De un vistazo</h2><p> Este módulo genera perfiles de carga de demanda para calefacción de espacios y generación de agua caliente para una región seleccionada. Utiliza perfiles de carga predeterminados para el sector residencial y terciario en el nivel NUTS2 y lo escala a una demanda determinada en esos sectores y proporciona los perfiles de carga separados para calefacción de espacios y generación de agua caliente.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introducción</h2><p> Hotmaps proporciona perfiles de carga térmica genéricos y específicos del año en el nivel NUTS2. Es posible que el usuario desee adaptar los perfiles predeterminados en función de la información que tenga. Este módulo de cálculo da la posibilidad de escalar diferentes elementos constitutivos de los perfiles hacia arriba o hacia abajo y obtener un nuevo perfil.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Entradas y salidas</h2><h3><a class="anchor" id="input-layers-and-parameters" href="#input-layers-and-parameters"><i class="fa fa-link"></i></a> Capas de entrada y parámetros</h3><p> Se utilizan varias capas y perfiles del conjunto de datos predeterminado de Hotmaps:</p><ul><li> Sector residencial de densidad de calor</li><li> Sector no residencial de densidad de calor</li><li> Superficie bruta residencial</li><li> Superficie bruta no residencial</li><li> Base de datos industrial</li><li> Perfiles de carga para la industria</li><li> Perfiles de carga para calefacción residencial</li><li> Perfiles de carga para calefacción terciaria</li><li> Perfiles de carga para agua caliente sanitaria residencial</li><li> Perfiles de carga para agua caliente sanitaria terciaria</li></ul><h3><a class="anchor" id="user-inputs" href="#user-inputs"><i class="fa fa-link"></i></a> Entradas de usuario</h3><p> El usuario debe proporcionar los siguientes parámetros:</p><ul><li> <strong>Factor de calefacción residencial</strong> : la demanda de calefacción residencial se multiplica por este factor.</li><li> <strong>Factor de calefacción terciario</strong> : la demanda de calefacción terciaria se multiplica por este factor.</li><li> <strong>Factor de suministro de agua caliente residencial</strong> : la demanda de agua caliente residencial se multiplica por este factor.</li><li> <strong>Factor de suministro de agua caliente terciaria</strong> : La demanda de agua caliente terciaria se multiplica por este factor.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Salida</h3><h4><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Indicadores</h4><ul><li> <strong>Demanda de calor por industria</strong> : demanda de calor anual por industria en el área seleccionada.</li><li> <strong>Demanda de calor por calefacción residencial</strong> : Calor anual requerido para calefacción residencial en el área seleccionada.</li><li> <strong>Demanda de calor por suministro de agua caliente residencial</strong> : Calor anual requerido para el suministro de agua caliente residencial en el área seleccionada.</li><li> <strong>Demanda de calor por calefacción del sector terciario</strong> : Calor anual requerido para la calefacción del sector terciario en el área seleccionada.</li><li> <strong>Demanda de calor por suministro de agua caliente del sector terciario</strong> : Calor anual requerido para el suministro de agua caliente del sector terciario en el área seleccionada.</li><li> <strong>Demanda total de cabeza</strong> : calor total anual requerido en el área seleccionada.</li></ul><h4><a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Gráficos</h4><ul><li> <strong>Perfiles de demanda de calor</strong> : un gráfico que muestra los indicadores enumerados anteriormente durante el año.</li></ul><p> El eje x representa el tiempo y el eje y muestra la potencia en MW. Puede resultar útil ocultar ciertos sectores para ver mejor otros. Esto se puede hacer haciendo doble clic en la leyenda.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="method" href="#method"><i class="fa fa-link"></i></a> Método</h2><h3><a class="anchor" id="overview" href="#overview"><i class="fa fa-link"></i></a> Visión de conjunto</h3><p> Los perfiles de carga correspondientes se asignan a sitios industriales, edificios residenciales y terciarios en la selección del usuario.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="details" href="#details"><i class="fa fa-link"></i></a> Detalles</h3><h4><a class="anchor" id="residential-warm-water-supply" href="#residential-warm-water-supply"><i class="fa fa-link"></i></a> Suministro de agua caliente residencial</h4><p> Según la superficie bruta de suelo de los edificios residenciales y su ID NUTS 0, la energía utilizada para el suministro de agua caliente se calcula con la siguiente lista. Con el ID de NUTS 2 del área seleccionada, se asigna un perfil de carga. El suministro de agua caliente residencial se multiplica por el factor de suministro de agua caliente residencial en caso de que el usuario desee ajustar el valor.</p><p> <em>Energía específica por área para suministro de agua caliente en edificios residenciales en diferentes países</em></p><p> | Nuts0 ID | Suministro de agua caliente en kWh / m² / a | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21,38 | | CY | 8,80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14,35 | | FI | 10,15 | | FR | 9,66 | | DE | 8,27 | | EL | 12,51 | | HU | 13,66 | | IE | 15,91 | | IT | 14,01 | | LV | 15,71 | | LT | 13,36 | | LU | 8,29 | | MT | 10,99 | | NL | 8,91 | | PL | 10,00 | | PT | 9,48 | | RO | 11,48 | | SK | 21,51 | | SI | 21,74 | | ES | 23,34 | | SE | 13,54 | | Reino Unido | 49,03 |</p><h4><a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Calefacción residencial</h4><p> Se utiliza la densidad de demanda de calor para el sector residencial. Dado que este conjunto de datos ya contiene la energía utilizada para el agua caliente, se le resta la energía del agua caliente calculada anteriormente. Según el ID de NUTS 2, se asigna un perfil de calefacción residencial para cada mosaico de la densidad de calor en la selección del usuario. La calefacción residencial se multiplica por el factor de calefacción residencial en caso de que el usuario desee ajustar el valor.</p><h4><a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Suministro terciario de agua caliente</h4><p> De manera similar al suministro de agua caliente residencial, la energía necesaria se calcula con una tabla. Con el ID NUTS 2 de un área específica, se asigna un perfil de carga. El suministro de agua caliente terciaria se multiplica por el factor de suministro de agua caliente terciaria en caso de que el usuario desee ajustar el valor.</p><p> <em>Energía específica por área para suministro de agua caliente en edificios terciarios en diferentes países</em></p><p> | Nuts0 ID | Suministro de agua caliente en kWh / m² / a | | ------------- |: -------------: | | AT | 6,57 | | BE | 13,88 | | BG | 15,88 | | HR | 9,42 | | CY | 6,26 | | CZ | 9,18 | | DK | 8,03 | | EE | 14,13 | | FI | 10,52 | | FR | 9,57 | | DE | 3,05 | | EL | 6,99 | | HU | 9,51 | | IE | 10,87 | | IT | 5,62 | | LV | 7,16 | | LT | 10,46 | | LU | 7,20 | | MT | 10,45 | | NL | 6,89 | | PL | 9,55 | | PT | 21,47 | | RO | 13,85 | | SK | 8,49 | | SI | 27,73 | | ES | 12,44 | | SE | 19,62 | | Reino Unido | 13,45 |</p><h4><a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Calefacción terciaria</h4><p> La calefacción terciaria se calcula de forma similar a la calefacción residencial. El calentamiento terciario se multiplica por el factor de calentamiento terciario en caso de que el usuario quiera ajustar el valor.</p><h4><a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Demanda de calor industrial</h4><p> La demanda de calor por industria se busca en la base de datos industrial y, basándose en el NUTS 0 ID y el subsector, se asigna un perfil de carga.</p><h4><a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Suma</h4><p> Los cinco perfiles resultantes se agregan de forma absoluta para obtener el perfil final.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Implementación</h3><h4><a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Perfiles de carga</h4><p> Los perfiles de carga mencionados constan de 8760 puntos que representan la carga por cada hora de los 365 días. <strong><a href="https://gitlab.com/hotmaps/load_profile">Puede encontrar</a></strong> más información sobre los <strong><a href="https://gitlab.com/hotmaps/load_profile">perfiles de carga aquí.</a></strong> Todos los perfiles de carga están normalizados para que la integral sea igual a 1.</p><h4><a class="anchor" id="industrial-sites" href="#industrial-sites"><i class="fa fa-link"></i></a> Sitios industriales</h4><p> Las fuentes de calor se toman de la <strong><a href="https://gitlab.com/hotmaps/industrial_sites/industrial_sites_Industrial_Database">base de datos industrial.</a></strong> En base a su exceso de calor, NUTS 0 ID y sector industrial se crea un perfil de carga que cubre todas las horas del año para cada sitio.</p><h4><a class="anchor" id="heat-densities" href="#heat-densities"><i class="fa fa-link"></i></a> Densidades de calor</h4><p> La densidad de demanda de calor de los sectores residencial y no residencial son archivos ráster que se pueden encontrar <strong><a href="https://gitlab.com/hotmaps/heat">aquí.</a></strong></p><h4><a class="anchor" id="gross-floor-areas" href="#gross-floor-areas"><i class="fa fa-link"></i></a> Áreas de piso brutas</h4><p> El área bruta del piso residencial y el área bruta del piso no residencial son archivos ráster que se pueden encontrar <strong><a href="https://gitlab.com/hotmaps/gfa_res_curr_density">aquí</a></strong> y <strong><a
