<h1><a class="anchor" id="guidelines-for-writing-a-hotmaps-wiki-page" href="#guidelines-for-writing-a-hotmaps-wiki-page"><i class="fa fa-link"></i></a>Directrices para escribir una página Wiki de Hotmaps</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Tabla de contenido</h2><ul><li> <a href="#introduction">Introducción</a></li><li> <a href="#wiki-toolbar-and-sidebar">Barra de herramientas Wiki y barra lateral</a></li><li> <a href="#add-figures-and-images">Agregar figuras e imágenes</a></li><li> <a href="#classification-of-wiki-pages">Clasificación de páginas wiki</a></li><li> <a href="#default-structures-of-the-wiki-pages">Estructuras predeterminadas de las páginas wiki</a><ul><li> <a href="#default-structures-of-the-wiki-pages_hotmaps-data-set-repository-structure">Estructura del repositorio de conjuntos de datos de Hotmaps</a></li><li> <a href="#default-structures-of-the-wiki-pages_general-functionalities-of-the-toolbox">Funcionalidades generales de la caja de herramientas</a></li><li> <a href="#default-structures-of-the-wiki-pages_calculation-modules">Módulos de cálculo</a></li><li> <a href="#default-structures-of-the-wiki-pages_how-to-apply-the-hotmaps-toolbox">Cómo aplicar la caja de herramientas Hotmaps</a></li><li> <a href="#default-structures-of-the-wiki-pages_developers-area">Área de desarrolladores</a></li></ul></li><li> <a href="#authors-and-reviewers">Autores y revisores</a></li><li> <a href="#license">Licencia</a></li><li> <a href="#acknowledgement">Reconocimiento</a></li></ul><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introducción</h2><p> La plataforma wiki Hotmaps proporciona funcionalidades flexibles para crear y editar las páginas wiki. Para poder crear o modificar cualquier página wiki, debe iniciar sesión en la plataforma wiki. Esto solo es posible si tiene una cuenta de GitHub y se le asigna un permiso de proveedor de wiki. Para obtener un permiso, envíe un correo electrónico a &quot;support (at) hotmaps.eu&quot; incluyendo su ID de GitHub. Luego evaluaremos su solicitud y le responderemos en consecuencia. Para iniciar sesión, use el botón de inicio de sesión de la esquina superior derecha. Una vez que haya iniciado sesión, puede crear y modificar las páginas.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="wiki-toolbar-and-sidebar" href="#wiki-toolbar-and-sidebar"><i class="fa fa-link"></i></a> Barra de herramientas Wiki y barra lateral</h2><p> En la parte superior de cada página, verá la barra de herramientas de edición. La siguiente figura muestra esta barra de herramientas y explica brevemente la función de cada botón.</p><img alt="" src="/en/Guidelines-for-writing-a-Hotmaps-Wiki-page/toolbar_new.png"/><p> La barra lateral es visible en todas las páginas wiki. Tenga en cuenta que <strong>cualquier modificación en la barra lateral se aplica a todas las páginas wiki</strong> . En otras palabras, la barra lateral no es específica de la página que está editando. Para modificar la barra lateral, presione el botón &quot;editar&quot; en la barra de herramientas wiki. La modificación de la barra lateral se puede realizar a través de la sección en la parte inferior de la página de edición como se muestra en la siguiente figura.</p><img alt="" src="/en/Guidelines-for-writing-a-Hotmaps-Wiki-page/sidebar.png"/><p> Para ampliar la ventana de edición, arrástrela hacia la esquina inferior derecha como se muestra en la siguiente figura.</p><img alt="" src="/en/Guidelines-for-writing-a-Hotmaps-Wiki-page/enlarge_edit_window_new.png"/> [** `To Top` **] (# tabla de contenido)<h2><a class="anchor" id="add-figures-and-images" href="#add-figures-and-images"><i class="fa fa-link"></i></a> Agregar figuras e imágenes</h2><p> Para agregar una imagen a su página wiki, tiene dos opciones:</p><p> <strong>Método I:</strong></p><ul><li> Primero, cargue la imagen en su página wiki deseada usando el botón &quot;Cargar&quot; en la barra de herramientas.</li><li> Anote el enlace proporcionado para la imagen cargada.</li><li> Comience a editar esa página wiki y use el enlace a la imagen.</li></ul><p> <strong>Método II:</strong></p><ul><li> Empiece a editar la página wiki que desee.</li><li> Arrastre y suelte la imagen en el lugar del texto que le gustaría mostrar la figura.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="classification-of-wiki-pages" href="#classification-of-wiki-pages"><i class="fa fa-link"></i></a> Clasificación de páginas wiki</h2><p> Las páginas wiki de Hotmaps se clasifican en cinco clases diferentes:</p><ul><li> Estructura del repositorio de conjuntos de datos de Hotmaps</li><li> Funcionalidades generales de la caja de herramientas</li><li> Módulos de cálculo</li><li> Cómo aplicar la caja de herramientas Hotmaps</li><li> Área de desarrolladores</li></ul><p> Para cada clase de página wiki, se sugiere una estructura con varias secciones (vea la siguiente sección en esta página). Las secciones en <strong>NEGRITA</strong> deben ser parte de la página Wiki. El resto, por otro lado, debe agregarse si corresponde. Como página de muestra, eche un vistazo a <a href="/en/CM-District-heating-potential-areas-user-defined-thresholds">esta página wiki</a> .</p><p> Las páginas wiki pueden llegar a ser muy largas debido a las figuras incrustadas. Para ayudar a los lectores a navegar en una página fácil y fácilmente encontrar lo que están buscando, agregue una “ <strong>Tabla de contenido</strong> ” en la parte superior de cada página que esté escribiendo. Además, utilizar un “ <a href="#table-of-contents"><strong><code>To Top</code></strong></a> ” en la parte final de cada sección con un hipervínculo a la “Tabla de contenidos” en la parte superior de la página wiki.</p><p> En caso de que deba usar sus propias figuras, primero cree una nueva carpeta para su página wiki aquí y luego inserte todas las figuras en la carpeta. Evite también el uso de imágenes voluminosas ya que el repositorio tiene una capacidad limitada.</p><p> Una vez que haya terminado de escribir su página wiki, pida a sus colegas que revisen su página y le den sus comentarios. De esta forma, podemos minimizar los errores escritos y asegurarnos de que las páginas wiki estén escritas con claridad. Agregue el nombre del revisor en la sección &quot;Autores y revisores&quot;. Siga la misma estructura que en la página de muestra proporcionada.</p><p> <strong>NOTA</strong> : Las páginas Wiki generalmente están escritas en lenguaje Markdown (también son posibles algunos formatos adicionales). Si no está familiarizado con Markdown, lea <a href="https://guides.github.com/features/mastering-markdown/">aquí</a> el breve tutorial proporcionado por GitHub.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="default-structures-of-the-wiki-pages" href="#default-structures-of-the-wiki-pages"><i class="fa fa-link"></i></a> Estructuras predeterminadas de las páginas wiki</h2><h3><a class="anchor" id="hotmaps-data-set-repository-structure" href="#hotmaps-data-set-repository-structure"><i class="fa fa-link"></i></a> Estructura del repositorio de conjuntos de datos de Hotmaps</h3><p> Aquí, el punto más importante es realizar un seguimiento de todos los repositorios de Hotmaps en GitLab y mantener los enlaces actualizados.
