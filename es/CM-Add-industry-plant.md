<h1><a class="anchor" id="cm-add-industry-plant" href="#cm-add-industry-plant"><i class="fa fa-link"></i></a>CM Agregar planta industrial</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Tabla de contenido</h2><ul><li> <a href="#introduction">Introducción</a></li><li> <a href="#inputs-and-outputs">Entradas y salidas</a></li><li> <a href="#method">Método</a></li><li> <a href="#github-repository-of-this-calculation-module">Repositorio de GitHub de este módulo de cálculo</a></li><li> <a href="#references">Referencias</a></li><li> <a href="#how-to-cite">Cómo citar</a></li><li> <a href="#authors-and-reviewers">Autores y revisores</a></li><li> <a href="#license">Licencia</a></li><li> <a href="#acknowledgement">Reconocimiento</a></li></ul><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introducción</h2><p> Este módulo brinda la posibilidad de agregar sitios industriales adicionales con su demanda de calefacción y refrigeración y el exceso de potencial de calor a la caja de herramientas Hotmaps. Es posible agregar industrias adicionales intensivas en energía, así como industrias menos intensivas en energía. El usuario ingresa los datos necesarios en una herramienta de Excel independiente, que luego genera un archivo CSV para cargarlo en la caja de herramientas Hotmaps.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Entradas y salidas</h2><h3><a class="anchor" id="important-pre-settings-before-using-the-tool" href="#important-pre-settings-before-using-the-tool"><i class="fa fa-link"></i></a> Preconfiguraciones importantes antes de usar la herramienta</h3><p> 🔺<mark> <strong>Nota:</strong> Para la generación de archivos ( <em>.csv</em> ) en el formato correcto, es necesario que el delimitador predeterminado esté configurado como coma ( <strong>estándar inglés</strong> ). Esto <strong>no se puede configurar desde Excel</strong> y debe hacerse globalmente desde el Panel de control de Windows (como se describe <a href="https://www.itsupportguides.com/knowledge-base/office-2013/excel-20132016-how-to-change-csv-delimiter-character/">aquí</a> ). La siguiente figura muestra la configuración ejemplar para las PC con Windows alemanas.</mark></p><img alt="Fig. 6" src="https://github.com/HotMaps/hotmaps_wiki/raw/master/Images/cm_add_industry_plant/Delimiter2.PNG" title="Delimitador2"/><p> 🔺<mark> <strong>Nota:</strong> Para el uso adecuado de la herramienta basada en Excel, debe utilizar el <strong>punto</strong> como separador decimal (estándar en inglés, por ejemplo, 40,50). Esto debe configurarse desde Excel (como se describe <a href="https://support.microsoft.com/en-us/office/change-the-character-used-to-separate-thousands-or-decimals-c093b545-71cb-4903-b205-aebb9837bd1e">aquí</a> ). La siguiente figura muestra la configuración ejemplar para Excel alemán.</mark></p><img alt="Fig. 7" src="https://github.com/HotMaps/hotmaps_wiki/raw/master/Images/cm_add_industry_plant/Decimal.png" title="Decimal"/><h3><a class="anchor" id="data-input-in-excel-tool-by-user" href="#data-input-in-excel-tool-by-user"><i class="fa fa-link"></i></a> Entrada de datos en la herramienta Excel por usuario</h3><p> Descargue la herramienta Excel proporcionada desde <a href="https://github.com/HotMaps/add_industry_plant_cm/blob/master/HotMaps_CM_Add_industry_plant_V14.xlsm">AQUÍ</a> .</p><p> El enfoque sobre cómo utilizar la herramienta Excel se ilustra en la figura siguiente y se describe con más detalle en las siguientes secciones.</p><figure><img alt="" src="../images/cm_add_industry_plant/approach2.png"/><figcaption> <i><br/>Enfoque paso a paso de cómo utilizar la herramienta Excel.</i></figcaption></figure><h4><a class="anchor" id="1-add-general-information" href="#1-add-general-information"><i class="fa fa-link"></i></a> 1) Agregar información general</h4><p> Vaya a la hoja de pestañas:<figure><img alt="" src="../images/cm_add_industry_plant/General_information.PNG"/></figure></p><p> En el primer paso, ingrese toda la información general necesaria sobre los sitios para los que se debe calcular la demanda de calefacción y refrigeración y el potencial de exceso de calor. Es posible agregar hasta 10 sitios industriales.</p><figure><img alt="" src="../images/cm_add_industry_plant/General_information_Box.PNG"/><figcaption> <i><br/>Ejemplo de tabla de Excel para ingresar información general de empresas como base para el cálculo e importar a la caja de herramientas HotMaps.</i></figcaption></figure><h4><a class="anchor" id="2-choose-calculation-option" href="#2-choose-calculation-option"><i class="fa fa-link"></i></a> 2) Elija la opción de cálculo</h4><p> En el segundo paso, hay 3 opciones sobre cómo se pueden estimar la demanda de calefacción y refrigeración y el potencial de calor excesivo. Tenga en cuenta que es posible cambiar entre las tres opciones para las diferentes empresas; pero no dentro de la propia empresa.</p><p> Con respecto al exceso de temperatura de calor, hay que mencionar que el calor a baja temperatura (&lt;100 ° C) se puede ingresar en la herramienta Excel, pero aún no se evalúa en la caja de herramientas Hotmaps. Si se debe considerar el calor a baja temperatura, es necesaria la aplicación de una bomba de calor. Por lo tanto, el usuario puede incluir la demanda de electricidad de una bomba de calor en la demanda de energía final para la electricidad y elevar la temperatura del exceso de calor generado hasta el rango de 100-200 ° C.</p><h5><a class="anchor" id="option-1--manual-input" href="#option-1--manual-input"><i class="fa fa-link"></i></a> Opción 1: entrada manual</h5><p> Vaya a la hoja de pestañas:<figure><img alt="" src="../images/cm_add_industry_plant/Option1.PNG"/></figure></p><p> Elija esta opción, si los datos sobre la demanda de calefacción / refrigeración y el exceso de potencial de calor y distribución de temperatura están disponibles para la empresa seleccionada y se pueden completar manualmente.</p><h5><a class="anchor" id="option-2--plant-selection" href="#option-2--plant-selection"><i class="fa fa-link"></i></a> Opción 2: selección de plantas</h5><p> Vaya a la hoja de pestañas:<figure><img alt="" src="../images/cm_add_industry_plant/Option2.PNG"/></figure></p><p> Elija esta opción si no hay información disponible sobre la demanda de calefacción / refrigeración y el exceso de potencial de calor para la empresa seleccionada. Sobre la base de la base de datos específica de la planta, la demanda típica de calefacción / refrigeración y los potenciales de calor excesivos con distribución de temperatura se pueden seleccionar para una serie de plantas y productos / procesos. Como entrada necesaria para la conversión de los datos específicos del producto, se debe ingresar un valor como base de cálculo (por ejemplo, producción, área, etc.). Para obtener más información sobre el método de cálculo, consulte <a href="#method">Método</a> .</p><h5><a class="anchor" id="option-3--sector-selection" href="#option-3--sector-selection"><i class="fa fa-link"></i></a> Opción 3: Selección de sector</h5><p> Vaya a la hoja de pestañas:<figure><img alt="" src="../images/cm_add_industry_plant/Option3.PNG"/></figure></p><p> Elija esta opción, si su tipo de planta no está disponible en la opción 2. Sobre la base de los datos de calor específicos del sector, se calculan la demanda típica de calefacción / refrigeración y los potenciales de exceso de calor. Como insumo necesario, se debe ingresar la demanda de combustible para suministro de calor (en GWh / año). Para obtener más información sobre el método de cálculo, consulte <a href="#method">Método</a> .</p><h3><a class="anchor" id="data-upload-on-hotmaps-toolbox" href="#data-upload-on-hotmaps-toolbox"><i calefacción / refrigeración y el potencial de calor excesivo de una planta, se ofrecen al usuario dos métodos basados en indicadores para el cálculo de la demanda de calor y el potencial de calor excesivo.</p><p> Cabe mencionar que estos valores son solo valores indicativos aproximados para plantas típicas (opción 2) o a nivel de sector (opción 3) y no reemplazan un análisis y medición detallados de la demanda de calor y el exceso de calor de una planta.</p><h3><a class="anchor" id="method---option-2--plant-selection" href="#method---option-2--plant-selection"><i class="fa fa-link"></i></a> Método - Opción 2: Selección de plantas</h3><p> Los datos de calor específicos de la planta se obtienen de la base de datos de pronóstico. Para un gran número de diferentes tipos de plantas que consumen mucha energía y menos energía, la demanda de calefacción / refrigeración del proceso y el potencial de calor excesivo se derivan de la demanda energética final específica de combustibles y electricidad. Es importante tener en cuenta que, debido a la base de datos subyacente, solo el calor del proceso y el enfriamiento del proceso se cubren con este método; La calefacción de habitaciones y el agua caliente no están incluidos aquí. Dependiendo del tipo de planta, se proporcionan diferentes insumos como base para el cálculo (por ejemplo, volumen de producción en <em>toneladas</em> o área en <em>m <sup>2</sup></em> ).</p><p> Para el cálculo de la demanda de calefacción y refrigeración, es necesario asumir las eficiencias de conversión de energía final a calefacción y refrigeración. Dado que la mayoría de las aplicaciones de calor se basan en vapor, se supone una eficiencia del 90%. Para aplicaciones de refrigeración, se asume una relación de eficiencia energética (EER) ponderada a nivel de temperatura:</p><p> | Nivel de temperatura | EER asumido (según la base de datos de Forecast) | | --- | -------------: | | &lt;-30 ° C | 0,01 | | -30 ° C - 0 ° C | 1,8 | | 0 ° C - 15 ° C y&gt; 15 ° C | 3,5 |</p><p> Toda la base de datos utilizada para la opción 2 está disponible aquí: <a href="https://github.com/HotMaps/add_industry_plant_cm/blob/master/CM_add_industry_plant_databases.xlsx">Option2 Database</a></p><h3><a class="anchor" id="method---option-3--sector-selection" href="#method---option-3--sector-selection"><i class="fa fa-link"></i></a> Método - Opción 3: Selección de sector</h3><p> La opción 3 proporciona una evaluación de base amplia de la demanda de calor y el exceso de calor para los sectores de la industria manufacturera (según NACE Rev. 2 [ <a href="#references">1</a> ]).</p><h4><a class="anchor" id="calculation-of-excess-heat-potentials-on-a-sector-level" href="#calculation-of-excess-heat-potentials-on-a-sector-level"><i class="fa fa-link"></i></a> Cálculo de potenciales excesivos de calor a nivel de sector</h4><p> Para el cálculo del potencial de exceso de calor de los diferentes sectores se utilizan factores de exceso de calor según Brückner 2016 [ <a href="#references">2</a> ] (ver tabla siguiente). El factor de exceso de calor se define como el calor residual generado por consumo de combustible. En [ <a href="#references">2</a> ], los datos disponibles para determinar el exceso de potencial de calor en la industria manufacturera proceden de la encuesta de emisiones realizada cada cuatro años a nivel estatal en Alemania. De acuerdo con la Ordenanza de Declaración de Emisiones (1. BImSchG), todos los operadores de plantas, que están sujetas a aprobación, deben presentar una declaración de sus emisiones cada cuatro años. Para el año 2008 se evalúan los datos a nivel de empresa que consisten en los flujos de volumen de los gases de escape y su nivel de temperatura. Junto con la información disponible sobre el tipo y la cantidad de consumo de combustible de las plantas, el factor de exceso de calor de una planta se calcula como:</p><p> <em>Exceso de factor de calor = Exceso de calor / consumo de combustible</em></p><p> Por último, el factor de exceso de calor se calcula no solo a nivel de empresa sino también a nivel de sector. Para obtener información detallada, consulte [ <a href="#references">2</a> ].</p><p> Los factores de exceso de calor según [ <a href="#references">2</a> ] incluyen el exceso de calor procedente del calor del proceso, así como la generación de calor espacial y el agua caliente. Esto se debe al hecho de que solo se analiza el flujo volumétrico de los gases de escape que salen de la planta sin separar el consumo de combustible en calefacción, agua caliente y calor de proceso. No se incluye el exceso de calor generado por aplicaciones basadas en electricidad.</p><p> La distribución de la proporción de exceso de calor por rango de temperatura se basa en suposiciones propias.</p><p> | NACE Rev. 2 | Sector | exceso de factor de calor | | --- | ------------- | -----: | | 10 | Elaboración de productos alimenticios | 0,10 | | 11 | Elaboración de bebidas | 0,14 | | 12 | Elaboración de productos del tabaco | 0,12 | | 13 | Fabricación de textiles | 0,29 | | 14 | Confección de prendas de vestir | 0,06 | | 15 | Fabricación de cueros y productos afines | 0,20 | | 16 | Fabricación de madera y productos de madera y corcho, excepto muebles; fabricación de artículos de paja y materiales trenzables | 0,10 | | 17 | Fabricación de papel y productos de papel | 0,09 | | 18 | Impresión y reproducción de soportes grabados | 0,03 | | 20 | Fabricación de químicos y productos químicos | 0,09 | | 21 | Fabricación de productos farmacéuticos básicos y preparados farmacéuticos | 0,08 | | 22 | Fabricación de productos de caucho y plástico | 0,17 | | 23 | Fabricación de otros productos minerales no metálicos | 0,15 | | 24 | Fabricación de metales básicos | 0,19 | | 25 | Fabricación de productos elaborados de metal, excepto maquinaria y equipo | 0,19 | | 26 | Fabricación de productos informáticos, electrónicos y ópticos | 0,18 | | 27 | Fabricación de material eléctrico | 0,31 | | 28 | Fabricación de maquinaria y equipo ncop | 0,16 | | 29 | Fabricación de vehículos de motor, remolques y semirremolques | 0,12 | | 30 | Fabricación de otros equipos de transporte | 0,38 | | 31 | Fabricación de muebles | 0,12 | | 32 | Otras manufacturas | 0,08 | | 33 | Reparación e instalación de maquinaria y equipo | 0,05 |</p><p> <em>Exceso de factores de calor para los sectores de la industria manufacturera (según [ <a href="#references">1</a> ])</em></p><h4><a class="anchor" id="calculation-of-heat-demand-on-a-sector-level" href="#calculation-of-heat-demand-on-a-sector-level"><i class="fa fa-link"></i></a> Cálculo de la demanda de calor a nivel sectorial</h4><p> El desglose de la demanda energética final de combustibles en rangos de temperatura a nivel sectorial se basa en un estudio alemán (base de datos para la evaluación de medidas de eficiencia energética [ <a href="#references">3</a> ]). Para el cálculo de la demanda de calor se supone que casi todas las compras de combustible se utilizan para calentar. De acuerdo con la opción 2, se supone una eficiencia de conversión de energía final a calor del 90% para la generación de calor a base de vapor.</p><p> Toda la base de datos utilizada para la opción 3 está disponible aquí: <a href="https://github.com/HotMaps/add_industry_plant_cm/blob/master/CM_add_industry_plant_databases.xlsx">Option3 Database</a></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="github-repository-of-this-calculation-module" href="#github-repository-of-this-calculation-module"><i class="fa fa-link"></i></a> Repositorio de GitHub de este módulo de cálculo</h2><p> <a href="https://github.com/HotMaps/add_industry_plant_cm">Aquí</a> obtiene el desarrollo de vanguardia para este módulo de cálculo.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="references" href="#references"><i class="fa fa-link"></i></a> Referencias</h2><p> [1].
