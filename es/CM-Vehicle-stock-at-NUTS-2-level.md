Los puntos de datos históricos se extrapolaron en función de las tendencias futuras para indicadores seleccionados derivados del escenario de referencia PRIMES – TREMOVE EU 2016.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Introducción</h2><p> Este módulo de cálculo demuestra el perfil de existencias de vehículos en el nivel NUTS 2 hasta el año 2030 según un escenario normal. El conjunto de datos también se puede encontrar en el repositorio de Hotmaps. Los números ilustrados en el nivel NUTS 2 se extrapolaron en función de los valores de los datos NUTS 0. Los puntos de datos históricos se extrapolaron en función de las tendencias futuras de indicadores seleccionados derivados del escenario de referencia PRIMES – TREMOVE UE 2016 <a href="#references">[1]</a> .</p><p> El módulo de cálculo solo se puede ejecutar en el nivel NUTS 2.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Entradas y salidas</h2><p> <strong>Entradas</strong> : Este CM no requiere ningún parámetro de entrada del usuario. El usuario solo debe seleccionar una o varias regiones NUTS 2.</p><p> <strong>Salidas</strong> :</p><ul><li> Indicadores:<ul><li> Stock de vehículos en el primer año del período</li><li> Stock de vehículos en el último año del período</li></ul></li><li> Gráficos:<ul><li> Se representa un gráfico de barras para mostrar las estatuas de las existencias de vehículos en diferentes años.</li></ul></li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="methodology" href="#methodology"><i class="fa fa-link"></i></a> Metodología</h2><p> Los datos NUTS2 (2006) son el resultado de una interpolación espacial de los datos de existencias de vehículos a nivel NUT0, utilizando los datos NUTS2 del número de existencias de vehículos y el PIB a nivel NUTS2 como proxy. Además, la resolución original de 5 años de los <a href="https://gitlab.com/hotmaps/transport/nuts0">datos a nivel NUTS0</a> se ha interpolado con una resolución anual. Los resultados se muestran en forma de indicadores y un gráfico.</p><p> <strong>Cabe señalar que en caso de selección de dos o más regiones, se presenta un valor agregado. Además, cabe señalar que los datos están disponibles para diferentes períodos dependiendo de la región NUTS 2 (por ejemplo, una región de 1999 a 2030 y la otra de 1995 a 2030). Por lo tanto, en caso de selección de dos o más regiones, el usuario también debe considerar los valores de las regiones constituyentes.</strong></p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="github-repository-of-this-calculation-module" href="#github-repository-of-this-calculation-module"><i class="fa fa-link"></i></a> Repositorio de GitHub de este módulo de cálculo</h2><p> <a href="https://github.com/HotMaps/vehicle_stock/tree/develop">Aquí</a> obtiene el desarrollo de vanguardia para este módulo de cálculo.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="sample-run" href="#sample-run"><i class="fa fa-link"></i></a> Ejecución de muestra</h2><img src="/en/CM-Vehicle-stock-at-NUTS-2-level/1.png"/><img src="/en/CM-Vehicle-stock-at-NUTS-2-level/2.png"/><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="references" href="#references"><i class="fa fa-link"></i></a> Referencias</h2><p> [1] Comisión Europea, &quot;Escenario de referencia de la UE 2016&quot;.
