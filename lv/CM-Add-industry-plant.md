pieprasījums un pārpalikuma siltuma potenciāls nav zināms, siltuma pieprasījuma un siltuma pārpalikuma aprēķināšanai lietotājam tiek piedāvātas divas metodes.</p><p> Jāpiemin, ka šīs vērtības ir tikai orientējošas aptuvenas vērtības tipiskām iekārtām (2. variants) vai nozares līmenī (3. variants) un neaizstāj detalizētu siltuma pieprasījuma un iekārtas pārpalikuma analīzi un mērījumus.</p><h3><a class="anchor" id="method---option-2--plant-selection" href="#method---option-2--plant-selection"><i class="fa fa-link"></i></a> Metode - 2. variants: Augu atlase</h3><p> Iekārtas siltuma dati tiek ņemti no Forecast datu bāzes. Lielam skaitam dažādu energoietilpīgu un mazāk energoietilpīgu iekārtu veidu apkures / dzesēšanas pieprasījums un siltuma pārpalikums tiek iegūts no degvielas un elektroenerģijas īpašā galīgā enerģijas pieprasījuma. Ir svarīgi atzīmēt, ka pamatā esošās datu bāzes dēļ šī metode attiecas tikai uz procesa siltumu un procesa dzesēšanu; telpu apsilde un karstā ūdens šeit nav iekļauta. Atkarībā no iekārtas veida aprēķinam tiek izmantotas dažādas izejvielas (piemēram, ražošanas apjoms <em>tonnās</em> vai platība <em>m <sup>2</sup></em> ).</p><p> Lai aprēķinātu apkures un dzesēšanas pieprasījumu, ir jāpieņem konversijas efektivitāte no galīgās enerģijas uz siltumu un dzesēšanu. Tā kā lielāko daļu siltuma lietojumu veic uz tvaika bāzes, tiek pieņemta 90% efektivitāte. Dzesēšanas vajadzībām tiek pieņemts temperatūras svērtais energoefektivitātes koeficients (EER):</p><p> | Temperatūras līmenis pieņemtais EER (saskaņā ar Forecast datubāzi) | --- | -------------: | | &lt;-30 ° C | 0,01 | | -30 ° C - 0 ° C | 1,8 | | 0 ° C - 15 ° C un&gt; 15 ° C | 3.5</p><p> Visa 2. variantam izmantotā datu bāze ir pieejama šeit: 2. <a href="https://github.com/HotMaps/add_industry_plant_cm/blob/master/CM_add_industry_plant_databases.xlsx">opcijas datu bāze</a></p><h3><a class="anchor" id="method---option-3--sector-selection" href="#method---option-3--sector-selection"><i class="fa fa-link"></i></a> Metode - 3. iespēja: nozares izvēle</h3><p> 3. risinājums nodrošina plašu siltuma pieprasījuma un siltuma pārpalikuma novērtējumu apstrādes rūpniecības nozarēs (saskaņā ar NACE 2. red. [ <a href="#references">1</a> ]).</p><h4><a class="anchor" id="calculation-of-excess-heat-potentials-on-a-sector-level" href="#calculation-of-excess-heat-potentials-on-a-sector-level"><i class="fa fa-link"></i></a> Pārmērīga siltuma potenciāla aprēķināšana nozares līmenī</h4><p> Dažādu nozaru siltuma pārpalikuma aprēķināšanai tiek izmantoti pārpalikuma siltuma koeficienti saskaņā ar Brückner 2016 [ <a href="#references">2</a> ] (sk. Tabulu zemāk). Pārpalikuma siltuma koeficientu nosaka kā izlietoto siltumu, kas rodas uz vienu degvielas patēriņu. [ <a href="#references">2</a> ] pieejamos datus siltuma pārpalikuma noteikšanai apstrādes rūpniecībā iegūst no emisiju apsekojuma, ko reizi četros gados veic valsts līmenī Vācijā. Saskaņā ar Emisijas deklarācijas rīkojumu (1. BImSchG) visiem iekārtu operatoriem, uz kuriem attiecas apstiprinājums, ik pēc četriem gadiem jāiesniedz deklarācija par emisijām. Par 2008. gadu tiek vērtēti dati par uzņēmuma līmeni, kas sastāv no izplūdes gāzu tilpuma plūsmām un to temperatūras līmeņa. Kopā ar pieejamo informāciju par iekārtu degvielas patēriņa veidu un daudzumu iekārtas pārpalikuma siltuma koeficientu aprēķina šādi:</p><p> <em>Pārmērīgs siltuma koeficients = Pārmērīgs siltuma / degvielas patēriņš</em></p><p> Visbeidzot, siltuma pārpalikuma koeficientu aprēķina ne tikai uzņēmuma, bet arī nozares līmenī. Sīkāku informāciju skatiet [ <a href="#references">2</a> ].</p><p> Pārpalikuma siltuma faktori saskaņā ar [ <a href="#references">2</a> ] ietver siltuma pārpalikumu, kas rodas no procesa siltuma, kā arī no telpas siltuma un karstā ūdens. Tas ir saistīts ar faktu, ka tiek analizēta tikai izplūdes gāzu plūsma, kas iziet no rūpnīcas, nedalot degvielas patēriņu telpu apsildē, karstā ūdens un procesa siltumā. Nav iekļauts siltuma pārpalikums, ko rada elektrības lietojumi.</p><p> Pārpalikuma siltuma daļas sadalījums pa temperatūras diapazonu ir balstīts uz pašu pieņēmumiem.</p><p> | NACE 2. red Nozare | siltuma pārpalikuma koeficients | | --- | ------------- | -----: | | 10 | Pārtikas produktu ražošana 0,10 | | 11 | Dzērienu ražošana 0,14 | | 12 | Tabakas izstrādājumu ražošana 0,12 | | 13 | Tekstilizstrādājumu ražošana 0,29 | | 14 | Apģērbu ražošana 0,06 | | 15 | Ādas un līdzīgu izstrādājumu ražošana 0,20 | | 16 | Koka un koka un korķa izstrādājumu ražošana, izņemot mēbeles; salmu un pīto izstrādājumu ražošana | 0,10 | | 17 | Papīra un papīra izstrādājumu ražošana 0,09 | | 18 | Iespiešana un ierakstu reproducēšana 0,03 | | 20 | Ķimikāliju un ķīmisko produktu ražošana 0,09 | | 21 | Farmaceitisko pamatproduktu un farmaceitisko preparātu ražošana 0,08 | | 22 | Gumijas un plastmasas izstrādājumu ražošana 0,17 | | 23 | Citu nemetālisko minerālu izstrādājumu ražošana 0,15 | | 24 | Metālu ražošana 0,19 | | 25 | Gatavo metālizstrādājumu ražošana, izņemot mašīnas un iekārtas 0,19 | | 26 | Datoru, elektronisko un optisko izstrādājumu ražošana 0,18 | | 27 | Elektrisko iekārtu ražošana 0,31 | | 28 | Citur neklasificētu mašīnu un iekārtu ražošana 0,16 | | 29 | Automobiļu, piekabju un puspiekabju ražošana 0,12 | | 30 | Citu transporta līdzekļu ražošana 0,38 | | 31 | Mēbeļu ražošana 0,12 | | 32 | Pārējā ražošana 0,08 | | 33 | Mašīnu un iekārtu remonts un uzstādīšana 0,05 |</p><p> <em>Liekie siltuma koeficienti apstrādes rūpniecības nozarēs (saskaņā ar [ <a href="#references">1</a> ])</em></p><h4><a class="anchor" id="calculation-of-heat-demand-on-a-sector-level" href="#calculation-of-heat-demand-on-a-sector-level"><i class="fa fa-link"></i></a> Siltuma pieprasījuma aprēķins nozares līmenī</h4><p> Degvielas galīgā enerģijas pieprasījuma sadalījums pa temperatūras diapazoniem nozares līmenī balstās uz Vācijas pētījumu (datu bāze energoefektivitātes pasākumu novērtēšanai [ <a href="#references">3</a> ]).
