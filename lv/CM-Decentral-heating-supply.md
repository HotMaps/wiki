<h1><a class="anchor" id="cm-decentral-heating-supply" href="#cm-decentral-heating-supply"><i class="fa fa-link"></i></a>CM Decentral apkures padeve</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Satura rādītājs</h2><ul><li> <a href="#in-a-glance">Vienā acu uzmetienā</a></li><li> <a href="#introduction">Ievads</a></li><li> <a href="#inputs-and-outputs">Ieejas un izejas</a><ul><li> <a href="#inputs-and-outputs_inputs">Ieejas</a><ul><li> <a href="#inputs-and-outputs_inputs_default-input-data">Noklusētie ievades dati</a></li><li> <a href="#inputs-and-outputs_inputs_manual-input-data">Manuāli ievades dati</a></li></ul></li><li> <a href="#inputs-and-outputs_outputs">Rezultāti</a><ul><li> <a href="#inputs-and-outputs_outputs_indicators">Rādītāji</a></li><li> <a href="#inputs-and-outputs_outputs_graphs">Grafiki</a></li></ul></li></ul></li><li> <a href="#methodology">Metodoloģija</a></li><li> <a href="#github-repository-of-this-calculation-module">Šī aprēķina moduļa GitHub krātuve</a></li><li> <a href="#sample-run">Izlases skrējiens</a><ul><li> <a href="#sample-run_test-run">Pārbaudes skrējiens</a></li></ul></li><li> <a href="#how-to-cite">Kā citēt</a></li><li> <a href="#authors-and-reviewers">Autori un recenzenti</a></li><li> <a href="#license">Licence</a></li><li> <a href="#acknowledgement">Pateicība</a></li></ul><h2><a class="anchor" id="in-a-glance" href="#in-a-glance"><i class="fa fa-link"></i></a> Vienā acu uzmetienā</h2><p> Šis modulis aprēķina siltumapgādes izmaksas ēkās, izmantojot decentralizētas siltumapgādes tehnoloģijas. Ievads modulī ir ieguldījumu izmaksas, O&amp;M izmaksas, enerģijas cenas, siltuma pieprasījuma stundas slodzes profils, kā arī nolietojuma laiks un procentu likme. Turklāt ir nepieciešams ēku sastāvs un ēku renovācijas stāvokļi analizētajā apgabalā. Rezultāti ir dažādu decentralizētas siltumapgādes tehnoloģiju siltumapgādes izmaksas noteiktām ēkām un cenu pieņēmumi, kā arī decentralizētās piegādes atsauces izmaksas analizētajai teritorijai un saistītās CO2 emisijas.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Ievads</h2><p> Šis aprēķina modulis aprēķina izlīdzinātās siltuma izmaksas (LCOH) dažādām decentralizētām siltumapgādes tehnoloģijām siltuma piegādei lietotāja noteiktai ēkai.</p><img alt="" src="https://raw.githubusercontent.com/HotMaps/hotmaps_wiki/master/Images/cm_decentral_heating/new/1.png"/><p> Piemēram, tehnoloģijas ir šādas:</p><ul><li> Siltumsūknis gaiss-gaiss, gaiss-ūdens un sālījums-ūdens,</li><li> Elektriskais sildītājs,</li><li> Bio-eļļas katls,</li><li> Eļļas katls,</li><li> Automātiskā un manuālā biomasa,</li><li> Malkas plīts,</li><li> Dabasgāze,</li><li> Saules siltuma</li></ul><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code></ins></p><h2><a class="anchor" id="inputs-and-outputs-" href="#inputs-and-outputs-"><i class="fa fa-link"></i></a> Ieejas un izejas:</h2><h3><a class="anchor" id="inputs" href="#inputs"><i class="fa fa-link"></i></a> Ieejas</h3><h4><a class="anchor" id="default-input-data" href="#default-input-data"><i class="fa fa-link"></i></a> Noklusētie ievades dati</h4><p> Daļa ievades datu nāk no Hotmaps noklusējuma datu kopām. Tie ietver:</p><ul><li> NUTS rastrs</li><li> Ēku fonds (NUTS0 līmenis): lietderīgās enerģijas pieprasījums pēc ēkas klases un būvniecības perioda</li><li> Slodzes profili (NUTS2 līmenis)</li><li> Finanšu dati - apkures sistēmas (NUTS0 līmenis)</li><li> Finanšu dati - enerģijas nesēju cenas (NUTS0 līmenis)</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h4><a class="anchor" id="manual-input-data" href="#manual-input-data"><i class="fa fa-link"></i></a> Manuāli ievades dati</h4><p> Pēc apgabala izvēles manuāli jāievada šādi ievades dati:</p><ul><li> Ēkas vecums,</li><li> Procentu likme,</li><li> Bruto grīdas platība,</li><li> Ēkas tips,</li><li> Gads (finanšu datiem),</li><li> Ietaupījumi telpu apsildē,</li><li> Emisijas koeficienti uz vienu enerģijas nesēju.</li></ul><p> <strong>SVARĪGA PIEZĪME</strong> Hotmaps biomasas emisijas netiek uzskatītas par nullejām, jo Hotmaps ir rīks, kura mērķis ir atbalstīt notiekošo enerģijas pāreju. Uzskatot biomasu par oglekļa neitrālu enerģijas avotu, tas nozīmē, ka biomasas sadedzināšana samazina pašreizējās kopējās emisijas, salīdzinot ar fosilo degvielu, kas nav taisnība. Oglekļa dioksīda uzkrāšanās kokos notiek gadu desmitiem ilgi, savukārt tā izdalīšanās atmosfērā notiek vienlaikus. Tāpēc var secināt, ka biomasas neto oglekļa bilance ilgtermiņā ir nulle, taču īstermiņā tā nav, un klimata pārmaiņu mazināšanas pasākumiem īsā laikā jāsniedz rezultāti oglekļa emisiju samazināšanā.</p><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code></ins></p><h3><a class="anchor" id="outputs-" href="#outputs-"><i class="fa fa-link"></i></a> Rezultāti:</h3><h4><a class="anchor" id="indicators-" href="#indicators-"><i class="fa fa-link"></i></a> Rādītāji:</h4><p> Rādītājus veido ievades parametru un aprēķināto vērtību kopsavilkums:</p><ul><li> Atbilstošais NUTS kods: izvēlētā reģiona NUTS3 kods (ja izmanto hektāru vai LAU līmeni),</li><li> iegūti ietaupījumi telpu apsildē,</li><li> ievadīta bruto platība,</li><li> ievadītais gads,</li><li> ievadītā procentu likme,</li><li> ievadīts ēkas vecums,</li><li> ievadītais ēkas tips,</li><li> ēkas lietderīgās enerģijas gada pieprasījums,</li><li> Siltuma slodze,</li><li> Sektors,</li><li> Katram finanšu datiem izmantots ēkas tips.</li></ul><p><ins> <code><strong><a href="#table-of-contents">To Top</a></strong></code></ins></p><h4><a class="anchor"
