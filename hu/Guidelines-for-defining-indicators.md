<h1><a class="anchor" id="guidelines-for-defining-indicators" href="#guidelines-for-defining-indicators"><i class="fa fa-link"></i></a>Iránymutatások a mutatók meghatározásához</h1><h2><a class="anchor" id="table-of-contents" href="#table-of-contents"><i class="fa fa-link"></i></a> Tartalomjegyzék</h2><ul><li> <a href="#indicators">Mutatók</a><ul><li> <a href="#indicators_simple-indicators">Egyszerű mutatók</a></li><li> <a href="#indicators_cross-indicators">Keresztmutatók</a></li></ul></li><li> <a href="#indicator-result">Mutató eredmény</a></li><li> <a href="#how-to-cite">Hogyan lehet idézni</a></li><li> <a href="#authors-and-reviewers">Szerzők és lektorok</a></li><li> <a href="#license">Engedély</a></li><li> <a href="#acknowledgement">Elismerés</a></li></ul><h2><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Mutatók</h2><p> Ez az oldal elmagyarázza a mutatók kezelését a Hotmaps eszköztáron. A mutató egy olyan érték, amely a Hotmaps eszköztáron jelenik meg az ügyfél oldalon. A mutató egy réteg része. A mutató egy adott térbeli régió értéke, egységgel. Az indikátor egy rétegszótár része. Egy réteg objektum tartalmaz egy &quot;indikátorok&quot; nevű tömböt. Ez a táblázat felsorolja az összes mutatót, amelyet a fóliához látni szeretne.</p><pre> <code>layers = { &#39;heat_tot_curr_density_tif&#39;:{ &#39;tablename&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;from_indicator_name&#39;:&#39;stat_heat_tot_curr_density_tif&#39;, &#39;schema_scalelvl&#39;: &#39;stat&#39;, &#39;schema_hectare&#39;: &#39;geo&#39;, &#39;crs&#39;: &#39;3035&#39;, &#39;geo_column&#39;: &#39;geometry&#39;, &#39;table_type&#39;:&#39;raster&#39;, &#39;data_lvl&#39;:[&#39;NUTS 0&#39;,&#39;NUTS 1&#39;,&#39;NUTS 2&#39;,&#39;NUTS 3&#39;,&#39;LAU 2&#39;,&#39;Hectare&#39;], &#39;data_aggregated&#39;:True, &#39;scalelvl_column&#39;:&#39;&#39;, &#39;indicators&#39;:[ {&#39;table_column&#39;: &#39;sum&#39;, &#39;unit&#39;: &#39;MWh&#39;,&#39;indicator_id&#39;:&#39;consumption&#39;}, {&#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;,&#39;indicator_id&#39;:&#39;count_cell&#39;}, { &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;: &#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;: &#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;: &#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;: &#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; } ] } }</code></pre><ul><li> „tablename”</li></ul><p> A DB tábla neve. (Példa: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;from_indicator_name&#39;</li></ul><p> Alnevek a mutatók kiválasztásához. <strong>Egyedinek kell lennie.</strong> (Példa: &#39;stat_heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;data_aggregated&#39;</li></ul><p> Az adatok összesítve vannak-e vagy sem (Értékek: Igaz vagy Hamis)</p><ul><li> &#39;scalelvl_column&#39;</li></ul><p> Méretszint oszlop neve, ha eltér az alapértelmezettől (példa: &#39;kód&#39;)</p><ul><li> &#39;data_lvl&#39;</li></ul><p> Az adatbázisban elérhető adatokhoz rendelkezésre álló szintek</p><ul><li> &#39;schema_scalelvl&#39;</li></ul><p> Táblázat séma helye a diószintnél. (Példa: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> &#39;schema_hectare&#39;</li></ul><p> Táblázat séma helye a hektárszinthez. (Példa: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;)</p><ul><li> „crs”</li></ul><p> A geometria vetülete (példa: &#39;3035&#39;, &#39;4326&#39;, &#39;4258&#39;)</p><ul><li> &#39;geo_column&#39;</li></ul><p> Az adatbázis geometriai oszlopának neve (Példa: &#39;geom&#39;, &#39;geometry&#39;)</p><ul><li> &#39;table_type&#39;</li></ul><p> A réteg típusa az adatbázisban (Értékek: &#39;vektor&#39; vagy &#39;raszter&#39;).</p><p> <em><strong>Fontos:</strong></em> Ha raszterről van szó, a rendelkezésre álló oszlopok <strong>száma: összeg, átlag, stddev, min és max</strong></p><ul><li> „Mutatók”</li></ul><p> Kétféle indikátor létezik (egyszerű mutatók és keresztmutatók).</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="simple-indicators" href="#simple-indicators"><i class="fa fa-link"></i></a> Egyszerű mutatók</h3><p> Az egyszerű mutató egy 3 paraméterű objektum.</p><pre> <code>{ &#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;, &#39;indicator_id&#39;:&#39;count_cell&#39; }</code></pre><ul><li> &#39;table_column&#39;</li></ul><p> Ez az a táblázat oszlop, amely a táblázatban van kiválasztva. (Példa: &#39;count&#39;)</p><img alt="asztaloszlop kiválasztása" src="https://github.com/HotMaps/Hotmaps-toolbox-service/raw/master/api/assets/table_image.png"/><ul><li> &#39;Mértékegység&#39;</li></ul><p> Ez a mutató egysége. (Példa: &#39;cellák&#39;, &#39;MWh&#39;)</p><ul><li> &#39;indikátor_id&#39;</li></ul><p> Ez a mutató indikátorazonosítója (Mint egy azonosító). Ennek a névnek <strong>egyedinek kell lennie</strong> az indikátorok tömbjében.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="cross-indicators" href="#cross-indicators"><i class="fa fa-link"></i></a> Keresztmutatók</h3><p> A keresztmutató egy 7 paraméterű objektum. Ezt a nevet <strong>meg kell határozni</strong> az indikátor tömbben. Ez az 1. számú érték.</p><ul><li> &#39;referencia_tábla neve_indicator_id_1&#39;</li></ul><p> A réteg táblanév hivatkozása, amely az 1. értékre hivatkozik. (Példa: &#39;heat_tot_curr_density_tif&#39;)</p><ul><li> &#39;operátor&#39;</li></ul><p> A 2 értékre alkalmazandó számítási szabály (értékek: &#39;/&#39; vagy &#39;*&#39; vagy &#39;+&#39; vagy &#39;-&#39;)</p><ul><li> &#39;referencia_indicator_id_1&#39;</li></ul><p> Megfelel egy egyszerű mutató azonosítójának. Ezt a nevet <strong>meg kell határozni</strong> az indikátor tömbben. Ez a 2. számú érték.</p><ul><li> &#39;referencia_tábla neve_indicator_id_2&#39;</li></ul><p> A réteg táblanév hivatkozása, amely a 2. értékre hivatkozik. (Példa: &#39;pop_tot_curr_density_tif&#39;)</p><ul><li> &#39;Mértékegység&#39;</li></ul><p> Ez a mutató egysége. (Példa: &#39;cellák&#39;, &#39;MWh&#39;)</p><ul><li> &#39;név&#39;</li></ul><p> Ez a mutató neve (például egy azonosító). Ennek a névnek <strong>egyedinek kell lennie</strong> az indikátorok tömbjében.</p><h5><a class="anchor" id="note--for-this-example,-the-calculation-below-is-done." href="#note--for-this-example,-the-calculation-below-is-done."><i class="fa fa-link"></i></a> Megjegyzés: Ennél a példánál az alábbi számítás megtörtént.</h5><pre> <code>reference_tablename_indicator_id_1.reference_indicator_id_1 / reference_tablename_indicator_id_2.reference_indicator_id_2 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code></pre><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="indicator-result" href="#indicator-result"><i class="fa fa-link"></i></a> Mutató eredmény</h2><p> A mutatók eredménye a következő:</p><pre> <code>{ &quot;values&quot;: [ { &quot;unit&quot;: &quot;MWh&quot;, &quot;name&quot;: &quot;heat_tot_curr_density_tif_consumption&quot;, &quot;value&quot;: &quot;4112030.46&quot;
