Egy adott terület Nuts2 ID-jével egy terhelési profil lesz hozzárendelve. A lakossági melegvíz-ellátást meg kell szorozni a lakossági melegvíz-ellátási tényezővel, ha a felhasználó beállítani akarja az értéket. </p><p> <em>Sajátos energia területre eső melegvíz-ellátáshoz lakóépületekben különböző országokban</em> </p><p> | Nuts0 ID | Meleg vízellátás kWh / m² / évben | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21,38 | | CY | 8.80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14,35 | | FI | 10.15 | | FR | 9,66 | | DE | 8,27 | | EL | 12.51 | | HU | 13,66 | | IE | 15,91 | | IT | 14.01 | | LV | 15,71 | | LT | 13,36 | | LU | 8,29 | | MT | 10.99 | | NL | 8.91 | | PL | 10.00 | | PT | 9.48 | | RO | 11.48 | | SK | 21,51 | | SI | 21,74 | | ES | 23,34 | | SE | 13,54 | | Egyesült Királyság | 49,03 | </p><h4> <a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Lakossági fűtés </h4><p> A lakossági szektor hősűrűségét használják. Mivel ez az adatkészlet már tartalmazza a meleg vízhez felhasznált energiát, az előző számított melegvíz energiát kivonják belőle. A Nuts2 ID alapján lakossági fűtési profilt rendelnek hozzá a felhasználó által választott hősűrűség minden egyes csempéjéhez. A lakásfűtés megsokszorozódik a lakásfűtés tényezőjével, ha a felhasználó beállítani akarja az értéket. </p><h4> <a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Harmadik szintű melegvízellátás </h4><p> A lakossági meleg vízellátáshoz hasonlóan a szükséges energiát egy táblázattal kell kiszámítani. Egy adott terület Nuts2 ID-jével egy terhelési profil lesz hozzárendelve. A harmadlagos melegvízellátást meg kell szorozni a harmadlagos melegvízellátási tényezővel, ha a felhasználó beállítani akarja az értéket. </p><p> <em>Sajátos energia területre meleg vízellátáshoz a különféle országok tercier épületeiben</em> </p><p> | Nuts0 ID | Meleg vízellátás kWh / m² / évben | | ------------- |: -------------: | | AT | 6.57 | | BE | 13,88 | | BG | 15,88 | | HR | 9.42 | | CY | 6,26 | | CZ | 9,18 | | DK | 8.03 | | EE | 14.13 | | FI | 10.52 | | FR | 9.57 | | DE | 3.05 | | EL | 6,99 | | HU | 9.51 | | IE | 10,87 | | IT | 5.62 | | LV | 7.16 | | LT | 10.46 | | LU | 7,20 | | MT | 10.45 | | NL | 6,89 | | PL | 9.55 | | PT | 21,47 | | RO | 13,85 | | SK | 8,49 | | SI | 27,73 | | ES | 12.44 | | SE | 19,62 | | Egyesült Királyság | 13.45 | </p><h4> <a class="anchor" id="tertiary-heating" href="#tertiary-heating"><i class="fa fa-link"></i></a> Harmadik fűtés </h4><p> A harmadlagos fűtést a lakásfűtéshez hasonló módon számítják ki. A harmadlagos fűtést meg kell szorozni a harmadlagos fűtési tényezővel, ha a felhasználó beállítani akarja az értéket. </p><h4> <a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Ipari hőigény </h4><p> Az iparonkénti hőigényt felkutatják az ipari adatbázisból, és a Nuts0 ID és az alszektor alapján egy terhelési profilt rendelnek. </p><h4> <a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Összegzés </h4><p> Az öt kapott profilt abszolút módon összeadjuk. </p><h3> <a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Végrehajtás </h3><h4> <a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Profilok betöltése </h4><p> Az említett terhelési profilok 8760 pontból állnak, amelyek a 365 nap minden órájára vonatkoznak.
