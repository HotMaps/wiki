Alapértelmezett terhelési profilokat használ a lakossági és a tercier szektor számára NUTS2 szinten, és az adott szektor igényeinek megfelelően méretezi, és biztosítja a teremfűtéshez és a meleg víz előállításához elkülönített terhelési profilokat.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Bevezetés</h2><p> A Hotmaps általános és évspecifikus hőterhelési profilokat nyújt NUTS2 szinten. A felhasználó módosíthatja az alapértelmezett profilokat a rendelkezésére álló információk alapján. Ez a számítási modul lehetőséget nyújt a profilok különböző alkotóelemeinek felfelé vagy lefelé méretezésére, és új profil megszerzésére.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Bemenetek és kimenetek</h2><h3><a class="anchor" id="input-layers-and-parameters" href="#input-layers-and-parameters"><i class="fa fa-link"></i></a> Beviteli rétegek és paraméterek</h3><p> A Hotmaps alapértelmezett adatsorából több réteget és profilt használnak:</p><ul><li> Hőszűrűségű lakossági szektor</li><li> Hőszűrűség nem lakossági szektor</li><li> Bruttó alapterületű lakó</li><li> Bruttó alapterület nem lakó</li><li> Ipari adatbázis</li><li> Terhelési profilok az ipar számára</li><li> Terhelési profilok lakossági fűtéshez</li><li> Terhelési profilok harmadlagos fűtéshez</li><li> A lakossági használati melegvíz terhelési profiljai</li><li> A felsőfokú használati melegvíz terhelési profiljai</li></ul><h3><a class="anchor" id="user-inputs" href="#user-inputs"><i class="fa fa-link"></i></a> Felhasználói bemenetek</h3><p> A felhasználónak meg kell adnia a következő paramétereket:</p><ul><li> <strong>Lakossági fűtési tényező</strong> : A lakossági fűtési igényt megszorozzuk ezzel a tényezővel.</li><li> <strong>Tercier fűtési tényező</strong> : A tercier fűtési igényt megszorozzuk ezzel a tényezővel.</li><li> <strong>Lakossági melegvíz-ellátási tényező</strong> : A lakossági melegvíz-igényt megszorozzuk ezzel a tényezővel.</li><li> <strong>Harmadlagos melegvíz-ellátási tényező</strong> : A harmadlagos melegvíz-igényt megszorozzuk ezzel a tényezővel.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Kimenet</h3><h4><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Mutatók</h4><ul><li> <strong>Hőigény ipar szerint</strong> : <strong>Ipari</strong> hőigény évente a kiválasztott területen.</li><li> <strong>Hőszükséglet lakossági fűtéssel</strong> : Évente szükséges hő a lakossági fűtéshez a kiválasztott területen.</li><li> <strong>Hőszükséglet a lakossági melegvízellátás alapján</strong> : Évente szükséges hő a lakossági melegvízellátáshoz a kiválasztott területen.</li><li> <strong>Hőigény harmadlagos szektor fűtése szerint</strong> : A <strong>harmadlagos szektor fűtéséhez</strong> szükséges éves hőmennyiség a kiválasztott területen.</li><li> <strong>Hőigény harmadlagos szektorban meleg vízellátás</strong> : Éves hőigény szükséges a harmadlagos szektor melegvízellátásához a kiválasztott területen.</li><li> <strong>Teljes</strong> fejigény: Éves teljes hőigény a kiválasztott területen.</li></ul><h4><a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Grafika</h4><ul><li> A kiválasztott terület NUTS 2 azonosítójának felhasználásával terhelési profilt rendelünk hozzá. A lakossági melegvíz-ellátás megszorozódik a lakossági melegvízellátási tényezővel abban az esetben, ha a felhasználó módosítani kívánja az értéket.</p><p> <em>Területenként fajlagos energia a meleg vízellátáshoz a lakóépületek különböző országaiban</em></p><p> | Dió0 azonosító | Meleg vízellátás kWh / m² / a | | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21,38 | | CY | 8.80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14.35 | | FI | 10.15 | | FR | 9,66 | | DE | 8,27 | | EL | 12.51 | | HU | 13,66 | | IE | 15,91 | | IT | 14.01 | | LV | 15,71 | | LT | 13,36 | | LU | 8,29 | | MT | 10,99 | | NL | 8,91 | | PL | 10.00 | | PT | 9,48 | | RO | 11,48 | | SK | 21.51 | | SI | 21,74 | | ES | 23,34 | | SE | 13.54 | | Egyesült Királyság | 49,03 |</p><h4><a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Lakossági fűtés</h4><p> A lakossági hőigény sűrűségét használják. Mivel ez az adatsor már tartalmazza a meleg vízhez felhasznált energiát, az előző számított meleg víz energiát levonják belőle. A lakóhelyiség fűtését meg kell szorozni a lakossági fűtőtényezővel, ha a felhasználó szeretné módosítani az értéket.</p><h4><a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Harmadlagos meleg vízellátás</h4><p> A lakossági melegvízellátáshoz hasonlóan a szükséges energiát egy táblázattal számoljuk.
