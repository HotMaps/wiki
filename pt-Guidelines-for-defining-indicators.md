<h2> Índice </h2><ul><li> <a href="#Indicators">Indicadores</a> <ul><li> <a href="#Simple-indicator">Indicador simples</a> </li><li> <a href="#Cross-indicator">Indicador cruzado</a> </li></ul></li><li> <a href="#Indicator-result">Resultado do indicador</a> </li><li> <a href="#references">Referências</a> </li><li> <a href="#how-to-cite">Como citar</a> </li><li> <a href="#authors-and-reviewers">Autores e revisores</a> </li><li> <a href="#license">Licença</a> </li><li> <a href="#acknowledgement">Reconhecimento</a> </li></ul><h2> Indicadores </h2><p> Esta documentação explica como gerenciar indicadores na caixa de ferramentas Hotmaps. Um indicador é um valor que é exibido na caixa de ferramentas Hotmaps no lado do cliente. O indicador faz parte de uma camada. Indicador é um valor, para uma região espacial específica, com uma unidade. Um indicador faz parte de um dicionário de camadas. Um objeto de camada contém uma matriz chamada "indicadores". Esta tabela será usada para listar todos os indicadores que você deseja ver para a camada. </p><pre> <code>layers = { 'heat_tot_curr_density_tif':{ 'tablename':'heat_tot_curr_density_tif', 'from_indicator_name':'stat_heat_tot_curr_density_tif', 'schema_scalelvl': 'stat', 'schema_hectare': 'geo', 'crs': '3035', 'geo_column': 'geometry', 'table_type':'raster', 'data_lvl':['NUTS 0','NUTS 1','NUTS 2','NUTS 3','LAU 2','Hectare'], 'data_aggregated':True, 'scalelvl_column':'', 'indicators':[ {'table_column': 'sum', 'unit': 'MWh','indicator_id':'consumption'}, {'table_column': 'count', 'unit': 'cells','indicator_id':'count_cell'}, { 'reference_indicator_id_1': 'consumption', 'reference_tablename_indicator_id_1': 'heat_tot_curr_density_tif', 'operator': '/', 'reference_indicator_id_2': 'count_cell', 'reference_tablename_indicator_id_2': 'pop_tot_curr_density_tif', 'unit':'MWh/person', 'indicator_id': 'heat_tot_curr_density_tif_per_pop_tot_curr_density_tif' } ] } }</code> </pre><ul><li> 'Nome da tabela' </li></ul><p> Nome da tabela do banco de dados. (Exemplo: 'heat_tot_curr_density_tif') </p><ul><li> 'from_indicator_name' </li></ul><p> Nome do subtítulo para a seleção de indicadores. <strong>Deve ser único.</strong> (Exemplo: 'stat_heat_tot_curr_density_tif') </p><ul><li> 'data_aggregated' </li></ul><p> Os dados são agregados ou não (Valores: Verdadeiro ou Falso) </p><ul><li> 'scalelvl_column' </li></ul><p> Nome da coluna no nível da escala, se diferente do padrão (exemplo: 'código') </p><ul><li> 'data_lvl' </li></ul><p> Níveis disponíveis para os dados no banco de dados </p><ul><li> 'schema_scalelvl' </li></ul><p> Localização do esquema da tabela para o nível das porcas. (Exemplo: 'geo', 'stat', 'public') </p><ul><li> 'schema_hectare' </li></ul><p> Localização do esquema da tabela para o nível do hectare. (Exemplo: 'geo', 'stat', 'public') </p><ul><li> 'crs' </li></ul><p> Projeção da geometria (exemplo: '3035', '4326', '4258') </p><ul><li> 'geo_column' </li></ul><p> Nome da coluna de geometria no banco de dados (exemplo: 'geom', 'geometry') </p><ul><li> 'table_type' </li></ul><p> Tipo da camada no banco de dados (Valores: 'vector' ou 'raster'). </p><p> <em><strong>Importante:</strong></em> Se for uma varredura, as colunas disponíveis são <strong>count, sum, mean, stddev, min e max</strong> </p><ul><li> 'Indicadores' </li></ul><p> Existem 2 tipos de indicadores (indicadores simples e indicadores cruzados). </p><h3> Indicador simples </h3><p> Um indicador simples é um objeto com 3 parâmetros. </p><pre> <code>{ 'table_column': 'count', 'unit': 'cells', 'indicator_id':'count_cell' }</code> </pre><ul><li> 'table_column' </li></ul><p> Esta é a coluna da tabela selecionada na tabela. (Exemplo: 'contagem') </p><p><img alt="tablecolumnselection" src="/api/assets/table_image.png"/></p><ul><li> 'unidade' </li></ul><p> Esta é a unidade do indicador. (Exemplo: 'células', 'MWh') </p><ul><li> 'indicator_id' </li></ul><p> Este é o identificador do indicador (como um ID). Este nome <strong>deve ser exclusivo</strong> na matriz do indicador. </p><h3> Indicador cruzado </h3><p> Um indicador cruzado é um objeto com 7 parâmetros. O objetivo deste indicador é fazer um cálculo entre indicadores simples e indicadores cruzados. </p><pre> <code>{ 'reference_indicator_id_1': 'consumption', 'reference_tablename_indicator_id_1':'heat_tot_curr_density_tif', 'operator': '/', 'reference_indicator_id_2':'count_cell', 'reference_tablename_indicator_id_2':'pop_tot_curr_density_tif', 'unit':'MWh/person', 'indicator_id':'heat_tot_curr_density_tif_per_pop_tot_curr_density_tif' }</code> </pre><ul><li> 'reference_indicator_id_1' </li></ul><p> Corresponde ao identificador de um indicador simples. Este nome <strong>deve ser definido</strong> na matriz do indicador. É o valor número 1. </p><ul><li> 'reference_tablename_indicator_id_1' </li></ul><p> Referência do nome da tabela da camada que faz referência ao valor número 1. (Exemplo: 'heat_tot_curr_density_tif') </p><ul><li> 'operador' </li></ul><p> Calcule a regra a ser aplicada aos 2 valores (Valores: '/' ou '*' ou '+' ou '-') </p><ul><li> 'reference_indicator_id_1' </li></ul><p> Corresponde ao identificador de um indicador simples. Este nome <strong>deve ser definido</strong> na matriz do indicador. É o valor número 2. </p><ul><li> 'reference_tablename_indicator_id_2' </li></ul><p> Referência do nome da tabela da camada que faz referência ao valor número 2. (Exemplo: 'pop_tot_curr_density_tif') </p><ul><li> 'unidade' </li></ul><p> Esta é a unidade do indicador. (Exemplo: 'células', 'MWh') </p><ul><li> 'nome' </li></ul><p> Este é o nome do indicador (como um ID). Este nome <strong>deve ser exclusivo</strong> na matriz do indicador. </p><h5> Nota: Para este exemplo, o cálculo abaixo é feito. </h5><pre> <code>reference_indicator_id_1.reference_indicator_id_1 / reference_indicator_id_1.reference_indicator_id_1 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code> </pre><h2> Resultado do indicador </h2><p> O resultado dos indicadores é o seguinte: </p><pre> <code>{ "values": [ { "unit": "MWh", "name": "heat_tot_curr_density_tif_consumption", "value": "4112030.46" }, { "unit": "cells", "name": "heat_tot_curr_density_tif_count_cell", "value": "46764" }, { "unit": "MWh/person", "name": "heat_tot_curr_density_tif_per_pop_tot_curr_density_tif", "value": "38.0092476775893146" } ], "name": "heat_tot_curr_density_tif" }</code> </pre><h2> Referências </h2><h2> Como citar </h2><p> Mostafa Fallahnejad, em Hotmaps-Wiki, https://github.com/HotMaps/hotmaps_wiki/wiki/Guidelines-for-defining-indicators (abril de 2019) </p><h2> Autores e revisores </h2><p> Esta página foi escrita por Mostafa Fallahnejad *. </p><ul><li> [] Esta página foi revisada por Lukas Kranzl *. </li></ul><p> * <a href="https://eeg.tuwien.ac.at/">Grupo de Economia de Energia -
