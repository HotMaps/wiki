<h2> Índice </h2><ul><li> <a href="#Indicators">Indicadores</a> <ul><li> <a href="#Simple-indicator">Indicador simples</a> </li><li> <a href="#Cross-indicator">Indicador cruzado</a> </li></ul></li><li> <a href="#Indicator-result">Resultado do indicador</a> </li><li> <a href="#references">Referências</a> </li><li> <a href="#how-to-cite">Como citar</a> </li><li> <a href="#authors-and-reviewers">Autores e revisores</a> </li><li> <a href="#license">Licença</a> </li><li> <a href="#acknowledgement">Reconhecimento</a> </li></ul><h2> Indicadores </h2><p> Esta documentação explica como gerenciar indicadores na caixa de ferramentas Hotmaps. Um indicador é um valor que é exibido na caixa de ferramentas Hotmaps no lado do cliente. O indicador faz parte de uma camada. Indicador é um valor, para uma região espacial específica, com uma unidade. Um indicador faz parte de um dicionário de camadas. Um objeto de camada contém uma matriz chamada &quot;indicadores&quot;. Esta tabela será usada para listar todos os indicadores que você deseja ver para a camada. </p><pre> <code>layers = { &#39;heat_tot_curr_density_tif&#39;:{ &#39;tablename&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;from_indicator_name&#39;:&#39;stat_heat_tot_curr_density_tif&#39;, &#39;schema_scalelvl&#39;: &#39;stat&#39;, &#39;schema_hectare&#39;: &#39;geo&#39;, &#39;crs&#39;: &#39;3035&#39;, &#39;geo_column&#39;: &#39;geometry&#39;, &#39;table_type&#39;:&#39;raster&#39;, &#39;data_lvl&#39;:[&#39;NUTS 0&#39;,&#39;NUTS 1&#39;,&#39;NUTS 2&#39;,&#39;NUTS 3&#39;,&#39;LAU 2&#39;,&#39;Hectare&#39;], &#39;data_aggregated&#39;:True, &#39;scalelvl_column&#39;:&#39;&#39;, &#39;indicators&#39;:[ {&#39;table_column&#39;: &#39;sum&#39;, &#39;unit&#39;: &#39;MWh&#39;,&#39;indicator_id&#39;:&#39;consumption&#39;}, {&#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;,&#39;indicator_id&#39;:&#39;count_cell&#39;}, { &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;: &#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;: &#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;: &#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;: &#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; } ] } }</code> </pre><ul><li> &#39;Nome da tabela&#39; </li></ul><p> Nome da tabela do banco de dados. (pt-Exemplo: &#39;heat_tot_curr_density_tif&#39;) </p><ul><li> &#39;from_indicator_name&#39; </li></ul><p> Nome do subtítulo para a seleção de indicadores. <strong>Deve ser único.</strong> (Exemplo: &#39;stat_heat_tot_curr_density_tif&#39;) </p><ul><li> &#39;data_aggregated&#39; </li></ul><p> Os dados são agregados ou não (Valores: Verdadeiro ou Falso) </p><ul><li> &#39;scalelvl_column&#39; </li></ul><p> Nome da coluna no nível da escala, se diferente do padrão (exemplo: &#39;código&#39;) </p><ul><li> &#39;data_lvl&#39; </li></ul><p> Níveis disponíveis para os dados no banco de dados </p><ul><li> &#39;schema_scalelvl&#39; </li></ul><p> Localização do esquema da tabela para o nível das porcas. (Exemplo: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;) </p><ul><li> &#39;schema_hectare&#39; </li></ul><p> Localização do esquema da tabela para o nível do hectare. (Exemplo: &#39;geo&#39;, &#39;stat&#39;, &#39;public&#39;) </p><ul><li> &#39;crs&#39; </li></ul><p> Projeção da geometria (exemplo: &#39;3035&#39;, &#39;4326&#39;, &#39;4258&#39;) </p><ul><li> &#39;geo_column&#39; </li></ul><p> Nome da coluna de geometria no banco de dados (exemplo: &#39;geom&#39;, &#39;geometry&#39;) </p><ul><li> &#39;table_type&#39; </li></ul><p> Tipo da camada no banco de dados (Valores: &#39;vector&#39; ou &#39;raster&#39;). </p><p> <em><strong>Importante:</strong></em> Se for uma varredura, as colunas disponíveis são <strong>count, sum, mean, stddev, min e max</strong> </p><ul><li> &#39;Indicadores&#39; </li></ul><p> Existem 2 tipos de indicadores (indicadores simples e indicadores cruzados). </p><h3> Indicador simples </h3><p> Um indicador simples é um objeto com 3 parâmetros. </p><pre> <code>{ &#39;table_column&#39;: &#39;count&#39;, &#39;unit&#39;: &#39;cells&#39;, &#39;indicator_id&#39;:&#39;count_cell&#39; }</code> </pre><ul><li> &#39;table_column&#39; </li></ul><p> Esta é a coluna da tabela selecionada na tabela. (Exemplo: &#39;contagem&#39;) </p><p><img alt="tablecolumnselection" src="/api/assets/table_image.png"/></p><ul><li> &#39;unidade&#39; </li></ul><p> Esta é a unidade do indicador. (Exemplo: &#39;células&#39;, &#39;MWh&#39;) </p><ul><li> &#39;indicator_id&#39; </li></ul><p> Este é o identificador do indicador (como um ID). Este nome <strong>deve ser exclusivo</strong> na matriz do indicador. </p><h3> Indicador cruzado </h3><p> Um indicador cruzado é um objeto com 7 parâmetros. O objetivo deste indicador é fazer um cálculo entre indicadores simples e indicadores cruzados. </p><pre> <code>{ &#39;reference_indicator_id_1&#39;: &#39;consumption&#39;, &#39;reference_tablename_indicator_id_1&#39;:&#39;heat_tot_curr_density_tif&#39;, &#39;operator&#39;: &#39;/&#39;, &#39;reference_indicator_id_2&#39;:&#39;count_cell&#39;, &#39;reference_tablename_indicator_id_2&#39;:&#39;pop_tot_curr_density_tif&#39;, &#39;unit&#39;:&#39;MWh/person&#39;, &#39;indicator_id&#39;:&#39;heat_tot_curr_density_tif_per_pop_tot_curr_density_tif&#39; }</code> </pre><ul><li> &#39;reference_indicator_id_1&#39; </li></ul><p> Corresponde ao identificador de um indicador simples. Este nome <strong>deve ser definido</strong> na matriz do indicador. É o valor número 1. </p><ul><li> &#39;reference_tablename_indicator_id_1&#39; </li></ul><p> Referência do nome da tabela da camada que faz referência ao valor número 1. (Exemplo: &#39;heat_tot_curr_density_tif&#39;) </p><ul><li> &#39;operador&#39; </li></ul><p> Calcule a regra a ser aplicada aos 2 valores (Valores: &#39;/&#39; ou &#39;*&#39; ou &#39;+&#39; ou &#39;-&#39;) </p><ul><li> &#39;reference_indicator_id_1&#39; </li></ul><p> Corresponde ao identificador de um indicador simples. Este nome <strong>deve ser definido</strong> na matriz do indicador. É o valor número 2. </p><ul><li> &#39;reference_tablename_indicator_id_2&#39; </li></ul><p> Referência do nome da tabela da camada que faz referência ao valor número 2. (Exemplo: &#39;pop_tot_curr_density_tif&#39;) </p><ul><li> &#39;unidade&#39; </li></ul><p> Esta é a unidade do indicador. (Exemplo: &#39;células&#39;, &#39;MWh&#39;) </p><ul><li> &#39;nome&#39; </li></ul><p> Este é o nome do indicador (como um ID). Este nome <strong>deve ser exclusivo</strong> na matriz do indicador. </p><h5> Nota: Para este exemplo, o cálculo abaixo é feito. </h5><pre> <code>reference_indicator_id_1.reference_indicator_id_1 / reference_indicator_id_1.reference_indicator_id_1 = heat_tot_curr_density_tif.consumption / pop_tot_curr_density_tif.count_cell</code> </pre><h2> Resultado do indicador </h2><p> O resultado dos indicadores é o seguinte: </p><pre> <code>{ &quot;values&quot;: [ { &quot;unit&quot;: &quot;MWh&quot;, &quot;name&quot;: &quot;heat_tot_curr_density_tif_consumption&quot;, &quot;value&quot;: &quot;4112030.46&quot;
