Používa predvolené profily zaťaženia pre rezidenčný a terciárny sektor na úrovni NUTS2 a upravuje ich podľa daného dopytu v týchto sektoroch a poskytuje profily zaťaženia oddelene pre vykurovanie miestností a ohrev teplej vody.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="introduction" href="#introduction"><i class="fa fa-link"></i></a> Úvod</h2><p> Hotmaps poskytuje všeobecné a ročné profily tepelného zaťaženia na úrovni NUTS2. Používateľ môže chcieť upraviť predvolené profily na základe informácií, ktoré má. Tento výpočtový modul poskytuje možnosť škálovania rôznych konštitučných prvkov profilov nahor alebo nadol a získať nový profil.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h2><a class="anchor" id="inputs-and-outputs" href="#inputs-and-outputs"><i class="fa fa-link"></i></a> Vstupy a výstupy</h2><h3><a class="anchor" id="input-layers-and-parameters" href="#input-layers-and-parameters"><i class="fa fa-link"></i></a> Vstupné vrstvy a parametre</h3><p> Z predvolenej množiny údajov Hotmaps sa používa niekoľko vrstiev a profilov:</p><ul><li> Hustota tepla v obytnom sektore</li><li> Hustota tepla nebytový sektor</li><li> Hrubá podlahová plocha obytná</li><li> Hrubá podlahová plocha neobytná</li><li> Priemyselná databáza</li><li> Profily zaťaženia pre priemysel</li><li> Profily zaťaženia pre vykurovanie domácností</li><li> Zaťažovacie profily pre terciárne vykurovanie</li><li> Profily zaťaženia teplej úžitkovej vody pre domácnosť</li><li> Profily zaťaženia pre terciálnu teplú úžitkovú vodu</li></ul><h3><a class="anchor" id="user-inputs" href="#user-inputs"><i class="fa fa-link"></i></a> Vstupy používateľov</h3><p> Užívateľ by mal uviesť nasledujúce parametre:</p><ul><li> <strong>Faktor vykurovania domácnosti</strong> : Potreba tepla na bývanie sa vynásobí týmto faktorom.</li><li> <strong>Faktor</strong> terciárneho vykurovania: Potreba terciárneho vykurovania sa vynásobí týmto faktorom.</li><li> <strong>Faktor zásobovania teplou vodou</strong> pre domácnosti: Potreba teplej vody pre domácnosť sa vynásobí týmto faktorom.</li><li> <strong>Faktor</strong> terciárneho <strong>zásobovania teplou vodou</strong> : Potreba terciárneho <strong>zásobovania</strong> teplou vodou sa vynásobí týmto faktorom.</li></ul><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="output" href="#output"><i class="fa fa-link"></i></a> Výkon</h3><h4><a class="anchor" id="indicators" href="#indicators"><i class="fa fa-link"></i></a> Ukazovatele</h4><ul><li> <strong>Potreba tepla podľa priemyslu</strong> : Ročná potreba tepla podľa priemyslu vo vybranej oblasti.</li><li> <strong>Potreba tepla na vykurovanie domácností</strong> : Ročné teplo potrebné na vykurovanie domácností vo vybranej oblasti.</li><li> <strong>Potreba tepla dodávkou teplej úžitkovej vody</strong> : Ročné teplo potrebné na dodávku teplej úžitkovej vody vo vybranej oblasti.</li><li> <strong>Potreba tepla terciálnym sektorovým vykurovaním</strong> : Ročné teplo potrebné na vykurovanie terciárnym sektorom vo vybranej oblasti.</li><li> <strong>Potreba tepla dodávkou teplej vody z terciárneho sektoru</strong> : Ročné teplo potrebné na dodávku teplej vody z terciárneho sektoru vo vybranej oblasti.</li><li> <strong>Celková potreba hlavy</strong> : Celkové ročné teplo potrebné vo vybranej oblasti.</li></ul><h4><a class="anchor" id="graphics" href="#graphics"><i class="fa fa-link"></i></a> Grafika</h4><ul><li> <strong>Profily dopytu po teple</strong> : Grafika znázorňujúca vyššie uvedené ukazovatele v priebehu roka.</li></ul><p> Os x predstavuje čas a os y ukazuje výkon v MW. Pomocou ID NUTS 2 vybranej oblasti sa priradí záťažový profil. Dodávka teplej vody pre domácnosť sa vynásobí činiteľom dodávky teplej vody pre domácnosť v prípade, že užívateľ chce hodnotu upraviť.</p><p> <em>Špecifická energia na plochu na zásobovanie teplou vodou v obytných budovách v rôznych krajinách</em></p><p> | Orechy0 ID | Dodávka teplej vody v kWh / m² / a | ------------- |: -------------: | | AT | 21,67 | | BE | 31,95 | | BG | 12,93 | | HR | 21,38 | | CY | 8,80 | | CZ | 22,83 | | DK | 9,64 | | EE | 14,35 | | FI | 10.15 | | FR | 9,66 | | DE | 8,27 | | EL | 12,51 | | HU | 13,66 | | IE | 15,91 | | IT | 14.01 | | LV | 15,71 | | LT | 13,36 | | LU | 8,29 | | MT | 10,99 | | NL | 8,91 | | PL | 10,00 | | PT | 9,48 | | RO | 11,48 | | SK | 21,51 | | SI | 21,74 | | ES | 23,34 | | SE | 13,54 | | Spojené kráľovstvo | 49,03 |</p><h4><a class="anchor" id="residential-heating" href="#residential-heating"><i class="fa fa-link"></i></a> Obytné kúrenie</h4><p> Využíva sa hustota potreby tepla pre bytový sektor. Pretože táto množina údajov už obsahuje energiu použitú na teplú vodu, odčíta sa od nej predchádzajúca vypočítaná energia teplej vody. Na základe ID NUTS 2 je pre každú dlaždicu tepelnej hustoty pri výbere používateľa priradený profil vykurovania domácnosti. Vykurovanie v domácnosti sa vynásobí činiteľom vykurovania v prípade, že chce užívateľ upraviť hodnotu.</p><h4><a class="anchor" id="tertiary-warm-water-supply" href="#tertiary-warm-water-supply"><i class="fa fa-link"></i></a> Terciárny prísun teplej vody</h4><p> Podobne ako pri dodávke teplej vody pre domácnosť sa potrebná energia počíta z tabuľky. S ID NUTS 2 konkrétnej oblasti je priradený záťažový profil. Terciárne vykurovanie sa vynásobí koeficientom terciárneho vykurovania v prípade, že užívateľ chce upraviť hodnotu.</p><h4><a class="anchor" id="industrial-heat-demand" href="#industrial-heat-demand"><i class="fa fa-link"></i></a> Priemyselná potreba tepla</h4><p> Potreba tepla podľa priemyslu sa vyhľadáva v priemyselnej databáze a na základe ID NUTS 0 a subsektora sa priraďuje záťažový profil.</p><h4><a class="anchor" id="summation" href="#summation"><i class="fa fa-link"></i></a> Zhrnutie</h4><p> Päť výsledných profilov sa agreguje absolútnym spôsobom, aby sa získal konečný profil.</p><p> <a href="#table-of-contents"><strong><code>To Top</code></strong></a></p><h3><a class="anchor" id="implementation" href="#implementation"><i class="fa fa-link"></i></a> Implementácia</h3><h4><a class="anchor" id="load-profiles" href="#load-profiles"><i class="fa fa-link"></i></a> Načítať profily</h4><p> Uvedené profily zaťaženia pozostávajú z 8760 bodov, ktoré predstavujú zaťaženie za každú hodinu z 365 dní.
